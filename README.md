# vibe-coding-lab

Vibe coding 学习经验、踩坑笔记与项目记录。

## 关于 Vibe Coding

通过自然语言描述意图，借助 AI 协同生成与迭代代码的开发方式。重点在快速试验、跟着感觉走，把更多精力放在「想做什么」而非「怎么写出来」。

但 "vibe" 不是放飞 —— **代码之前先对齐，完成之前先验证**，仍然适用，甚至更重要。本仓库收录的第一份内容就是围绕这条原则的工作流规范。
