### 建议人工重点测什么（自动化覆盖不到）

1. **真机移动端**（AC14）— DevTools 模拟跟真机有差异：触摸响应、IME 时序、Safari 特殊行为
2. **LLM 边界用例**（合并 T15 推后到这里）— 30 个用例：常见异称 / 错字 / 似是而非（如"诸葛大军师" / "曹孟德" / 繁体"諸葛亮"）。每个抽 2-3 个 figure 测，记失败率
3. **视觉感觉**（OQ9 中国风落地是否好看）— 字体加载、米色配朱砂红是否舒适、移动端字号合适
4. **首页 daily 状态显示**（AC12）— 首次访问 / 玩完 daily 后 / 跨天后 三种状态正确
5. **daily fix(T14) 验证** — CF Pages deploy 完成后 curl /api/daily 验证 day_index >= 0
6. **题库错误 spot check**（T25）— 随机抽 10 人审：事实准确 / 难度梯度 / 异称完整。发现错的用 `fix(T5.N):` commit 修
