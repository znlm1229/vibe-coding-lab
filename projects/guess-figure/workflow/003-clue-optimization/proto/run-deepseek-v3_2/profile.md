# 诸葛亮

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字孔明 / 谥号忠武侯 / 别号卧龙 / 后世尊称诸葛武侯 / 无庙号
- 生卒年 / 朝代区间 / 籍贯: 181年—234年10月8日 / 东汉末年至三国时期 / 琅琊郡阳都县（今山东省临沂市沂南县）
- 主要职业 / 身份: 蜀汉丞相、政治家、军事家、发明家、散文家

## 主要事迹
- 建安十二年（207年），刘备三顾茅庐，诸葛亮提出《隆中对》（三分天下战略），奠定蜀汉立国基础。 [重要]
- 建安十三年（208年），出使江东说服孙权联合抗曹，促成赤壁之战胜利。 [重要]
- 建安十九年（214年），与张飞、赵云溯江入蜀，分定郡县，助刘备夺取益州。 [重要]
- 建安二十六年（221年），以耿纯故事劝刘备称帝，刘备即位后拜其为丞相。 [一般]
- 章武三年（223年），刘备于永安病逝，临终托孤，诸葛亮承诺“竭股肱之力，效忠贞之节”。 [重要]
- 建兴三年（225年），率军南征，平定南中叛乱，稳定后方并获取资源。 [重要]
- 建兴五年至十二年（227-234年），主持五次北伐中原，以图兴复汉室。 [重要]
- 建兴十二年（234年），北伐中病逝于五丈原军中，遗命葬汉中定军山。 [重要]

## 性格 / 风格特征
- 自视甚高，年轻时常自比管仲、乐毅，有宏大抱负。（源自《三国志》“每自比於管仲、樂毅”）
- 处事谨慎周密，刘备评价“知臣謹慎”，北伐前上《出师表》亦体现其细致谋划。
- 执法严明，赏罚必信，陈寿评其“科教嚴明，賞罰必信，無惡不懲，無善不顯”。
- 廉洁自律，临终自表财产仅“桑八百株，薄田十五顷”，死后家无余财。

## 典故 / 标志事件
- 三顾茅庐：刘备三次拜访诸葛亮于隆中，始得相见。
- 草庐对（隆中对）：诸葛亮为刘备分析天下形势，提出占据荆益、三分天下的战略。
- 白帝城托孤：刘备临终将幼子刘禅与国事托付给诸葛亮。
- 鞠躬尽瘁，死而后已：诸葛亮在《出师表》中表达为蜀汉竭尽全力的决心，后成为其精神象征。
- 挥泪斩马谡：因马谡失街亭，诸葛亮虽惜其才但仍依法处斩，并自贬三级。

## 关键作品
- 《隆中对》（又称《草庐对》）：奠定蜀汉战略方向的政论。
- 《出师表》（前、后）：北伐前上奏后主的表文，体现其政治理念与忠诚。
- 《诸葛亮集》（二十四篇）：其言论、教令、书奏的合集。
- 木牛流马：为解决山地运粮难题而发明的运输工具。
- 八阵图：推演兵法而成的阵法。

## 关系网
- 老师 / 前辈: 庞德公（荆州名士，结交请教）
- 同辈 / 友人: 徐庶（元直）、崔州平、石广元、孟公威（早年挚友）
- 弟子 / 下属: 马谡（受其赏识但兵败被斩）、蒋琬、费祎（后继者）
- 政敌 / 对手: 司马懿（曹魏统帅，北伐主要对手）
- 家人: 诸葛瑾（兄，仕于吴）、诸葛均（弟）、黄氏（妻，黄承彦之女）、诸葛瞻（子）

## 历史评价
- 正面: 陈寿评其“治戎为长，奇谋为短，理民之干，优于将略”，司马懿观其营垒称“天下奇才”。后世尊为忠臣与智者典范。
- 负面: 陈寿指出其“奇谋为短”，且北伐“屡耀其武”但未能成功；部分史家认为其治国过于严苛。
- 后世神话/演义: 明代《三国演义》将其塑造为“智绝”，通晓阴阳、料事如神，形象高度神话化。

## 反差 / 鲜为人知点
- 其妻黄氏被描述为“黄头黑色”的丑女，诸葛亮却因其才华而娶之，时人作为笑谈。
- 根据《魏略》和《九州春秋》记载，存在诸葛亮主动求见刘备而非“三顾”的版本，与主流记载相悖。
- 北伐并非纯粹“以攻为守”，其素志是“进欲龙骧虎视，包括四海”，有强烈的主动进取中原的野心。
