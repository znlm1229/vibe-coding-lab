# 诸葛亮

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字孔明，号卧龙，谥忠武侯，别号诸葛武侯
- 生卒年 / 朝代区间 / 籍贯: 181年—234年10月8日 / 东汉末至三国蜀汉 / 琅琊郡阳都县（今山东省临沂市沂南县）
- 主要职业 / 身份: 蜀汉丞相、政治家、军事家、发明家、散文家

## 主要事迹
- 建安十二年（207年）冬，刘备三顾茅庐，诸葛亮献《隆中对》，提出三分天下之策 [重要]
- 建安十三年（208年），主动请缨出使孙权，以利害分析说服孙权联合抗曹，促成赤壁之战胜利 [重要]
- 建安十四年（209年），以军师中郎将身份督零陵、桂阳、长沙三郡，调赋税供军需 [重要]
- 建安十七年（212年）至建安十九年（214年），参与攻取益州，与张飞、赵云等围成都 [重要]
- 建安二十六年（221年），劝说刘备接受群臣拥立，刘备称帝后拜为丞相 [重要]
- 章武二年（222年），刘备白帝城托孤，诸葛亮涕泣发誓辅佐后主 [重要]
- 建兴三年（225年）春，率众南征平定南中叛乱，获得大量军资 [一般]
- 建兴五年（227年）至建兴十二年（234年），五次北伐，与魏军对峙，最后卒于五丈原 [重要]
- 建兴六年（228年）首次北伐攻祁山，因马谡违命街亭之战失利，自贬三等 [重要]
- 建兴九年（231年）第三次北伐，以木牛流马运粮，射杀魏将张郃 [一般]

## 性格 / 风格特征
- 谨慎自律：自表后主称"成都有桑八百株，薄田十五顷"，临终如约不留余财，廉洁自守 [史料出处：二十四史本传]
- 赏罚分明："科教严明，赏罚必信，无恶不懲，无善不显，至于吏不容奸，人怀自厉" [史料出处：二十四史本传]
- 知人善任但不容失误：重用马谖却因其违命街亭战败而"戮谖以谢众"，说明用人严格但也能承担责任 [史料出处：二十四史本传]
- 志存高远但理民优于将略：虽"屡耀其武"北伐，但史评"于治戎为长，奇谋为短，理民之干，优于将略" [史料出处：二十四史本传]

## 典故 / 标志事件
- 三顾茅庐：刘备三次拜访诸葛亮于隆中草庐，始得见面
- 隆中对（草庐对）：诸葛亮献三分天下之计，分析荆益二州战略价值
- 舌战群儒：诸葛亮出使孙权，以利害分析说服孙权联合抗曹
- 赤壁之战：诸葛亮促成孙刘联盟，参与赤壁战役胜利
- 街亭之战：马谖违命导致街亭失守，诸葛亮斩马谖自贬以示负责
- 五丈原之役：诸葛亮最后一次北伐，卒于五丈原军中

## 关键作品
- 《出师表》：北伐前上疏后主，阐述北伐大义，劝诫后主亲贤远小人，传世名篇
- 《诸葛氏集》：二十四篇共十万四千一百一十二字，包括权制、南征、北出、兵要、法检、军令等
- 木牛流马：诸葛亮发明的运输工具，用于北伐时运送粮草
- 诸葛连弩：改进的弩机，提高了军队远程火力
- 八阵图：推演兵法创作的阵法图示，据传可用于训练军队

## 关系网
- 老师 / 同门：司马徽（荆州名士，推荐诸葛亮给刘备）、龐德公（荆州名士，与诸葛亮结交）
- 同辈 / 友人：徐庶（潁川人，与诸葛亮交好，后投曹操）、崔州平（博陵人，与诸葛亮交好）、龐统（鳳雛，师出同门齐名）
- 主君 / 上级：刘备（主公，三顾茅庐始见）、刘禅（后主，白帝托孤后辅佐）
- 政敌 / 对手：司马懿（魏将，五丈原对峙）、曹真（魏大将军，北伐对手）、张郃（魏将，多次交战）
- 家人：诸葛玄（叔父，袁术署为豫章太守）、诸葛瑾（兄长，出仕孙权）、诸葛均（弟弟，官至长水校尉）、黄月英（妻子，黄承彦之女）、诸葛瞻（儿子，嗣爵武乡侯）

## 历史评价
- 正面：常被后世认为是智慧和忠义的典范；司马懿赞"天下奇才也"；毛宗岗评《三国演义》将诸葛亮称为"三绝"中的"智绝"；"鞠躬尽瘁、死而后已"是中国传统文化忠臣与智者的代表
- 负面：北伐五次皆未成功，耗费蜀汉国力；史评"奇谋为短"，军事才能不如治国理政；过度集权于一身，导致后主刘禅权力被架空
- 后世神话/演义：民间小说、戏曲谓其通晓阴阳，料事如神；《三国演义》中被塑造为近乎完美的智者，能掐会算，通晓天文地理；被神化为"卧龙先生"，具有超人的预见能力

## 反差 / 鲜为人知点
- 娶丑女为妻：黄承彦主动献女，称"身有丑女，黄头黑色，而才堪相配"，诸葛亮欣然接受，时人讥笑，乡里谚曰"莫作孔明择婦，正得阿承醜女"——这与后世将其神化为完美人物的形象形成反差
- 早年被同时人否定：躬耕隆中时"每自比于管仲、乐毅，时人莫之许也"，只有崔州平、徐庶等少数人认可，说明其名声并非一开始就显赫
- 对刘琦的冷漠：刘琦多次请教自安之术被诸葛亮"动辄拒绝"，直到刘琦上屋抽梯逼迫才给出建议，显示其并非对所有人都热心相助
