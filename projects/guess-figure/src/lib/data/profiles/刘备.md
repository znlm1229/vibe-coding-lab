# 刘备

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字玄德 / 无 / 谥昭烈 / 庙号烈祖 / 别号汉先主
- 生卒年 / 朝代区间 / 籍贯: 161年—223年 / 东汉末年至三国时期 / 涿郡涿县（今河北涿州）
- 主要职业 / 身份: 政治家、军事家、蜀汉开国皇帝

## 主要事迹
- 早年参与镇压黄巾起义，因功授安喜尉，后鞭打督邮弃官逃亡 [一般]
- 救援北海孔融，后依附徐州牧陶谦，并在陶谦死后被推举为徐州牧 [一般]
- 被吕布偷袭夺取徐州，后联合曹操消灭吕布，但未重掌徐州 [一般]
- 参与“衣带诏”密谋诛杀曹操，后借口脱离曹操控制，占据徐州反曹 [重要]
- 被曹操击败后投奔袁绍，后又南下依附荆州刘表 [一般]
- 采纳诸葛亮建议，联合孙权于赤壁之战击败曹操 [重要]
- 采纳张松、法正建议，进军益州，击败刘璋，占据蜀地 [重要]
- 击败曹操夺取汉中，自立为汉中王 [重要]
- 在成都称帝，建立蜀汉政权 [重要]
- 为夺回荆州兴兵伐吴，于夷陵之战大败，病逝白帝城 [重要]

## 性格 / 风格特征
- 喜怒不形于色，善于待人接物，能得人心（刺客不忍刺杀，饥民争相归附）
- 有雄心壮志，少时曾言“吾必当乘此羽葆盖车”（即帝王车驾）
- 不甚乐读书，喜狗马、音乐、美衣服，好交结豪侠
- 行事风格酷似先祖刘邦，时人评为有高祖之风

## 典故 / 标志事件
- 三顾茅庐请诸葛亮出山。
- 与曹操煮酒论英雄时失匕箸（筷子掉落）。
- 赤壁之战联合孙权击败曹操。
- 占据益州建立蜀汉政权。
- 夷陵之战大败于孙权。

## 关键作品
- 建立蜀汉政权（章武元年称帝）。
- 颁布《蜀科》（与诸葛亮、法正等人制定律法）。
- 与诸葛亮、法正等共同奠定蜀汉治国基础（政绩）。
- 遗诏托孤诸葛亮于白帝城（政治遗产）。

## 关系网
- 老师: 卢植
- 同辈: 关羽、张飞（结义兄弟/部将）
- 弟子: 无明确弟子，可视为刘禅（继承人）
- 政敌: 曹操、孙权
- 家人: 刘禅（子）、甘夫人/糜夫人（妻）

## 历史评价
- 正面: 弘毅宽厚，知人待士，有高祖之风，百折不挠终成帝业。
- 负面: 机权干略不及曹操，前期屡战屡败，颠沛流离；晚年伐吴决策失误致国力大损。
- 后世神话/演义: 《三国演义》将其塑造为仁德宽厚、重情重义的理想化君主形象，与关羽、张飞的兄弟情谊被大力渲染。

## 反差 / 鲜为人知点
- 早年曾以贩履织席（卖草鞋、织草席）为业，出身并非显贵。
- 在许都依附曹操时，曾闭门种芜菁（一种蔬菜）以韬光养晦，避免曹操猜忌。
- 其“汉室宗亲”身份存在争议，《典略》记载其为“临邑侯枝属”，可能与中山靖王刘胜关系不直接。
