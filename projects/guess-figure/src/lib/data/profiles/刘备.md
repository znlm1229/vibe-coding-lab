# 刘备

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字玄德 / 谥号昭烈皇帝 / 庙号烈祖 / 别号汉先主、昭烈帝
- 生卒年 / 朝代区间 / 籍贯: 161年—223年6月10日 / 东汉末年至三国时期 / 涿郡涿县（今河北省涿州市）
- 主要职业 / 身份: 政治家、军事家、蜀汉开国皇帝

## 主要事迹
- 184年，率部参与镇压黄巾起义，因功被任命为安喜尉。[一般]
- 194年，徐州牧陶谦病逝，在麋竺、陈登等人支持下入主徐州。[一般]
- 196-198年，与吕布反复争夺徐州，最终兵败投奔曹操。[一般]
- 200年，参与“衣带诏”事件密谋诛杀曹操，后借机脱离曹操控制，占据徐州。[重要]
- 208年，联合孙权于赤壁之战击败曹操，奠定三分天下基础。[重要]
- 211-214年，应刘璋之邀入益州，后反客为主，击败刘璋夺取益州。[重要]
- 219年，于定军山之战击败曹军，斩杀夏侯渊，夺取汉中，自立为汉中王。[重要]
- 221年，于成都称帝，建立蜀汉政权。[重要]
- 222-223年，为夺回荆州兴兵伐吴，于夷陵之战大败，病逝于白帝城。[重要]

## 性格 / 风格特征
- 喜怒不形于色，善于待人接物，能得人心（刺客受其礼遇不忍下手）。[源自《三国志》]
- 有高祖（刘邦）之风，好交结豪侠，年轻时便有志向（曾言欲乘羽葆盖车）。[源自《三国志》]
- 不甚乐读书，喜狗马、音乐、美衣服，生活品味接近贵族。[源自《三国志》]
- 在关键时刻常表现出仁德与不忍（如不忍攻刘琮、不忍弃百姓）。[源自《三国志》]

## 典故 / 标志事件
- 三顾茅庐：刘备三次拜访诸葛亮，请他出山辅佐。
- 煮酒论英雄：曹操试探刘备，称“今天下英雄，唯使君与操耳”，刘备惊失匕箸。
- 携民渡江：曹操南征时，刘备不忍弃追随的十余万荆州百姓，同行缓慢撤退。
- 白帝城托孤：刘备病逝前于白帝城将儿子刘禅与国事托付给诸葛亮。
- 夷陵之战：刘备为夺回荆州兴兵伐吴，遭火攻大败，国力大损。

## 关键作品
- 建立蜀汉政权：在成都称帝，开创三国鼎立之一的蜀汉王朝。
- 夺取并治理益州：击败刘璋，取得战略根据地，并有效整合当地势力。
- 夺取汉中：通过定军山之战击败曹操，取得汉中这一战略要地。
- 联吴抗曹：促成孙刘联盟，并在赤壁之战中击败曹操。
- 《遗诏敕后主》：临终前给儿子刘禅的诏书，嘱托“勿以恶小而为之，勿以善小而不为”。

## 历史评价
- 正面: 被视为仁德之君，善于团结人才，有复兴汉室的志向，得民心。
- 负面: 被认为早期颠沛流离，缺乏稳固根基；晚年伐吴决策失误，导致国力大损。
- 后世神话/演义: 在《三国演义》中被塑造为仁义的典型代表，形象高度理想化，与关羽、张飞的“桃园三结义”广为流传。

## 反差 / 鲜为人知点
- 刘备早年曾因督邮傲慢，直接闯入将其捆绑杖打二百下后弃官逃亡，展现出刚烈的一面。[源自《三国志》]
- 刘备在许都依附曹操时，曾闭门种芜菁（萝卜类蔬菜）以韬光养晦，并非一直积极图谋。[源自《三国志》注引《吴历》]
- 其出身虽有“中山靖王之后”的说法，但亦有史料记载可能是“临邑侯枝属”，宗室身份存在不同记载。[源自裴松之注引《典略》]
