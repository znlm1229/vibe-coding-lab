# 柳宗元

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字子厚 / 柳河东 / 柳柳州 / 无谥号庙号
- 生卒年 / 朝代区间 / 籍贯: 773年-819年 / 唐朝中期 / 河东（今山西运城）
- 主要职业 / 身份: 文学家、思想家、政治家

## 主要事迹
1. 793年进士及第，踏入仕途 [一般]
2. 805年参与王叔文领导的永贞革新，任礼部员外郎 [重要]
3. 805年革新失败后，被贬为邵州刺史，途中改贬永州司马，开始了长达十年的流放生涯 [重要]
4. 在永州期间，创作了大量文学作品，包括著名的山水游记“永州八记” [重要]
5. 815年初被召回长安，但不到两个月后再次外放，任柳州刺史 [一般]
6. 在柳州任上，为百姓凿井、教化土著、解放奴婢 [一般]
7. 819年卒于柳州，享年46岁 [次要]

## 性格 / 风格特征
1. 自视儒家改革者，怀有强烈的济世安民理想。
2. 流放后性格转向孤愤、抑郁，借山水诗文寄托内心悲慨。
3. 文学创作上主张“文以明道”，风格冷静超逸，善用曲笔讽喻。

## 典故 / 标志事件
1. 参与永贞革新旋即失败获罪贬谪。
2. 长期流放永州，创作“永州八记”。
3. 在柳州为百姓凿井、解放奴婢。
4. 卒于柳州，柳州人建庙纪念。
5. 与韩愈齐名，同为唐代古文运动领袖。

## 关键作品
1. 《永州八记》（山水游记典范）
2. 《捕蛇者说》（揭露苛政，关怀民苦）
3. 《封建论》（政治思想论述）
4. 《天对》（回应屈原《天问》，阐述理性思想）
5. 《江雪》（著名诗作，体现孤寂不屈精神）

## 关系网
- 政治盟友 / 同辈: 王叔文 (革新领袖)、刘禹锡 (同贬“八司马”之一)、韩愈 (古文运动盟友)
- 家人: 父亲柳镇 (官员)、母亲卢氏 (范阳卢氏)、妻子杨氏 (弘农杨氏)
- 弟子 / 后学: 无明确记载的弟子，但其文学思想影响后世古文家。

## 历史评价
- 正面: 文学成就极高，与韩愈并称，推动古文运动；思想具有理性主义色彩，关怀民生。
- 负面: 政治生涯因参与激进改革而失败，被视为“罪臣”；部分作品被认为过于愤激或隐晦。
- 后世神话/演义: 在柳州被建庙祭祀，带有地方神化色彩；文学形象常与屈原类比，象征贬谪文人的不屈与悲愤。

## 反差 / 鲜为人知点
1. 出身显赫的河东柳氏，但在武则天时期家族遭受打击，政治地位衰落，成为“旧族”，并非一直处于权力中心。
2. 他认同佛教，认为儒释可以互补，这与通常被视为纯粹儒家士大夫的形象有所反差。
3. 其山水游记《始得西山宴游记》可能并非真实游记，而是借用道家《列子》思想的虚构寓言，旨在表达精神冥合。
