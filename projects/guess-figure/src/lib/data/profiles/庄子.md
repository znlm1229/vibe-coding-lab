# 庄子

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字子休 / 号无 / 谥号无 / 庙号无 / 别号南华真人
- 生卒年 / 朝代区间 / 籍贯: 约前369年—前286年 / 战国中期 / 宋国蒙（今河南商丘民权县一说）
- 主要职业 / 身份: 思想家、哲学家、文学家、漆园吏

## 主要事迹
- 曾任宋国蒙地漆园吏，管理漆树园 [一般]
- 拒绝楚威王聘请其为宰相的邀请，比喻宁为泥中活龟，不为庙堂卜龟 [重要]
- 与名家代表人物惠施交好，常进行哲学辩论 [重要]
- 隐居不仕，一生淡泊名利，追求精神自由 [重要]
- 著书十余万言，创作《庄子》一书，以寓言为主 [重要]
- 其思想继承并发展老子学说，后世与老子并称“老庄” [重要]
- 唐玄宗天宝初年被诏封为“南华真人”，其著作称《南华经》 [一般]
- 宋徽宗时被追封为“微妙元通真君” [次要]

## 性格 / 风格特征
- 旷达泰然，主张“安之若命”，面对无可奈何之事能安然接受
- 言辞“洸洋自恣以適己”，汪洋恣肆以适应自己的心意，不受拘束
- 清静无为，深居简出，过着隐居生活
- 对君主残暴与虚伪人性有深刻批判，具有愤世嫉俗的一面

## 典故 / 标志事件
- 拒绝楚威王相位，宁做泥中活龟。
- 与惠施辩论“子非鱼，安知鱼之乐”。
- 梦见自己化为蝴蝶，醒后不知是庄周梦蝶还是蝶梦庄周。
- 庖丁解牛，以无厚入有间，游刃有余。
- 斥儒家仁义“攖人之心”，会导致天下大乱。

## 关键作品
- 《庄子》（又称《南华经》），内篇、外篇、杂篇共33篇
- 《逍遥游》（内篇），阐述“无待”逍遥境界
- 《齐物论》（内篇），论述万物齐一、是非相对
- 《秋水》（外篇），包含与惠施的著名辩论
- 《养生主》（内篇），阐述顺其自然的养生之道

## 关系网
- 老师: 老子 (思想继承者)
- 同辈: 惠施 (好友兼辩论对手)
- 弟子: 无明确记载的弟子，但其思想影响后世道家
- 政敌: 无明确政敌，但其思想批判儒家、墨家圣贤观
- 家人: 无史料明确记载

## 历史评价
- 正面: 被尊为道家代表人物，与老子并列；其著作被誉为“中国古代精神自由史上的第一名著”；唐、宋皇帝予以封号尊崇。
- 负面: 部分观点认为其行为是愤世嫉俗的表现；其反对圣贤、仁义的思想被儒家主流批评；被视为“最早的无政府主义者”。
- 后世神话/演义: 在道教中被神化，封为“南华真人”、“微妙元通真君”；民间文学如《警世通言》有其故事演义。

## 反差 / 鲜为人知点
- 曾任“漆园吏”这一具体管理职务，并非完全脱离世俗事务。
- 其哀怨被清代学者胡文英认为甚于屈原，哀在天下与万世，而非一国一时。
- 《庄子》原书有52篇，现存33篇为晋代郭象删定后的版本，部分内容已失传。
