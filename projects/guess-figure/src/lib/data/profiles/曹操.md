# 曹操

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字孟德 / 小字阿瞒 / 谥号武皇帝 / 庙号太祖 / 无
- 生卒年 / 朝代区间 / 籍贯: 155年—220年 / 东汉末年 / 沛国谯县（今安徽亳州）
- 主要职业 / 身份: 军事家、政治家、丞相、魏王、诗人

## 主要事迹
- 早年任洛阳北部尉，棒杀违禁宦官蹇硕叔父，以严法治都。[一般]
- 189年拒绝董卓拉拢，逃离洛阳，途中误杀吕伯奢家人。[次要]
- 190年起兵参与讨伐董卓，汴水之战败于徐荣。[一般]
- 192年击败青州黄巾军，收编精锐组建“青州兵”，奠定军事基础。[重要]
- 196年迎汉献帝至许县，开始“挟天子以令诸侯”。[重要]
- 200年官渡之战击败袁绍，奠定统一北方的基础。[重要]
- 208年进位丞相，同年南征，赤壁之战败于孙刘联军。[重要]
- 211年渭南之战击破关中联军。[一般]
- 216年自魏公进爵魏王。[重要]
- 220年病逝于洛阳，其子曹丕同年建立曹魏，追尊其为魏武帝。[重要]

## 性格 / 风格特征
- 机警权数，少年时曾装病欺骗叔父，使其失宠于父亲。
- 为人多疑，行事果断，有“宁我负人，毋人负我”之言。
- 礼贤下士，重视人才，如鲍信、荀彧等皆得其重用。
- 性格放荡不羁，早年曾与袁绍一同偷新娘。

## 典故 / 标志事件
- 误杀吕伯奢家人后言“宁我负人，毋人负我”。
- 问许劭“我何如人”，得评“治世之能臣，乱世之奸雄”。
- 官渡之战以少胜多，击败袁绍。
- 赤壁之战遭火攻大败，奠定三国鼎立格局。
- 组建“青州兵”作为核心军事力量。

## 关键作品
- 注释《孙子兵法》，作《孙子略解》。
- 著有《孟德新书》（又名《兵法接要》）。
- 文学作品收录于《魏武帝集》（已佚），诗歌如《短歌行》等流传。
- 迎汉献帝，确立“挟天子以令诸侯”的政治方略。
- 统一北方，为曹魏政权奠基。

## 关系网
- 老师 / 同辈: 桥玄（赏识者）/ 袁绍（早年盟友，后期主要政敌）
- 弟子: 无明确弟子，重要部属有荀彧、郭嘉等。
- 政敌: 袁绍、刘备、孙权。
- 家人: 曹嵩（父）、曹丕（子，魏文帝）。

## 历史评价
- 正面: 杰出的军事家、政治家，统一北方，礼贤下士，诗文成就高。
- 负面: 被视为“奸雄”，多疑残忍，如徐州之战“多所残戮”。
- 后世神话/演义: 《三国演义》中塑造为“奸绝”形象，强化了其奸诈多疑的一面。

## 反差 / 鲜为人知点
- 早年曾私入宦官张让宅邸，被发现后手舞戟越墙逃脱，显示其胆大与身手。
- 少年时在家乡曾击退鳄鱼，但此事鲜少被后世提及。
- 其远祖据称为汉代相国曹参，但亦有史料称曹氏出自夏侯氏，身世存疑。
