# 老子

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字聃 / 号无 / 谥号无 / 庙号无 / 别号伯阳
- 生卒年 / 朝代区间 / 籍贯: 约前571年-前470年 / 春秋时期 / 楚国苦县厉乡曲仁里（今河南鹿邑）
- 主要职业 / 身份: 思想家、道家学派创始人、周守藏室之史（柱下史）

## 主要事迹
- 任周守藏室之史（柱下史），掌管典籍。 [一般]
- 孔子曾赴洛邑向其请教关于礼的问题，老子告诫孔子去除骄气与多欲。 [重要]
- 见周室衰微，离开东周都城。 [一般]
- 西出函谷关时，应关令尹喜之请，著写《道德经》五千言。 [重要]
- 留下著作后骑青牛而去，不知所终。 [一般]

## 性格 / 风格特征
- 主张自隐无名，以清静无为为务。
- 深藏若虚，盛德容貌若愚（孔子评价）。
- 好养精气，贵接而不施（《列仙传》记载）。

## 典故 / 标志事件
- 孔子问礼于老子。
- 紫气东来，老子出关。
- 关令尹喜强请著书，留下《道德经》。
- 骑青牛西去，不知所终。
- 被唐朝尊为先祖，立庙追尊。

## 关键作品
- 《道德经》（又称《老子》），分《道经》《德经》上下篇。
- 其学说被庄子、杨朱、列御寇等人发展。
- 开创道家无为而治、天人合一的思想体系。

- 家人：无明确记载

## 历史评价
- 正面：被尊为道家及道教奠基者，东方三大圣人之一；唐朝立道教为国教，追尊为太上玄元皇帝。
- 负面：二十世纪中叶以来，部分西方学者质疑其生平真实性，认为《道德经》可能是多人编纂、后世塑造的形象。
- 后世神话/演义：被神化为乘白马朱鬣的老叟现身预言唐室；《列仙传》等记载其神异事迹。

## 反差 / 鲜为人知点
- 姓氏存在争议，一说姓老，一说姓利，主流记载为李氏。
- 其故里“苦县厉乡曲仁里”地名存在音韵转译，“厉乡、濑乡、赖乡”实为一地异名。
- 《道德经》最古老的竹简文本（郭店楚简）仅对应现存文本约三分之一，且夹杂其他内容，表明文本可能经历过累积演变而非一次性创作。
