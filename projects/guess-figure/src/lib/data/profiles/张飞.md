# 张飞

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 益德 / 无 / 桓侯 / 无 / 无
- 生卒年 / 朝代区间 / 籍贯: 167年？ - 221年 / 东汉末年至三国时期 / 涿郡（今河北涿州）
- 主要职业 / 身份: 武将 / 蜀汉车骑将军、司隶校尉、西乡侯

## 主要事迹
- 184年于涿郡追随刘备起兵，与关羽同为别部司马，随刘备周旋征战。[一般]
- 198年随刘备依附曹操，被曹操拜为中郎将。[次要]
- 208年于当阳长坂坡据水断桥，率二十骑断后，喝退曹军追兵，掩护刘备撤退。[重要]
- 214年随诸葛亮入蜀支援刘备，在江州生擒巴郡太守严颜并义释之。[重要]
- 215年任巴西太守，于宕渠击败曹魏名将张郃，稳定巴西局势。[重要]
- 219年刘备称汉中王，拜张飞为右将军、假节。[一般]
- 221年刘备称帝，拜张飞为车骑将军、领司隶校尉，进封西乡侯。[一般]
- 221年刘备东征孙权前夕，在阆中被部将张达、范彊刺杀。[重要]

## 性格 / 风格特征
- 雄壮威猛，勇力过人，与关羽并称“万人敌”、“熊虎之将”。
- 爱敬君子（士大夫）而不恤小人（士卒），善待士人而常鞭挞麾下兵士。
- 性格暴躁，刑罚过度，刘备曾告诫其“取祸之道”。

## 典故 / 标志事件
- 据水断桥：当阳长坂坡独率二十骑断后，喝退曹军。
- 义释严颜：攻江州时生擒巴郡太守严颜，因其宁死不降而敬释放。
- 巴西破张郃：在宕渠与张郃相持五十余日，以精兵迂回破敌。
- 鞭挞士卒：常鞭打健儿却仍留其在身边，最终因此被帐下将领刺杀。
- 阆中遇害：出征东吴前，在阆中被部将张达、范彊刺杀。

## 关键作品
- 无传世文学作品或著作。
- 军事战绩：长坂坡断后、入蜀战役、巴西之战破张郃。
- 政绩：历任宜都太守、南郡太守、巴西太守，领司隶校尉。

## 关系网
- 同辈 / 盟友：关羽 (结义兄弟)、刘备 (主君兼兄弟)
- 家人：夏侯氏 (妻，夏侯渊侄女)、张苞 (长子)、张绍 (次子)、敬哀张皇后 (长女)、张皇后 (次女)
- 政敌 / 对手：张郃 (曹魏将领，巴西之战对手)

## 历史评价
- 正面：与关羽并称“万人敌”、“世之虎臣”；义释严颜有“国士之风”；程昱、周瑜等敌国谋臣将领亦盛赞其勇。
- 负面：陈寿评其“暴而无恩，以短取败”；刘备亦批评其刑罚过差、鞭挞健儿，是“取祸之道”。
- 后世神话/演义：民间艺术及《三国演义》中塑造为粗犷刚烈、好酒豪爽的形象，并列为“五虎上将”之一；后世被加封为王（如武义忠显英烈灵惠助顺王）。

## 反差 / 鲜为人知点
- 表字为“益德”（通“鷁”，一种水鸟），而非民间普遍误认的“翼德”。
- 其妻夏侯氏为曹操大将夏侯渊的侄女，是十三四岁时外出砍柴被张飞掳走纳为妻室。
- 所封“西乡侯”实为县侯（涿郡西乡县），而非乡侯，是以本土之地封赏。
