# 忽必烈

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 文武皇帝 / 世祖 / 薛禅皇帝
- 生卒年 / 朝代区间 / 籍贯: 1215年9月23日—1294年2月18日 / 元朝 (1271-1368) / 漠北草原
- 主要职业 / 身份: 元朝开国皇帝，蒙古大汗

## 主要事迹
- 1253-1254年率军远征并灭亡大理国，为后续进攻南宋建立跳板。[重要]
- 1259年蒙哥汗死后，拒绝立即北归，继续进攻南宋鄂州以图建功。[一般]
- 1260年5月在开平自立为大蒙古国皇帝，起用年号“中统”，成为首位使用汉制“皇帝”称号和年号的蒙古统治者。[重要]
- 1260-1264年与弟阿里不哥进行汗位争夺战并最终获胜，但导致蒙古帝国实质分裂。[重要]
- 1271年12月取《易经》“大哉乾元”之义，改国号为“大元”，正式建立元朝。[重要]
- 1273年攻陷襄阳，为灭宋扫清关键障碍。[一般]
- 1276年2月元军攻陷临安，宋恭帝投降，基本灭亡南宋。[重要]
- 1279年崖山海战消灭南宋最后抵抗力量，统一全国。[重要]
- 晚年两度入侵日本、三次侵略越南均告失利，国力受损。[次要]

## 性格 / 风格特征
- 仁明英睿，事太后至孝，尤善抚下（源自史料记载）。
- 重视汉地治理，早年便积极招揽汉族幕僚儒士，形成“金莲川幕府”。
- 晚年因丧妻丧子意志消沉，开始酗酒，饱受痛风与肥胖困扰。

## 典故 / 标志事件
- 建立金莲川幕府，招揽大批汉族儒士。
- 接受“儒教大宗师”称号。
- 与阿里不哥进行四年汗位争夺战。
- 改国号为“大元”，建立元朝。
- 崖山海战灭亡南宋最后抵抗力量。

## 关键作品
- 建立元朝中央集权官制（中书省、枢密院、御史台等）。
- 兴建大都（今北京）。
- 修凿运河。
- 颁布《建国号诏》、《归附安民诏》等重要诏书。
- 统治期间将云南地区纳入中国版图并长期管辖。

## 关系网
- 老师 / 同辈: 汉儒刘秉忠、姚枢等（幕僚/老师）；阿里不哥（弟/政敌）；旭烈兀（弟/支持者）
- 弟子 / 政敌: 无明确弟子记载；阿里不哥（主要政敌）
- 家人: 父亲拖雷，母亲唆鲁禾帖尼，兄蒙哥，妻察必皇后，子真金，孙铁穆耳（元成宗）

## 历史评价
- 正面: 雄才大略，统一中国，结束长期分裂局面；推行汉化，建立制度，加强中央集权。
- 负面: 晚年远征失利，耗损国力；汗位争夺导致蒙古帝国分裂；统治后期民生困苦，内部斗争严峻。
- 后世神话/演义: 在部分文学作品和西方记述中被描绘为富有且权力无限的东方君主形象。

## 反差 / 鲜为人知点
- 他曾被儒士元好问和张德辉请求并接受了“儒教大宗师”的称号。
- 在攻打鄂州期间，其妻察必派使者密报阿里不哥的夺权行动，催促他北归，这是他决策班师的关键因素之一。
- 晚年因痛风和肥胖健康恶化，且因爱妻察必和爱子真金先后去世而悲痛消沉，过度饮酒。
