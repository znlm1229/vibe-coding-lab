# 刘秀

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字文叔 / 无 / 光武 / 世祖 / 铜马帝
- 生卒年 / 朝代区间 / 籍贯: 前5年1月15日—57年3月29日 / 西汉末年至东汉初年 / 南阳郡蔡阳县（今湖北枣阳）
- 主要职业 / 身份: 东汉开国皇帝

## 主要事迹
- 地皇三年（22年）与其兄刘縯在宛城起兵反新莽 [重要]
- 更始元年（23年）指挥昆阳之战，以少胜多击败王莽主力 [重要]
- 更始三年（25年）于鄗城登基称帝，建立东汉，改元建武 [重要]
- 建武年间逐步扫平赤眉、张步、隗嚣等割据势力，统一中国 [重要]
- 建武十二年（36年）平定公孙述成家政权，完成全国统一 [重要]
- 在位期间多次发布释放奴婢诏书，减轻赋税，兴修水利，史称“光武中兴” [一般]
- 建武中元二年（57年）授予倭奴国“汉委奴国王”金印 [一般]
- 统一后“偃武修文”，削弱国防，罢郡国都尉，导致地方豪强武装发展 [次要]

## 性格 / 风格特征
- 待人诚恳简约，宽厚有信，能令窦融、马援等对手归心。
- 勤于政事，“每旦视朝，日仄乃罢，数引公卿郎将议论经理，夜分乃寐”。
- 处事谨慎，兄长刘縯被杀后，能隐忍谢罪，饮食言笑如常以自保。
- 统一后“厌武事，不言军旅”，主张“柔道”治国。

## 典故 / 标志事件
- 昆阳之战：以三千绿林军击败王莽万余主力。
- 铜马帝：在河北大破铜马等割据势力后，被关西人称为“铜马帝”。
- 仕宦当作执金吾，娶妻当得阴丽华：早年于长安所见所愿。
- 云台二十八将：其子汉明帝将功劳最大的二十八将画像于云台阁。
- 汉委奴国王金印：授予倭奴国国王金印，为中日交往早期证明。

## 关键作品
- 建立东汉王朝，实现“光武中兴”。
- 裁并四百余县，精简官员，改革行政。
- 引南匈奴内迁入塞，安置于北方缘边八郡（影响深远）。
- 削弱三公权力，扩大尚书台职权，强化皇权集中。
- “偃武修文”政策，奠定东汉初期文治基调。

## 历史评价
- 正面：被誉为宽厚简易，能不计前嫌招降对手；善于“退功臣而进文吏”，实现“光武中兴”；被赞“明慎政体，总揽权纲”。
- 负面：统一后削弱国防，废郡国兵制，导致后期无力抵御外患，豪强地主武装发展；晚年“法理严察”，对大臣“捶扑牵曳”，借故搜捕王侯宾客致数千人死亡。
- 后世神话/演义：出生时有“赤光照耀整个房间”，嘉禾“一茎九穗”；戏剧中有诛杀功臣的“不白之冤”（实则未杀）。

## 反差 / 鲜为人知点
- 其父刘钦为济阳县令，因其住宅低湿，刘秀出生于汉武帝行宫的后殿中。
- 早年勤于农事，常被爱好游侠的兄长刘縯取笑，比作刘邦的兄弟刘喜（不善进取）。
- 统一后，虽废郭皇后及太子刘彊，但能让郭皇后到其子封国安享余年，两子间不生嫌隙，此事未受后人过多议论。
