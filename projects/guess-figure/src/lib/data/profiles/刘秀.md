# 刘秀

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字文叔 / 谥号光武 / 庙号世祖 / 别号铜马帝
- 生卒年 / 朝代区间 / 籍贯: 前5年1月15日 - 57年3月29日 / 西汉末至东汉初 / 南阳郡蔡阳县（今湖北枣阳）
- 主要职业 / 身份: 东汉开国皇帝

## 主要事迹
- 地皇三年（22年）与兄刘縯在南阳宛城起兵反莽 [重要]
- 更始元年（23年）昆阳之战，以少胜多击败王莽主力 [重要]
- 兄长刘縯被害后，隐忍谢罪，获封破虏大将军、武信侯 [一般]
- 更始三年（25年）于鄗城称帝，改元建武，建立东汉 [重要]
- 建武年间逐步扫平赤眉、张步、隗嚣等割据势力，统一全国 [重要]
- 建武十二年（36年）平定公孙述，完成统一 [重要]
- 在位期间释放奴婢、精简郡县、发展农业，史称“光武中兴” [重要]
- 建武中元二年（57年）倭奴国使者来朝，赐“汉委奴国王”金印 [一般]

## 性格 / 风格特征
- 勤于稼穑，早年曾被兄长讥笑像刘邦的兄弟刘喜（只知务农）
- 待人诚恳简约，宽厚有信，能使窦融、马援等归心
- 统一后“知天下疲耗”，厌武事，不言军旅，拒绝臧宫、马武请击匈奴之议
- 每旦视朝，日昃乃罢，夜分乃寐，勤于政事

## 典故 / 标志事件
- 昆阳之战：以数千兵对阵王莽数十万大军，大获全胜
- 推赤心置人腹中：收降铜马军后，轻骑巡营以安降者之心
- 得陇望蜀：平定陇右后，方图蜀地
- 投戈讲艺，息马论道：统一后偃武修文
- 倭奴国王金印：赐予日本使者，现存“汉委奴国王”金印实物

## 关键作品
- 建立东汉王朝，开创“光武中兴”局面
- 释放奴婢、抑制豪强、精简行政的系列政令
- 引南匈奴内迁安置的边防政策
- 确立“事归台阁”，强化尚书台权力的中央集权制度
- 其治国方略被概括为“以柔道行之”

## 关系网
- 家人：兄刘縯（被更始帝刘玄杀害）、皇后阴丽华、郭圣通
- 同辈/盟友：更始帝刘玄（初期共主，后成对手）、邓禹、吴汉、耿弇
- 对手/敌人：王莽、赤眉军（立刘盆子）、隗嚣、公孙述
- 归附者：窦融、马援（因其待人宽厚而归心）

## 历史评价
- 正面：宽厚简易，能不计前嫌招降对手；统一后偃武修文，勤政爱民；开创中兴局面。
- 负面：削弱国防，废郡国兵制，导致后期边防无力；晚年“灾变不息”却拒击匈奴，被批评为消极。
- 后世神话/演义：云台二十八将的传说；其起兵符瑞“刘秀发兵捕不道”的谶纬故事。

## 反差 / 鲜为人知点
- 起兵前曾“卖谷于宛”，是个从事粮食贸易的地方豪族，并非纯粹农民。
- 昆阳之战后，其兄刘縯被杀，他不仅不敢服丧，还向更始帝谢罪，“饮食言笑如平常”，隐忍至极。
- 晚年严厉抑制外戚功臣，建武二十八年（52年）借故搜捕王侯宾客，“坐死者数千人”，与其宽厚形象形成反差。
