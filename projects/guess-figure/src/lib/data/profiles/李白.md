# 李白

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字太白，号青莲居士，人称“诗仙”
- 生卒年 / 朝代区间 / 籍贯: 701年5月19日—762年11月30日 / 唐朝 / 籍贯有争议（一说山东，一说陇西成纪，一说蜀地）
- 主要职业 / 身份: 诗人、官员（翰林供奉）

## 主要事迹
- 早年于蜀中成长，博览百家，喜好剑术与任侠，自称曾亲手杀人。[一般]
- 724年出蜀漫游，干谒官员以求荐拔，但屡次碰壁。[一般]
- 742年因名动京师（或得道士吴筠、玉真公主推荐），受唐玄宗征召，任翰林供奉，成为宫廷诗人。[重要]
- 在宫廷期间，曾乘醉令宦官高力士脱靴，并因诗作被杨贵妃排斥，744年请辞离开长安。[重要]
- 744年在汴州、宋州一带与杜甫、高适交游，留下文学佳话。[重要]
- 755年安史之乱爆发后，756年应永王李璘之召入幕。[重要]
- 永王兵败后，李白被牵连下狱，后流放夜郎，途中遇赦。[重要]
- 晚年投效李光弼军未果，762年病逝于当涂。[一般]

## 性格 / 风格特征
- 志气宏放，飘然有超世之心（《旧唐书》载）。[源自史料]
- 个性热烈，豪迈不羁，恃恩放恣（如醉登严武床榻）。[源自史料]
- 狂率不逊，如干谒韩朝宗时“醉酒失态”为借口。[源自史料]
- 轻财好施，与任侠之徒结交，自称曾亲杀人。[源自史料]

## 典故 / 标志事件
- 力士脱靴：醉后令高力士脱靴，得罪权贵。
- 谪仙人：贺知章读其诗后，称其为“天上谪仙人”。
- 竹溪六逸：与孔巢父等五人隐居徂徠山，酣歌纵酒。
- 天子呼来不上船：杜甫诗描述其醉态，天子召唤亦不上船。
- 应永王聘：安史之乱中应召入永王李璘幕府，后因此获罪。

## 关键作品
- 《蜀道难》：乐府歌行体代表作，想象奇崛。
- 《将进酒》：抒发豪放情怀与人生感慨的名篇。
- 《静夜思》：简洁深远的思乡绝句。
- 《早发白帝城》：遇赦后所作，轻快飘逸。
- 《黄鹤楼送孟浩然之广陵》：深情绵邈的送别诗。

- 同辈/友人：杜甫（诗友）、孟浩然（诗友）、高适（诗友）、元丹丘（道友）

## 历史评价
- 正面：诗仙，浪漫主义代表，与杜甫并称盛唐最伟大诗人，对后世影响巨大。
- 负面：自负放达，讥杜甫“龌龊”；政治识见浅，卷入永王叛乱。
- 后世神话/演义：形象被进一步浪漫化，如“骑鲸捉月”等传说。

## 反差 / 鲜为人知点
- 李白可能并非汉族，其家世有西域胡人、中亚华侨等争议说法，父亲名“李客”可能非汉名。[隐晦信息]
- 因家世或户籍问题，可能无法参加科举考试，故一生依赖干谒与荐举入仕。[隐晦信息]
- 晚年曾试图投效李光弼军队以建功立业，但因病半途而返，显示其晚年仍有强烈用世之心。[隐晦信息]
