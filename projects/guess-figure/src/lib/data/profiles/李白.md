# 李白

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 太白 / 青莲居士 / 谪仙人 / 无 / 诗仙
- 生卒年 / 朝代区间 / 籍贯: 701年5月19日 - 762年11月30日 / 唐朝 / 蜀地 (具体出生地有西域碎叶、四川彰明等多说)
- 主要职业 / 身份: 诗人、翰林供奉 (宫廷诗人)

## 主要事迹
- 724年秋出蜀漫游，干谒官员求荐，未果 [一般]
- 742年秋因名动京师 (一说受道士吴筠、玉真公主推荐) 被唐玄宗征召，任翰林供奉 [重要]
- 743-744年在长安宫廷期间，曾乘醉命高力士脱靴，遭谗言中伤，逐渐失宠 [重要]
- 744年三月请辞“还山”，离开长安，开始十年漫游生涯 [重要]
- 744年在齐州老子庙从道士高如贵受道箓 [一般]
- 755年安史之乱爆发后南奔，756年应永王李璘之聘加入其幕下 [重要]
- 757年初永王兵败，李白在彭泽自首，被拘于寻阳狱中；后遇赦 [重要]
- 758年初被判流放夜郎，途中遇赦而还 [重要]
- 762年病逝于当涂，葬青山之南 [一般]

## 性格 / 风格特征
- 狂放不羁，嗜酒纵情 (史料载“既嗜酒，日与饮徒醉于酒肆”、“傲诞如此”)
- 自负才高，有经世济民之志，亦怀出世隐居之愿
- 轻财好施，早年有任侠格斗之事，自称曾亲手杀人

## 典故 / 标志事件
- 力士脱靴 (乘醉令高力士脱靴，得罪权贵)
- 谪仙人 (贺知章赞其为“谪仙人”)
- 竹溪六逸 (与孔巢父等五人隐居徂徠山，酣歌纵酒)
- 供奉翰林 (被玄宗征召为宫廷诗人)
- 流放夜郎 (因参与永王李璘事被流放，途中遇赦)

## 关键作品
- 《静夜思》
- 《将进酒》
- 《蜀道难》
- 《行路难》
- 《早发白帝城》

## 关系网
- 老师 / 同辈: 无明确记载的老师 / 杜甫 (诗友)、孟浩然 (诗友)、贺知章 (赏识者)
- 弟子: 无
- 政敌: 高力士 (宦官，遭其非难)、杨贵妃 (因诗比喻得罪)
- 家人: 父亲李客 (记载模糊)、妻子许氏 (前宰相许圉师孙女)、妻子宗氏 (宗楚客后人)

## 历史评价
- 正面: 诗才天纵，飘逸奔放，想象大胆，富于独创性，盛唐最伟大诗人之一，对后世影响巨大。
- 负面: 性格“褊躁无器度”，恃恩放恣，政治参与(如入永王幕)被视为不慎。
- 后世神话/演义: 被尊为“诗仙”，形象浪漫化，饮酒、骑鲸、捉月等传说广为流传。

## 反差 / 鲜为人知点
- 李白曾自称亲手杀人，早年有任侠格斗经历，并非纯粹的文弱书生。
- 李白家庭可能为西域胡人或商人背景(“胡客”)，甚至有学者认为其有波斯或突厥血统，这使其无法参加科举。
- 李白晚年试图投效李光弼军中报国，但半途因病而回，其“愿格杀猛兽以自赎”的记载也显示其并非仅懂诗文。
