# 白居易

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字乐天 / 号香山居士 / 谥号文 / 别号醉吟先生
- 生卒年 / 朝代区间 / 籍贯: 772年2月28日—846年9月8日 / 唐朝 / 祖籍太原，生于河南新郑
- 主要职业 / 身份: 文学家、诗人、官员

## 主要事迹
- 贞元十六年进士及第，授秘书省校书郎。[一般]
- 元和元年授左拾遗，频繁上书言事，创作大量讽喻诗。[重要]
- 元和十年，宰相武元衡遇刺，白居易上表主张严缉凶手，被认为越职言事，加之被诽谤“有害名教”，贬为江州司马。[重要]
- 长庆二年任杭州刺史，疏浚六井、修筑西湖堤防。[重要]
- 长庆四年任苏州刺史，开凿山塘河（七里山塘）。[一般]
- 晚年定居洛阳履道里，与刘禹锡唱和，号“香山居士”、“醉吟先生”。[一般]
- 会昌二年以刑部尚书致仕，晚年出资开凿龙门石滩以利舟行。[次要]

## 性格 / 风格特征
- 直言敢谏，事无巨细皆上书言事，甚至令宪宗感到不快。
- 性情豁达，虽遭贬谪，仍能恬然自处，寄情山水诗酒。
- 笃信佛教，晚年尤甚，与僧人交游，为诗僧如满弟子。

## 典故 / 标志事件
- 顾况调侃其名“米价方贵，居亦弗易”，见诗后改口“居即易矣”。
- 因母看花坠井去世，被诬以作《赏花》《新井》诗“有害名教”而贬江州。
- 与元稹、白行简相遇于黄牛峡，三人同游之处被称为“三游洞”。
- 在忠州城东山坡种花，命名此地为“东坡”。
- 晚年于洛阳举行“七老会”，并合僧如满、李元爽画成“九老图”。

## 关键作品
- 《长恨歌》
- 《琵琶行》
- 《与元九书》
- 《秦中吟》及《新乐府》系列讽喻诗
- 《白氏长庆集》（文集）

## 关系网
- 老师 / 同辈 / 弟子 / 政敌 / 家人: 顾况（赏识者）/ 元稹（挚友，文学伙伴）、刘禹锡（晚年唱和好友）/ 无明确记载弟子 / 执政中恶其言事者（如王涯）/ 父白季庚、弟白行简、妻杨氏（杨汝士妹）

## 历史评价
- 正面: 文章精切，诗作平易近人、通俗易懂，有“老妪能解”之说；倡导新乐府运动，主张“文章合为时而著，歌诗合为事而作”；在世时作品已广为流传，影响深远至日本、新罗。
- 负面: 执政者恶其言事，认为其“浮华无行”；部分朝臣不满其频繁上疏、多管闲事的行为。
- 后世神话/演义: 被唐宣宗褒为“诗仙”（敕封诗仙），后世民间更将李白尊为“诗仙”；西湖白堤被后世误传为其修筑，实为“白沙堤”。

## 反差 / 鲜为人知点
- 白居易母亲陈氏与白家本有血亲关系（外祖母亦白氏出身）。
- 部分近代学者认为白居易可能是胡人后裔，一说龟兹王族后裔。
- 其子白景受为过继的侄子（兄白幼文之子），而非亲生。
