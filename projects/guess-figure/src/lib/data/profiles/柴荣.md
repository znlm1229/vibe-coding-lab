# 柴荣

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 睿武孝文 (谥号简称) / 世宗 / 无
- 生卒年 / 朝代区间 / 籍贯: 921年10月27日—959年7月27日 / 五代后周 (954-959在位) / 邢州尧山柴家庄 (今河北隆尧)
- 主要职业 / 身份: 皇帝 (后周第二位皇帝)

## 主要事迹
- 954年即位后，招抚流亡、减少赋税、整顿吏治，复苏民生。[一般]
- 954年高平之战，力排众议亲征，大破北汉与辽联军，稳定新立政权。[重要]
- 955年推行“显德毁佛”，废寺院、令僧尼还俗，以铜材铸“周元通宝”。[重要]
- 955年后，采纳“先南后北”策略，征伐后蜀、南唐，拓展疆土至两淮。[重要]
- 在位期间制定并推行均田政策，缓和社会矛盾。[一般]
- 整军练卒，创建殿前诸班精锐，革除冗兵积弊。[一般]
- 959年北伐，连克瀛、莫二州及三关，图收复燕云十六州，因病中止。[重要]

## 性格 / 风格特征
- 性沉重寡言，但果断刚毅，敢于力排众议 (如高平之战)。[源自史料]
- 躬亲实践，不偷安，常身先士卒、亲临战阵督战。[源自史料]
- 务实利民，毁佛时称“若朕身可以济民，亦非所惜也”。[源自史料]

## 典故 / 标志事件
- 高平之战亲冒矢石督战，大破北汉。
- 推行“显德毁佛”，铸周元通宝钱。
- 制定“十年开拓天下，十年养百姓，十年致太平”的战略规划。
- 北伐燕云，连克二州三关，因病功败垂成。
- 以养子（外戚）身份继承帝位，为历代唯一。

## 关键作品
- 均田政策 (诏令均田，缓和矛盾)。
- 周元通宝 (毁佛铸钱，钱质铸量居五代之冠)。
- 《制旨兵法》 (命兵部撰集)。
- “先南后北”统一方略 (基于《平边策》确定)。
- 殿前诸班精锐 (整军成果，所向皆捷)。

## 关系网
- 老师 / 同辈 / 弟子 / 政敌 / 家人: 郭威 (养父/姑丈) / 冯道 (劝阻北伐的同僚) / 王朴 (献策大臣) / 刘崇 (北汉世祖，政敌) / 柴宗训 (子，周恭帝)

## 历史评价
- 正面: 《旧五代史》评其为“神武雄略，乃一代之英主”；司马光赞其毁佛是“不爱己身而爱民”。
- 负面: 毁佛行为被列入“三武一宗”，受到部分佛教徒批评。
- 后世神话/演义: 《水浒传》中柴进被描写为其嫡派子孙；漫画《天子传奇5》将其塑造为僧人玄武还俗。

## 反差 / 鲜为人知点
- 早年曾随商人颉跌氏在江陵贩卖茶叶，有经商经历。
- 后汉隐帝诛杀郭威家属时，其最年长的三个儿子亦同时遇害。
- 其曾孙柴叔夏在南宋绍兴五年（1135年）袭封崇义公，后世有录。
