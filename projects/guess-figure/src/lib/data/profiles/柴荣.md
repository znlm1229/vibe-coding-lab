# 柴荣

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 睿武孝文 / 世宗 / 无
- 生卒年 / 朝代区间 / 籍贯: 921年10月27日—959年7月27日 / 五代后周 / 邢州尧山柴家庄（今河北隆尧）
- 主要职业 / 身份: 后周皇帝

## 主要事迹
- 954年，北汉联合辽兵南下，柴荣力排众议，亲征高平并大破敌军，稳定政局。[重要]
- 954年即位后，招抚流亡、减少赋税、整顿吏治，使民生复苏、政治清明。[一般]
- 955年，推行“显德毁佛”，废寺院、毁铜像铸“周元通宝”，以解决钱荒。[重要]
- 955年，诏令群臣献《平边策》，采纳王朴“先南后北”的统一战略。[一般]
- 956-958年，三次亲征南唐，创建水军，攻取淮南十四州，迫使南唐求和。[重要]
- 957年，阅读唐元稹《均田图》，下诏推行均田政策，意图解决土地问题。[一般]
- 959年，北伐契丹，连克瀛、莫二州及三关（益津、瓦桥、高阳），准备收复幽州。[重要]
- 959年，因病中止北伐，返回京师后不久去世，年仅七岁的儿子柴宗训继位。[次要]

## 性格 / 风格特征
- 性沉重寡言，但处事果断，如高平之战亲临阵前督战。[源自史料]
- 务实敢为，不惜毁佛铸钱，并言“若朕身可以济民，亦非所惜也”。[源自史料]
- 虚心听纳，用人不疑，如采纳王朴战略，并保存李谷献策锦囊。[源自史料]

## 典故 / 标志事件
- 高平之战亲冒矢石督战。
- 推行“显德毁佛”以铜像铸钱。
- 采纳王朴“先南后北”统一方略。
- 阅读元稹《均田图》并诏令推行。
- 北伐契丹连克二州三关。

## 关键作品
- 铸行“周元通宝”（货币政策）。
- 推行“显德毁佛”诏令（宗教/经济政策）。
- 颁布均田诏令（土地政策）。
- 命兵部撰集《制旨兵法》（军事著作）。
- 考制度、修通礼、定正乐、议刑统（制度建设）。

## 历史评价
- 正面: 旧五代史评价为“神武雄略，乃一代之英主”，司马光赞其毁佛是“不爱己身而爱民”。
- 负面: 史家曾讥其北伐“轻社稷之重，而侥幸一胜于仓卒”。
- 后世神话/演义: 《水浒传》中柴进被描写为其嫡派子孙；京剧《三打陶三春》等文艺作品有其形象。

## 反差 / 鲜为人知点
- 早年曾随商人颉跌氏在江陵贩卖茶叶，有经商经历。
- 是郭威正室柴皇后的侄子，以外戚身份继承帝位，在中国历朝中唯一。
- 北伐契丹时，莫州刺史刘楚信、瀛州刺史高彦晖投降，兵不血刃取二州。
