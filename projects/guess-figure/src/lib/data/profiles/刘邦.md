# 刘邦

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字季 / 谥号高皇帝 / 庙号太祖 / 无 / 无
- 生卒年 / 朝代区间 / 籍贯: 前256年（一说前247年）—前195年6月1日 / 秦末汉初 / 沛丰邑中阳里（今江苏丰县）
- 主要职业 / 身份: 军事家、政治家、汉朝开国皇帝

## 主要事迹
- 前209年，释放押解的犯人，于芒砀山落草，后响应陈胜起义，据沛县反秦，被推为沛公。 [重要]
- 前208年，投奔项梁，获兵卒支持，反攻丰邑。 [一般]
- 前208年闰九月，受楚后怀王命西征攻秦，约定“先入关中者王之”。 [重要]
- 前207年，率军西进，沿途收编、破秦军，采纳张良建议先破宛城，后所过城邑多降。 [重要]
- 前206年，率先攻入关中，接受秦王子婴投降，但随后项羽入关，刘邦被分封为汉王，领巴、蜀、汉中。 [重要]
- 前206年至前202年，与项羽进行楚汉战争，历经多次战役，最终于垓下之战击败项羽。 [重要]
- 前202年，称帝，定都长安，建立汉朝。 [重要]
- 在位期间，实行休养生息政策，平定异姓诸侯王叛乱，如韩信、彭越等。 [一般]

## 性格 / 风格特征
- 豁达大度，善于结交和用人，能从底层吸纳人才如萧何、曹参、樊哙等。
- 有狡黠的一面，如赴吕公宴时自称“贺钱万”实则空手，以计谋争取利益。
- 早年仰慕信陵君，有游侠气，但也能在关键时刻展现果断，如斩白蛇起义、释放囚徒。
- 对天命之说敏感且利用，如“赤帝子斩白帝子”传说为其所用，以凝聚人心。

## 典故 / 标志事件
- 斩白蛇起义：路遇白蛇斩之，老妇称其为赤帝子杀白帝子，被视为取代秦朝的天命象征。
- 贺钱万：赴吕公宴时空手称献万钱，被吕公看重面相，得娶吕雉。
- 约法三章：入关中后废秦苛法，仅定“杀人者死，伤人及盗抵罪”三章以安民。
- 鸿门宴：项羽设宴于鸿门，刘邦险遭刺杀，后借机脱身。
- 白登之围：汉初率军亲征匈奴，于白登山被围七日，后用计脱险。

## 关键作品
- 建立汉朝，奠定两汉四百年基业。
- 推行“休养生息”政策，减轻赋税，恢复社会经济。
- 制定《汉律》九章，简化秦法，稳定社会秩序。
- 分封与郡县并行，初步确立中央集权与地方分权的格局。

## 关系网
- 老师 / 同辈：张耳（早年结交，曾为其门客） / 项羽（主要政敌，楚汉战争对手） / 萧何（同辈，重要辅臣）
- 弟子 / 政敌：无明确弟子 / 项羽（主要政敌） / 韩信、彭越（后期平定的主要异姓诸侯王）
- 家人：吕雉（妻子，皇后） / 刘盈（儿子，汉惠帝） / 刘肥（长子，外妇曹氏所生）

## 历史评价
- 正面：被视为善于用人、能从布衣崛起开创帝业的杰出君主，司马迁评其“性明达，好谋，能听”。
- 负面：被认为有狡诈、权谋的一面，为巩固权力诛杀功臣（如韩信、彭越），有时手段残酷。
- 后世神话/演义：在民间传说和《史记》中被赋予“赤帝子”等神话色彩，成为天命所归的象征；后世戏曲、小说常渲染其从平民到皇帝的传奇经历。

## 反差 / 鲜为人知点
- 刘邦早年曾为信陵君门客张耳的宾客，并终生敬仰信陵君，称帝后途经大梁常祭祀之。
- 其出生年份有前256年和前247年两种说法，学界仍有争议，并非确定无疑。
- 据《汉纪》等记载，其家族可能原为魏国（或晋国）低阶贵族，并非纯粹平民，在魏亡后迁居丰邑。
