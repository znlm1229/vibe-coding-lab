# 孔子

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字仲尼 / 号孔夫子 / 谥号文宣王 / 庙号无 / 别号至圣
- 生卒年 / 朝代区间 / 籍贯: 前551年-前479年 / 春秋时期 / 鲁国陬邑（今山东曲阜）
- 主要职业 / 身份: 思想家、教育家、儒家学派创始人

## 主要事迹
- 早年从事委吏、乘田等小吏，管理仓储畜牧 [一般]
- 51岁仕鲁，任中都宰、司空，后升任大司寇，行摄相事 [重要]
- 策划实施“隳三都”行动，试图削弱鲁国三桓势力 [重要]
- 因政治分歧离开鲁国，率弟子周游列国十四年 [重要]
- 在匡地被误认为阳虎遭围捕，在陈蔡之间被围困绝粮七日 [一般]
- 晚年被迎回鲁国尊为国老，专注于教育、古籍整理与编修《春秋》 [重要]

## 性格 / 风格特征
- 身材高大（约九尺六寸），被描述为“长人”
- 好学不倦，闻韶乐三月不知肉味
- 身处困境（如陈蔡绝粮）时仍“讲诵不绝”，镇定自若
- 对礼仪事务有浓厚兴趣，幼时便常摆设礼器练习行礼

## 典故 / 标志事件
- 周游列国时在郑国与弟子失散，被人形容为“丧家之犬”
- 路过宋国时，司马桓魋欲杀孔子，拔除其习礼的大树
- 夹谷之会，成功说服齐国归还侵占鲁国的汶阳等地
- 因卫灵公与夫人南子同车招摇过市而感到耻辱，离开卫国
- 晚年得知齐国陈恒弑君后，斋戒沐浴三天，恳请鲁哀公伐齐

## 关键作品
- 《论语》（言论汇编）
- 编修《春秋》
- 倡导并实践“有教无类”、“因材施教”的教育理念
- 确立以“仁”、“礼”为核心的儒家思想体系

## 关系网
- 老师: 郯子（请教）、师襄子（学琴）、苌弘（访乐）
- 同辈 / 政敌: 季孙氏、叔孙氏、孟孙氏（鲁国三桓，政治对手）、阳虎（季孙氏家臣，曾训斥拒绝孔子）
- 弟子: 颜回、子贡、子路、冉雍、曾参等
- 家人: 父亲叔梁纥、母亲颜徵在、妻子亓官氏、儿子孔鲤（字伯鱼）

## 历史评价
- 正面: 被尊为“至圣先师”、“万世师表”，其思想塑造了中国乃至东亚的文化伦理基础。
- 负面: 部分行动（如“隳三都”）被视为挑战现有贵族权力结构，导致政治失败；后世也有对其诛杀少正卯等事件的争议记载。
- 后世神话/演义: 被神格化为“太极上真九疑主宰文昌皇人玄圣道君”、“东海广桑山真君”等道教神祇称号；地位日益崇高，成为“智慧之神”。

## 反差 / 鲜为人知点
- 孔子父亲叔梁纥七十二岁时娶十八岁的颜徵在，因年龄悬殊且礼仪不合，史书称其为“野合”。
- 孔子早年生活困苦，曾从事多种基层工作（如管理仓库和畜牧），并非天生贵族。
- 其身高记载存在换算差异，按周制推算约186公分，按战国制推算则高达221.76公分，后者远超常人。
