# 朱熹

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字元晦 / 号晦庵 / 谥文 (文公) / 无 / 别号紫阳先生
- 生卒年 / 朝代区间 / 籍贯: 1130年10月18日 - 1200年4月23日 / 南宋 / 籍贯徽州婺源 (今江西婺源)，出生于福建尤溪
- 主要职业 / 身份: 理学家、思想家、教育家、官员

## 主要事迹
- 绍兴十八年 (1148年) 十九岁中进士，初任泉州同安县主簿，推行教化 [一般]
- 乾道年间，屡次拒绝朝廷征召，以“安贫守道”闻名 [次要]
- 淳熙五年 (1178年) 知南康军，修复白鹿洞书院，订立《白鹿洞书院教条》 [重要]
- 淳熙八年 (1181年) 提举浙东常平茶盐公事，单车巡访，赈济灾荒，革除弊政 [重要]
- 淳熙九年 (1182年) 六次上奏弹劾知台州唐仲友，引发“唐仲友案” [重要]
- 绍熙五年 (1194年) 任焕章阁待制兼侍讲，为宁宗皇帝讲学仅四十日 [一般]
- 庆元二年 (1196年) 遭监察御史沈继祖弹劾，学说被斥为“伪学”，卷入“庆元党禁” [重要]
- 庆元六年 (1200年) 病逝于建阳考亭沧州精舍 [一般]

## 性格 / 风格特征
- 耿直敢言，屡次上书直言批评皇帝“心术”与朝廷弊政，甚至言“莫大之祸，近在朝夕”
- 为政务实，救灾时“日钩访民隐，单车屏徒从”，革除丁钱、和买等不便民之政
- 学术上坚持己见，与林栗论《易》、《西铭》不合，遭其弹劾“本无学术”

## 典故 / 标志事件
- 幼年于沙上画八卦，显示早慧。
- 修复白鹿洞书院并订立著名学规。
- 六上奏章弹劾唐仲友，引发官场风波。
- 卷入庆元党禁，学说被斥为“伪学”。
- 临终前自称“头常如黏在颈上”，形容处境危殆。

## 关键作品
- 辑定并注释《大学》、《中庸》、《论语》、《孟子》，合为《四书》
- 撰《朱子语类》（语录汇编）
- 撰《四书章句集注》（成为后世科举标准）
- 修复白鹿洞书院并订立其学规（影响后世书院教育）
- 建立“理气二元论”哲学体系（程朱理学集大成）

## 关系网
- 老师: 李侗 (程颐三传弟子)
- 同辈: 吕祖谦 (学术交流，曾邀其讲学)、陆九渊 (学术论敌)
- 弟子: 黄榦 (女婿及重要传人)
- 政敌: 韩侂胄 (庆元党禁主导者)、唐仲友 (弹劾对象)
- 家人: 父朱松 (进士)、妹朱心、长子朱塾

## 历史评价
- 正面: 被尊为“朱子”，誉为“万世宗师”，理学集大成者，其学说被奉为官方哲学。
- 负面: 曾被斥为“伪学”，弹劾者指控其“诱引尼姑”、“冢妇不夫而自孕”等私德问题。
- 后世神话/演义: 明代小说《二刻拍案惊奇》有“硬勘案大儒争闲气”篇，描写其弹劾唐仲友及审讯官妓严蕊之事。

## 反差 / 鲜为人知点
- 早年曾研修佛学，后遇李侗才尽弃佛学，回归儒家。
- 相信鬼神并曾多次祷雨，甚至设醮祈禳，但日常不肯入庙宇，认为不合义理。
- 1195年曾欲上万言书声援赵汝愚，因占卜结果不利而听从弟子劝阻放弃上奏。
