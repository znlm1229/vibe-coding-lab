# 杨广

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字阿𡡉 / 谥号炀帝 / 庙号世祖 / 别号晋王
- 生卒年 / 朝代区间 / 籍贯: 569年—618年 / 隋朝 / 华阴（今陕西华阴）
- 主要职业 / 身份: 皇帝 / 军事统帅

## 主要事迹
- 开皇八年（588年）冬，以晋王身份挂帅灭陈，统一中国 [重要]
- 开皇二十年（600年），取代兄长杨勇被立为太子 [重要]
- 仁寿四年（604年）七月，即位为隋朝第二位皇帝 [重要]
- 大业元年（605年）起，营建东都洛阳并开凿大运河 [重要]
- 在位期间正式确立科举制度为国家政策 [重要]
- 大业八年（612年）起，三次大规模征讨高句丽，均告失败 [重要]
- 大业五年（609年），亲征平定吐谷浑，设置西海等四郡 [一般]
- 大业十一年（615年），北巡长城被突厥围困于雁门 [一般]
- 统治后期引发大规模隋末民变 [重要]
- 大业十四年（618年），在江都兵变中被宇文化及军队弑杀 [重要]

## 性格 / 风格特征
- 早年作风简朴、礼贤下士，以博取声望与文帝欢心
- 好大喜功，热衷于大规模工程与远征以彰显国力
- 后期骄奢淫逸，沉溺于巡游与享乐，逃避现实政务
- 急功近利且不容异议，曾因高颎、贺若弼议论奢靡而杀之

## 典故 / 标志事件
- 灭陈后欲纳陈后主宠妃张丽华，被高颎斩杀
- 开凿大运河，动用数百万民力，世界史上最长运河
- 三游江都，船队奢华绵长，沿途州县贡献食物
- 三征高句丽失败，耗尽国力，成为民变导火线
- 江都兵变前引镜自照，预感末日，言“好头颈，谁当斫之！”

## 关键作品
- 隋唐大运河（开创性工程，后演变为京杭大运河）
- 确立科举制度为国家政策，影响后世历代
- 营建东都洛阳，作为政治军事经济中心
- 亲征平定吐谷浑，拓疆设置四郡
- 著有诗文（如《春江花月夜》等，文学上有造诣）

## 关系网
- 父母: 隋文帝杨坚（父）、文献皇后独孤伽罗（母）
- 兄弟: 杨勇（兄，政敌）、杨谅（弟，叛军首领）
- 支持者/盟友: 杨素（重臣，夺储支持者）、独孤皇后（夺储支持者）
- 叛军/弑杀者: 宇文化及（江都兵变首领）、裴虔通（执行弑杀者）
- 后代: 杨倓（孙，被杀）、杨暕（子，被杀）

## 历史评价
- 正面: 完成统一，开创大运河与科举制，一度拓疆至吐谷浑
- 负面: 滥用民力，穷兵黩武，骄奢淫逸，导致隋朝速亡
- 后世神话/演义: 常被描绘为典型的暴君与亡国之君形象，与陈后主并列

## 反差 / 鲜为人知点
- 早年镇守江南时，为稳定局势颇有成效，作风简朴谨慎，博得朝野赞颂
- 灭陈后曾推行户籍制度与乡村重组，但因政策偏差引发江南大规模叛乱
- 其墓葬在2013年于扬州邗江区被考古确认，为隋炀帝与萧后最终埋葬地
