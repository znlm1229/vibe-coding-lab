# 杨广

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字阿𡡉 / 无 / 炀帝 / 世祖 / 无
- 生卒年 / 朝代区间 / 籍贯: 569年—618年 / 隋朝 / 华阴（今陕西华阴）
- 主要职业 / 身份: 隋朝第二位皇帝

## 主要事迹
- 开皇八年（588年）作为名义统帅参与灭陈之战，统一中国。[一般]
- 开皇二十年（600年）取代兄长杨勇被立为太子。[重要]
- 仁寿四年（604年）即位后，营建东都洛阳并开凿大运河。[重要]
- 大业五年（609年）亲征平定吐谷浑，设置西海等四郡。[一般]
- 大业八年至十年（612-614年）三次大举进攻高句丽，均告失败。[重要]
- 在位期间正式确立科举制度为国家政策。[重要]
- 频繁巡游与大兴土木，导致民力耗竭。[重要]
- 大业十四年（618年）在江都兵变中被宇文化及弑杀。[重要]

## 性格 / 风格特征
- 好大喜功，热衷于大规模工程与远征（如大运河、高句丽之战）。
- 骄奢淫逸，追求极致的享乐与排场（如西苑、龙舟、盛陈百戏）。
- 善于伪装，早年为夺储位表现得作风简朴、礼贤下士。

## 典故 / 标志事件
- 开凿大运河，动用数百万民力。
- 三征高句丽，耗尽国力导致民变。
- 江都兵变，被叛军缢弑。
- 西苑夜游，命宫女骑马奏《清夜游》曲。
- 盛饰洛阳市场，以丝织品缠树夸耀富足。

## 关键作品
- 隋唐大运河（世界史上最长运河）。
- 营建东都洛阳。
- 确立并推广科举制度（增置进士科）。
- 西苑等奢华宫殿苑囿。

## 关系网
- 老师 / 同辈 / 弟子 / 政敌 / 家人: 高颎（元帅长史，后为其杀害）/ 杨勇（兄长，政敌）/ 无 / 杨玄感（礼部尚书，起兵叛隋）/ 杨坚（父，隋文帝），独孤伽罗（母，文献皇后），萧皇后（妻）

## 历史评价
- 正面: 完成国家统一（灭陈），开创科举制度，修建大运河促进南北交流。
- 负面: 好大喜功、滥用民力、骄奢淫逸，导致隋朝迅速灭亡，被谥为“炀”。
- 后世神话/演义: 常被描绘为典型的暴君和亡国之君形象。

## 反差 / 鲜为人知点
- 早年镇守江南时，为稳定局势表现出简朴、谦恭，赢得了朝野赞颂，与其后期形象反差巨大。
- 灭陈后欲纳陈后主宠妃张丽华为妾，被高颎斩杀，此事暴露其早期已有贪恋美色的倾向。
- 江都兵变被围时，他首先怀疑作乱者是自己的次子齐王杨暕，可见父子关系猜忌至深。
