# 朱元璋

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字国瑞 / 无 / 高皇帝 / 太祖 / 无
- 生卒年 / 朝代区间 / 籍贯: 1328年10月21日—1398年6月24日 / 元末明初 / 濠州钟离县（今安徽凤阳）
- 主要职业 / 身份: 明朝开国皇帝、军事家、政治家、农民起义领袖

## 主要事迹
- 1352年投奔郭子兴红巾军，开始反元生涯。[一般]
- 1356年攻克集庆（南京），改名应天府，作为根据地。[重要]
- 1363年于鄱阳湖之战歼灭陈友谅，奠定江南霸权。[重要]
- 1367年攻灭张士诚，统一南方。[一般]
- 1368年正月在应天府登基称帝，建立明朝，同年九月北伐攻占元大都，终结元朝统治。[重要]
- 建国后推行“洪武之治”，恢复经济，整顿吏治，颁布《大明律》。[重要]
- 1380年诛杀胡惟庸，废除丞相制度，加强君主专制。[重要]
- 1393年处理蓝玉案，株连众多功臣。[一般]
- 1398年驾崩，传位嫡长孙朱允炆。[次要]

## 性格 / 风格特征
- 意志坚定，志向远大，早年“志意廓然，人莫能测”。
- 行事果决，手段严厉，对官吏贪污及功臣威胁采取严酷镇压。
- 重视民生，强调“克城以武，戡乱以仁”，攻城掠地时常告诫将领勿妄杀。
- 兼具务实与权谋，采纳“高筑墙、广积粮、缓称王”策略，并能设计诈降诱敌（如对陈友谅）。

## 典故 / 标志事件
- 鄱阳湖之战以火攻大破陈友谅巨舰。
- 废除丞相制度，皇帝直领六部。
- 设立锦衣卫作为秘密警察机构。
- 颁布《大明律》并推行严刑峻法。
- 洪武四大案（胡惟庸案、蓝玉案等）诛杀功臣。

## 关键作品
- 建立明朝，开创洪武之治。
- 颁布《大明律》。
- 创立卫所兵制与屯田制度。
- 组织编纂《大明集礼》等典制（虽未直接著作，但主导确立）。
- 留有若干诗作（如《示僧》等）。

## 关系网
- 老师 / 同辈: 郭子兴（早期领袖/岳父）、刘基（谋士）
- 弟子: 朱标（太子，培养的继承人）
- 政敌: 陈友谅（汉政权）、张士诚（吴政权）、元顺帝（元朝）
- 家人: 马皇后（孝慈高皇后）、朱棣（第四子，燕王）

## 历史评价
- 正面: 结束元末乱世，恢复中华统治；勤政爱民，恢复社会经济；整顿吏治，打击贪污。
- 负面: 君主专制极端化，滥杀功臣，株连广泛；特务统治（锦衣卫）开启恶例；法令严酷，限制百姓自由。
- 后世神话/演义: 出生时有“红光满室”等神异传说；民间故事中常有其从乞丐到皇帝的传奇渲染；部分演义将其描绘为猜忌残忍的暴君形象。

## 反差 / 鲜为人知点
- 早年曾为游方僧（和尚），并非一直务农或起义。
- 登基后曾拒绝方国珍进献的金玉马鞍，称“宝玩非所好也”，只重人才与物资。
- 洪武二十三年曾“以京民为逆不已大杀京民”，导致在京居民半数被杀或流放，此事不在常见功绩叙述中。
