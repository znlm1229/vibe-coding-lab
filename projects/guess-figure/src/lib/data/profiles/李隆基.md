# 李隆基

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 三郎 / 无 / 明孝皇帝 / 玄宗 / 唐明皇
- 生卒年 / 朝代区间 / 籍贯: 685年9月8日—762年5月3日 / 唐朝 (712-756在位) / 东都洛阳 (今河南洛阳)
- 主要职业 / 身份: 唐朝皇帝 (第9任)

## 主要事迹
- 710年发动唐隆政变，诛杀韦后及其党羽，巩固李氏皇权 [重要]
- 712年接受睿宗禅让，即位为帝 [重要]
- 713年发动先天政变，铲除太平公主势力，完全掌握军政大权 [重要]
- 开元年间任用贤相姚崇等，开创“开元盛世” [重要]
- 开元二年 (714年) 下令销毁乘舆服御金银器玩，殿前焚烧珠玉锦绣以纠奢靡风气 [一般]
- 晚年怠政，宠信奸佞 (如李林甫、杨国忠)，导致朝政败坏 [重要]
- 755年安史之乱爆发，756年仓皇出逃，丧失统治权 [重要]

## 性格 / 风格特征
- 少年时即显胆识，曾直面呵斥武懿宗 [源自史料]
- 行事果断，敢于冒险 (如唐隆、先天政变均未事先禀报父亲) [源自史料]
- 前期励精图治，能尊重并放手任用贤臣 (如对姚崇) [源自史料]
- 后期耽于逸乐，听信谗言，倒行逆施 [源自史料]

## 典故 / 标志事件
- 唐隆政变：诛韦后，定社稷
- 先天政变：除太平，夺权柄
- 开元盛世：任用贤相，国力巅峰
- 安史之乱：宠信奸佞，盛世崩塌
- 马嵬驿兵变：仓皇出逃，丧权失势

## 关键作品
- 开创“开元盛世” (政绩)
- 确立并巩固唐朝中央集权 (政绩)
- 前期推行一系列纠奢倡俭、整顿吏治的政策 (政绩)
- 其统治时期是唐朝诗歌、艺术发展的黄金时代 (文化影响)
- 个人音乐造诣高，精通羯鼓等乐器 (个人技艺)

## 关系网
- 父亲 / 政敌: 唐睿宗李旦 (父亲) / 太平公主 (政敌)
- 老师 / 同辈: 无明确记载的老师 / 岐王李范、薛王李业 (同辈兄弟，政变支持者)
- 臣子 / 奸佞: 姚崇、宋璟 (贤相) / 李林甫、杨国忠 (奸佞)
- 家人: 窦德妃 (生母，被武则天杀害)、王皇后 (早期王妃)、杨贵妃 (后期宠妃)

## 历史评价
- 正面: 开创唐朝极盛之世“开元盛世”，前期励精图治，善于用人。
- 负面: 晚年昏聩怠政，任用奸佞，直接导致安史之乱，使唐朝国势急转直下。
- 后世神话/演义: 被民间及文学作品 (如《长恨歌》) 塑造为兼具英明与悲剧色彩的复杂帝王形象，与杨贵妃的爱情故事广为流传。

## 反差 / 鲜为人知点
- 其母窦德妃因卷入巫蛊案件被武则天杀害，幼年由姨妈窦氏及父亲妾室豆卢氏抚养。
- 曾一度被武则天命令过继给孝敬帝李弘为子，承继香火。
- 早年妾室杨良媛怀孕时，因东宫中太平公主耳目密布，李隆基曾试图令其堕胎以避祸。
