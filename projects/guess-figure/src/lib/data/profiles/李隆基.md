# 李隆基

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 明孝皇帝 / 玄宗 / 唐明皇
- 生卒年 / 朝代区间 / 籍贯: 685年9月8日—762年5月3日 / 唐朝 / 东都洛阳
- 主要职业 / 身份: 唐朝皇帝

## 主要事迹
- 710年发动唐隆政变，诛杀韦后及其党羽，巩固李氏皇权。[重要]
- 712年接受睿宗禅让，即位为皇帝。[一般]
- 713年发动先天政变，铲除太平公主势力，完全掌握政权。[重要]
- 在位前期任用贤相，励精图治，开创“开元盛世”。[重要]
- 开元二年（714年）下令销毁金银器玩、焚烧珠玉锦绣，以纠奢华风气。[一般]
- 晚年怠于政事，宠信奸臣（如李林甫、杨国忠），沉溺享乐。[重要]
- 755年安史之乱爆发，次年仓皇出逃长安，途中发生马嵬驿兵变。[重要]
- 756年被迫退位，成为太上皇，晚年忧郁孤苦。[一般]

## 性格 / 风格特征
- 英断果敢，胆识过人，少年时便敢呵斥武懿宗维护自家仪仗。
- 多才多艺，尤知音律，善八分书，仪表伟丽。
- 前期能虚心纳谏，尊重大臣（如放手让姚崇处理郎吏任命），后期则偏听偏信。
- 对亲情与权力斗争冷酷，为巩固统治可诛杀姑母（太平公主）并防范兄弟。

## 典故 / 标志事件
- 唐隆政变诛韦后。
- 先天政变除太平公主。
- 开创开元盛世。
- 晚年引发安史之乱。
- 马嵬驿兵变诛杨国忠、缢死杨贵妃。

## 关键作品
- 开创“开元盛世”的治国政绩。
- 颁布《禁厚葬制》等节俭法令。
- 其生日被定为“千秋节”（天长节），成为国家庆典。
- 善音律，对唐代音乐舞蹈发展有推动作用（虽非具体作品，但影响深远）。

## 历史评价
- 正面：前期励精图治，任用贤能，开创唐朝极盛之世“开元盛世”，被视为明君。
- 负面：晚年昏聩，怠政享乐，任用奸佞，导致安史之乱爆发，使唐朝国势急转直下。
- 后世神话/演义：文学作品中常描绘其与杨贵妃的爱情故事（如《长恨歌》），形象复杂化。

## 反差 / 鲜为人知点
- 其母窦德妃因巫蛊案被武则天秘密杀害时，他年仅八岁，由父亲另一妾室豆卢氏和姨妈抚养长大。
- 他曾被武则天命令过继给早逝的孝敬帝李弘为子，以继其香火。
- 为防范太平公主，他曾因妾室杨良媛怀孕而试图令其堕胎，因东宫中遍布太平公主耳目。
