# 赵构

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字德基 / 号损斋 / 庙号高宗 / 谥号宪孝 (取自“昭仁宪孝皇帝”末字)
- 生卒年 / 朝代区间 / 籍贯: 1107年6月12日—1187年11月9日 / 南宋 (1127-1162在位) / 东京开封府 (今河南开封)
- 主要职业 / 身份: 南宋开国皇帝

## 主要事迹
- 靖康元年（1126年）以亲王身份出使金营为人质，后因身份疑点被释回汴京。[一般]
- 靖康二年（1127年）北宋灭亡后，于南京应天府登基称帝，建立南宋。[重要]
- 建炎南渡，躲避金兵追击，最终于绍兴八年（1138年）定都临安。[重要]
- 绍兴十一年（1141年）与金朝达成绍兴和议，对金称臣、割地、赔款。[重要]
- 绍兴十一年除夕（1142年1月27日）杀害岳飞、岳云、张宪。[重要]
- 绍兴十二年（1142年）成功迎回生母韦太后。[一般]
- 绍兴三十一年（1161年）金兵再次南侵，宋军在采石之战击退金兵。[一般]
- 绍兴三十二年（1162年）禅让皇位于养子赵昚（宋孝宗），退位为太上皇。[重要]

## 性格 / 风格特征
- 一心偏安，无心北伐，为求和不惜称臣割地。
- 在位初期重用主战派，后期重用主和派并打压主战将领。
- 禅位后仍对朝政有所干预，尤其在议和问题上反对孝宗北伐。

## 典故 / 标志事件
- 泥马渡康王（磁州崔府君庙显圣传说）。
- 十二道金牌召回岳家军。
- 绍兴和议（对金称臣、割地、杀岳飞）。
- 禅位孝宗，回归太祖一脉。
- 杀害岳飞、岳云、张宪。

## 关键作品
- 《赐岳飞手敕》（书法作品）。
- 《真草嵇康养生论书卷》（书法作品）。
- 建立南宋政权，延续宋祚151年。
- 达成绍兴和议，形成宋金南北对峙格局。
- 禅位孝宗，结束太宗一脉186年统治。

## 历史评价
- 正面：延续宋祚，建立南宋；资性郎悟，好学强记；节俭、不好女色、潜心治国。
- 负面：偏安求和，无心北伐；杀害忠臣岳飞；对金称臣割地，丧权辱国。
- 后世神话/演义：“泥马渡康王”传说渲染其登基正统性；传说宋太祖显灵托梦使其传位太祖后裔。

## 反差 / 鲜为人知点
- 其母韦太后南归后，因岳飞未朝见而询问，得知岳飞遇害后怒骂高宗，高宗对此深感后悔。
- 禅位后与孝宗关系总体良好，父慈子孝，并未阻挠孝宗为岳飞平反和肃清秦桧余党。
- 书法造诣颇高，自言五十年间“未始一日舍笔墨”，初学黄庭坚，后学米芾，终追摹魏晋法度。
