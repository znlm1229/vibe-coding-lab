# 王安石

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 介甫 / 半山 / 文 / 无 / 王荆公
- 生卒年 / 朝代区间 / 籍贯: 1021年12月19日—1086年5月21日 / 北宋 / 抚州临川县（今江西省抚州市）
- 主要职业 / 身份: 政治家、文学家、思想家、改革家

## 主要事迹
- 庆历二年进士及第，初任签书淮南判官，后历任鄞县知县、常州知州等地方官，期间试行水利、贷谷等新政雏形。[一般]
- 嘉祐年间上万言书，提出变法主张，强调“因天下之力以生天下之财”。[重要]
- 熙宁元年入朝为翰林学士兼侍讲，与神宗论治国之道，主张效法尧舜。[一般]
- 熙宁二年拜参知政事，设立制置三司条例司，主持推行青苗、均输、保甲、免役、市易等新法，史称“熙宁变法”。[重要]
- 变法引发新旧党争，与司马光、韩琦、富弼等保守派激烈冲突。[重要]
- 熙宁七年因旱灾及反对压力罢相，知江宁府。[一般]
- 熙宁八年复相，继续推行新法，但与吕惠卿矛盾加剧。[次要]
- 其子王雱卷入与吕惠卿的斗争，病死后王安石悲伤不已，再次罢相。[次要]
- 晚年退居江宁，封荆国公，元祐元年去世，赠太傅，谥文。[一般]

## 性格 / 风格特征
- 议论高奇，能以辩博济其说，果于自用，慨然有矫世变俗之志。（源自《宋史》）
- 性格倔强，曾因开封府判案争议不肯谢罪，言“我无罪”。
- 早年恬退，屡辞馆阁试职与美官，士大夫谓其无意于世。

## 典故 / 标志事件
- 上万言书阐述变法思想。
- 主持熙宁变法（王安石变法）。
- 与司马光等引发新旧党争。
- 驳开封府斗鹑案，主张捕盗杀人无罪。
- 旱灾时坚持“水旱常数，尧、汤所不免”，拒罢新法。

## 关键作品
- 《临川集》（文集）
- 《三经义》（经学著作）
- 上万言书（《上仁宗皇帝言事书》）
- 创立荆公新学（学派）
- 推行青苗法、保甲法等新法（政绩）

## 关系网

- 弟子 / 政敌: 吕惠卿（初期盟友后政敌）、司马光（政敌）、韩琦（政敌）
- 家人: 王雱（子）、王安礼（弟）

## 历史评价
- 正面: 被誉为唐宋八大家之一，文学成就高；变法立意被部分后世认为良善，试图解决财政与社会问题。
- 负面: 变法被认为操之过急，加剧社会矛盾，导致“天下骚然”；性格固执，排斥异己。
- 后世神话/演义: 在民间故事和部分文学作品中形象复杂，既是改革家，也被描绘为导致北宋衰落的争议人物。

## 反差 / 鲜为人知点
- 早年任地方官时，在鄞县试行贷谷（青苗法雏形）成功，但后来全国推行时引发巨大争议。
- 曾因在常州开凿运河工程中民夫死伤众多而愧疚悔恨，显示其并非全然不顾民生。
- 其子王雱曾暗中插手陷害吕惠卿，王安石起初不知情，得知后责备儿子，此事暴露了家庭内部的权力斗争。
