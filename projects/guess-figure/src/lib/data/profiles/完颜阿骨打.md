# 完颜阿骨打

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 旻 (汉名) / 无 / 武元 (谥号简称) / 太祖 / 无
- 生卒年 / 朝代区间 / 籍贯: 1068年8月1日—1123年9月19日 / 辽末金初 / 按出虎水（今黑龙江哈尔滨东南阿什河）
- 主要职业 / 身份: 女真族完颜部首领 / 金朝开国皇帝

## 主要事迹
- 1113年，继位女真部落联盟长，称都勃极烈。[一般]
- 1114年，率2500人起兵叛辽，攻破宁江州。[重要]
- 1114年，率3700人夜袭出河店，渡混同江大败辽军。[重要]
- 1115年1月28日，在会宁称帝，建立大金，年号收国。[重要]
- 1115年9月，攻占辽国黄龙府。[一般]
- 1120年，与宋朝订立“海上之盟”，联合攻辽。[重要]
- 1120年，攻陷辽上京临潢府。[一般]
- 1122年，攻取辽中京；年底攻陷燕京。[重要]
- 1123年，如约归还燕京等部分州县给宋朝。[一般]

## 性格 / 风格特征
- 豁达大度，知人善任，人乐为用（源自《金史》评价）。
- 痛恨辽国，但对宋相当和善，与后世金帝态度不同。
- 崇尚汉文化，其年长儿子深受影响并推动金国汉化。

## 典故 / 标志事件
- 宁江州之战：以2500人起兵叛辽，首战告捷。
- 出河店之战：率3700人夜袭渡江，大败辽军7000精兵。
- 建立大金：于会宁称帝，开创金朝。
- 海上之盟：与宋朝订立联合攻辽的盟约。
- 归还燕京：履约将燕京等部分州县归还宋朝。

## 关键作品
- 创立金朝政权及国家制度。
- 制定“收国”年号，开启金朝纪元。
- 攻灭辽国大半疆土，奠定金朝版图基础。
- 与宋订立“海上之盟”，确立初期外交格局。
- 初步推动女真部落联盟向中央集权国家转变。

## 历史评价
- 正面：元朝《金史》评价其“英谟睿略，豁达大度，知人善任，人乐为用”，数年之间算无遗策，底定大业。
- 负面：暂无显著负面评价（史料主要记载其功绩）。
- 后世神话/演义：作为金朝开国太祖，在相关历史叙事中常被描绘为雄才大略的君主。

## 反差 / 鲜为人知点
- 其汉名“完颜旻”并非登基后所改，而是在建国称帝时即改名旻。
- 他归还给宋朝的燕京等地，包括了并非石敬瑭所割“燕云十六州”之内的景州、易州。
- 1980年代在北京西南郊九龙山发现的金朝陵墓，后于2003年被证实为其石棺、遗骨及装饰物所在。
