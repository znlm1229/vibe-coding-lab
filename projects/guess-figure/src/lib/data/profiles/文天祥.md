# 文天祥

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字履善 / 号文山 / 谥号忠烈 / 无 / 无
- 生卒年 / 朝代区间 / 籍贯: 1236年6月6日—1283年1月9日 / 南宋 / 吉州庐陵富川（今江西吉安县）
- 主要职业 / 身份: 政治家、诗人、抗元将领、民族英雄

## 主要事迹
- 宝祐四年（1256年）殿试第一，考中状元，宋理宗赐字宋瑞。[重要]
- 开庆元年（1259年）蒙古攻鄂州，上疏请斩主张迁都的宦官董宋臣，并提出御敌之计。[一般]
- 德祐元年（1275年）任右丞相，赴元营谈判，怒斥伯颜，后被元军扣押。[重要]
- 被押送镇江时得义士相救脱险，后在南剑州开府，继续组织抗元。[一般]
- 1277年率军攻入江西，收复赣州、吉州等地，但不久在兴国大败，妻儿被俘。[一般]
- 1278年冬在五坡岭兵败被俘，吞冰片自杀未遂，被押往大都。[重要]
- 被囚大都期间，拒绝忽必烈及降元宋臣的多次劝降，并写下《过零丁洋》《正气歌》。[重要]
- 至元十九年十二月初九（1283年1月9日），于大都柴市从容就义。[重要]

## 性格 / 风格特征
- 早期生活奢侈，养有歌妓；国难当头后，散尽家财募兵，生活转为简朴。[源自史料]
- 谈及国事时常痛哭流涕，抚案而言，将他人之忧乐视为己任。[源自史料]
- 坚贞不屈，面对死亡威胁与劝降，始终秉持“义尽仁至”信念。[源自史料]
- 性格中带有黑色幽默，临安沦陷前曾以笑话回应幕僚劝其自杀的提议。[源自史料]

## 典故 / 标志事件
- 殿试时宋理宗见其名“天祥”，称“天之祥，乃宋之瑞也”，赐字宋瑞。
- 五坡岭兵败被俘，吞冰片自杀未遂。
- 被俘后于零丁洋写下“人生自古谁无死，留取丹心照汗青”。
- 囚于大都四年，拒绝忽必烈亲自劝降，从容赴死柴市。
- 衣带中发现绝笔自赞：“孔曰成仁，孟曰取义；惟其义尽，所以仁至。”

## 历史评价
- 正面: 被尊为民族英雄，其忠肝义胆与不屈精神备受推崇，与陆秀夫、张世杰并称“宋末三杰”。
- 负面: 早期生活奢侈；抗元军事策略与行动最终失败，未能挽救南宋。
- 后世神话/演义: 被后世文学、戏曲广泛颂扬，成为忠贞符号；民间及海外华人社区多建庙宇祭祀（如三忠庙）；在部分演义作品中（如《天子传奇6》）被赋予道家术士等传奇形象。

## 反差 / 鲜为人知点

- 被俘后曾绝食八日求死，但未成功。[不在维基主条目首段]
