# 康熙帝

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 玄烨 / 体元主人 / 仁皇帝 / 圣祖 / 无
- 生卒年 / 朝代区间 / 籍贯: 1654年5月4日 - 1722年12月20日 / 清朝 (1661-1722在位) / 北京
- 主要职业 / 身份: 清朝皇帝

## 主要事迹
- 1661年，八岁即位，索尼、苏克萨哈、遏必隆、鳌拜四大臣辅政。[一般]
- 1667年，亲政，但鳌拜专权。[一般]
- 1669年，设计擒拿权臣鳌拜，清除其势力，真正掌握皇权。[重要]
- 1673-1681年，平定三藩之乱（吴三桂、耿精忠、尚可喜）。[重要]
- 1683年，派施琅攻占台湾，消灭明郑政权。[重要]
- 1685-1688年，两次雅克萨之战，击败俄国，签订《尼布楚条约》。[重要]
- 1690-1697年，三次亲征噶尔丹，平定准噶尔叛乱。[重要]
- 1712年，颁布“滋生人丁，永不加赋”政策。[一般]
- 晚年发生九子夺嫡，皇位继承斗争激烈。[一般]

## 性格 / 风格特征
- 勤政好学，每日听政，博览群书，涉猎西方科学。[重要]
- 处事果断，少年时期即能设计铲除权臣鳌拜。[重要]
- 重视民生，多次减免赋税，巡视河工。[一般]
- 晚年趋于保守，对皇子争储问题处置犹豫。[一般]

## 典故 / 标志事件
- 少年智擒权臣鳌拜。
- 平定三藩之乱。
- 收复台湾。
- 签订《尼布楚条约》。
- 亲征噶尔丹。

## 关键作品
- 《康熙字典》
- 《律历渊源》（包括《数理精蕴》）
- 治理黄河、淮河的水利工程。
- 组织测绘并编制《皇舆全览图》。
- 主持编纂《古今图书集成》（未完成，雍正朝竣工）。

## 关系网
- 老师: 南怀仁 (西方科学导师)
- 同辈: 裕亲王福全 (兄)
- 弟子: 无明确弟子，但培养皇子如胤礽、胤禛等。
- 政敌: 鳌拜 (早期权臣)、吴三桂 (三藩首领)、噶尔丹 (准噶尔首领)
- 家人: 孝诚仁皇后赫舍里氏 (首任皇后)、孝恭仁皇后乌雅氏 (雍正帝母)、胤礽 (废太子)、胤禛 (雍正帝，继位者)

## 历史评价
- 正面: 被视为清朝最杰出的皇帝之一，开创“康乾盛世”，文治武功卓著，勤政爱民，学识渊博。
- 负面: 晚年吏治松弛，国库空虚；皇子夺嫡斗争残酷，处置不当；文字狱加剧；闭关政策萌芽。
- 后世神话/演义: 民间故事中常被描绘为英明神武、微服私访的圣君形象；在武侠小说（如《鹿鼎记》）中也有艺术加工。

## 反差 / 鲜为人知点
- 他精通数学、天文学等西方科学，曾亲自学习并运用欧几里得几何。
- 晚年因九子夺嫡心力交瘁，对臣子感叹“天下事不如意者十之八九”。
- 他身高可能不高，据一些西方传教士记载，其身材“中等偏矮”。
