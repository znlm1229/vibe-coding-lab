# 林则徐

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字少穆 / 号竢村老人 / 谥文忠 / 无 / 无
- 生卒年 / 朝代区间 / 籍贯: 1785年8月30日—1850年11月22日 / 清朝 / 福建侯官（今福州）
- 主要职业 / 身份: 政治家、思想家、诗人、钦差大臣、总督

## 主要事迹
- 1811年进士，步入仕途，历任翰林院庶吉士、御史等职 [一般]
- 1823年任江苏按察使，澄清吏治，处理积案，人称“林青天” [一般]
- 1831年擢升河东河道总督，整顿河工，查验料垛，治理水患 [一般]
- 1838年响应黄爵滋禁烟主张，提出“十年后无可用之兵”的警告，获道光帝赏识 [重要]
- 1839年任钦差大臣赴广东禁烟，收缴销毁鸦片于虎门，是为“虎门销烟” [重要]
- 1839-1840年整顿广东海防，视察炮台，增设防御工事，多次击退英军挑衅 [重要]
- 1840年鸦片战争爆发后，因局势不利被革职，后遣戍伊犁 [重要]
- 1844-1845年在伊犁期间，受命兴治屯田，垦田三万七千余顷 [一般]
- 1847年任云贵总督，处理汉回互斗，主张“止分良莠，不分汉回”，稳定边疆 [一般]
- 1850年受命为钦差大臣赴广西镇压太平军，途中病逝于潮州 [次要]

## 性格 / 风格特征
- 清廉刚正，不畏权势，曾直言弹劾满洲贵族琦善办事不力。
- 处事严谨务实，任河道总督时亲自逐垛查验料垛，治狱时亦亲自动手验尸。
- 重视人才选拔与吏治，任考官时公正严肃，并强调“察吏莫先于自察”。
- 对西方持开放态度，主张学习其科技文化，并组织翻译西方报刊书籍。

## 典故 / 标志事件
- 虎门销烟：在广东收缴鸦片并于虎门海滩公开销毁。
- 海到无涯天作岸，山登绝顶我为峰：少年时在学堂所作诗句，震惊四座。
- 察吏莫先于自察：任江苏巡抚时提出的吏治考核原则。
- 为中国患者，其俄罗斯乎：晚年对海疆形势的预言。
- 赴粤禁烟前向道光帝警告“十年后无可用之兵”。

## 关键作品
- 《海国图志》（主持编译，由魏源汇编）
- 《畿辅水利议》（原名《北直水利书》，治水方略）
- 云左山房杂录（读书札记）
- 整顿盐政、治理漕运的相关奏疏与改革实践
- 诗文创作

## 历史评价
- 正面: 近代中国“开眼看世界”的先驱，禁烟运动的民族英雄，为官清廉，治绩卓越。
- 负面: 禁烟操之过急，引发战争；部分时人及后世认为其策略未能充分考虑国际形势。
- 后世神话/演义: 常被塑造为坚决反抗外来侵略的完美忠臣形象，其开放学习西方的侧面有时被淡化。

## 反差 / 鲜为人知点
- 并非天生擅长治水，最初被任命为河道总督时曾以不懂河务为由推辞，后通过刻苦学习补足。
- 家境贫寒，童年时常将母亲和姐妹的女红工艺品拿到店铺寄卖以补贴家用。
- 晚年预言未来对中国的主要威胁将是俄罗斯，而非英国。
