# 耶律阿保机

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 无 / 太祖 / 耶律亿
- 生卒年 / 朝代区间 / 籍贯: 872年－926年9月6日 / 辽代 / 契丹迭剌部
- 主要职业 / 身份: 契丹可汗、大契丹国（辽朝）开国皇帝

## 主要事迹
- 901年被立为契丹迭剌部夷里堇（军事首领）兼于越，开始武力征服周边地区。[重要]
- 907年设坛祭天，加封自己为全体契丹人的可汗，试图将选举制改为世袭制。[重要]
- 平定因废除选汗制度引发的“诸弟之乱”，巩固权力。[一般]
- 916年正式称帝，建立“大契丹国”，建年号神册，创制契丹大字。[重要]
- 建国后持续扩张，灭渤海国，征服室韦、奚、阻卜诸部，招揽汉人发展生产。[重要]
- 926年去世于扶余城，为辽太宗灭后晋改国号“大辽”奠定基础。[一般]

## 性格 / 风格特征
- 雄武善战，东征西讨如折枯拉朽，信威万里。
- 有政治魄力，敢于废除传统选汗制度，建立世袭皇权。
- 善于采纳汉人建议，仿幽州建汉城，仿汉字创契丹文，重用韩延徽定名分。

## 典故 / 标志事件
- 设坛祭天自立为可汗，开启契丹世袭帝制。
- 平定“诸弟之乱”，巩固统治。
- 神册元年称帝，建立大契丹国。
- 创制契丹大字。
- 赐后族姓氏为萧，源于羡慕萧何辅刘邦的典故。

## 关键作品
- 建立大契丹国（后为辽朝）。
- 创制契丹大字。
- 制定君臣名分，确立国家制度。
- 征服渤海国，扩展疆域。
- 发展农业、冶铁，使部族殷富。

## 关系网
- 老师 / 同辈 / 弟子 / 政敌 / 家人 (各列 1-3 人，标关系类型)
- 重要辅臣：韩延徽（汉人谋士）。
- 政敌：诸弟（剌葛、安端等，叛乱者）。
- 家人：淳钦皇后述律平（妻）；耶律倍（长子）；耶律德光（次子，辽太宗）。

## 历史评价
- 正面：雄才大略的开国君主，东征西讨，建章立制，奠定辽朝二百年基业。
- 负面：废除传统选汗制度引发内部叛乱，手段强硬。
- 后世神话/演义：在动漫《画江湖之不良人》中以“耶律保”形象出现。

## 反差 / 鲜为人知点
- 其四子耶律牙里果曾随军出征后唐被俘，在太原居住十余年后才归辽。
- 他赐母族、妻族姓“萧”是出于个人对汉初名臣萧何的羡慕，而非单纯的政治联姻需要。
- 其建国年份（907年称汗与916年称帝）在史料中存在长期争议，非简单的单一事件。
