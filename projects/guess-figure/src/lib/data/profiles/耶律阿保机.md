# 耶律阿保机

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字阿保机 / 谥号烈 / 庙号太祖 / 汉名刘亿 / 无
- 生卒年 / 朝代区间 / 籍贯: 872年—926年9月6日 / 辽朝 / 契丹迭剌部霞瀨益石烈鄉耶律彌里人
- 主要职业 / 身份: 契丹可汗、大契丹国（辽朝）开国皇帝

## 主要事迹
- 901年被立为军事首领夷里堇，后成为迭剌部酋长，以武力征服周边部落。[重要]
- 907年设坛祭天，加封自己为全体契丹人的可汗，废除三年一次的选汗制度，引发诸弟之乱。[重要]
- 916年正式称帝，建立“大契丹国”，建年号神册，并创制契丹大字。[重要]
- 建国后东征西讨，灭渤海国，征服室韦、奚、阻卜诸部，招揽汉人发展生产。[重要]
- 926年东征渤海国后，于扶余城去世。[一般]

## 性格 / 风格特征
- 目光射人，身长九尺，丰上锐下，关弓三百斤，勇武过人。[源自《辽史》]
- 性虽敏黠，而蓄奸稔恶，安忍凶狠，贪黩无厌。[源自《辽史》对其弟的评价引申]
- 每优容反覆不悛者，但在处置叛乱时，又能果断诛杀首恶，恩威并施。[源自《辽史》记载]

## 典故 / 标志事件
- 燔柴告天，即皇帝位，开创契丹世袭帝制。
- 诸弟之乱（剌葛、迭剌、寅底石、安端谋反），平定后赦罪复用。
- 建元神册，创制契丹大字。
- 羡慕萧何辅佐刘邦典故，赐后族姓氏为萧。
- 东征渤海国，死于扶余城。

## 关键作品
- 建立大契丹国（辽朝）及一系列国家制度。
- 创制契丹大字。
- 征服并整合奚、室韦、渤海等部族，扩大版图。
- 建龙化州、开教寺、天雄寺等，引入佛教与文化。
- 设置铁冶，发展生产。

- 老师 / 同辈 / 弟子 / 政敌 / 家人 (各列 1-3 人，标关系类型)

## 历史评价
- 正面: 东征西讨，如折枯拉朽，信威万里，奠定辽朝二百年基业；建国立制，功业显赫。
- 负面: 诸弟之乱因其废除选举制度引发，蓄奸稔恶，贪黩无厌（史书对其弟的评价亦折射其家族内部矛盾）。
- 后世神话/演义: 出生时有神光异香，体如三岁儿，三月能行，知未然事，被赋予神异色彩。

## 反差 / 鲜为人知点
- 曾与河东节度使李克用在云州会盟，宴酣后易袍马，约为兄弟，并借兵助其攻打刘仁恭。[源自《辽史》]
- 平定诸弟之乱后，对三百余名逆党罪犯，赐宴一日，任其歌舞戏射，次日才论刑，展现复杂人性。[源自《辽史》]
- 其四子耶律牙里果在随军攻打后唐时被俘，安置在太原居住多年后才归辽。[源自维基家庭部分]
