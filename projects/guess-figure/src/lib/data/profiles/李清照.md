# 李清照

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 易安居士 / 无 / 无 / 李三瘦
- 生卒年 / 朝代区间 / 籍贯: 1084年3月13日—1155年5月12日 / 北宋末年至南宋初年 / 齐州（今山东省济南市）
- 主要职业 / 身份: 词人 / 文学家

## 主要事迹
- 建中靖国元年（1101年），18岁时与太学生、金石学家赵明诚结婚，婚后生活优裕，词作多写闺情相思。[一般]
- 婚后随夫移居青州，致力于与丈夫共同搜集、整理金石书画。[一般]
- 靖康之变后，于建炎二年（1128年）南渡至江宁（今南京），开始颠沛流离的生活。[重要]
- 建炎三年（1129年），丈夫赵明诚病逝于建康，此后生活陷入困顿，词风转为凄怆沉郁。[重要]
- 南渡后，所藏金石书画在流离中大量散佚，绍兴元年（1131年）在越州（今绍兴）寓所遭遇书画被盗，藏品几乎尽失。[一般]
- 晚年（约绍兴二年，1132年）曾改嫁张汝舟，但不久因不合而离异，并因此涉讼。[重要]
- 致力于文学创作与理论，提出“词，别是一家”的著名观点，并形成独特的“易安体”。[重要]
- 约卒于绍兴二十五年（1155年），具体晚年经历记载不详。[次要]

## 性格 / 风格特征
- 才情卓绝，对自己的文学才华颇为自信，曾作《词论》点评北宋名家。
- 性格中有刚毅豪迈的一面，南渡后诗词常流露出家国之痛与悲愤之情，被评“倜傥有丈夫气”。
- 情感细腻敏感，善于捕捉日常生活中的细微感受，并通过白描手法深刻表达。

## 典故 / 标志事件
- 赵明诚为胜《醉花阴》词，闭门苦作多首，友人仍指其中“莫道不消魂，帘卷西风，人比黄花瘦”三句最佳，正是李清照原作。
- 南渡后珍藏的金石书画在流亡途中于越州钟氏宅邸一夜被盗，毕生心血几乎散尽。
- 晚年曾改嫁张汝舟，因不堪其辱，不久便诉诸公堂要求离婚。
- 创作《声声慢》开篇连用七组叠字“寻寻觅觅，冷冷清清，凄凄惨惨戚戚”，被誉为创意奇绝。
- 少女时期作《如梦令》“知否？知否？应是绿肥红瘦”，以“绿肥红瘦”新颖描摹风雨后花叶情态，广为传诵。

## 关键作品
- 词：《声声慢·寻寻觅觅》、《醉花阴·薄雾浓云愁永昼》、《一剪梅·红藕香残玉簟秋》
- 诗：《夏日绝句》（生当作人杰）
- 文：《金石录后序》、《词论》

## 关系网
- 家人：父亲李格非（进士、礼部员外郎）、丈夫赵明诚（金石学家）
- 老师/同辈：其父李格非受苏轼赏识，但无直接师承记载；与辛弃疾并称“济南二安”。
- 弟子/政敌：曾欲将学问传授给孙氏（陆游提及），但被婉拒；无明确政敌记载。

## 历史评价
- 正面：被誉为“千古第一才女”，宋代朱熹称“本朝妇人能文者，惟魏夫人及李易安二人而已”，后世杨慎赞其词“冠绝”。
- 负面：其改嫁与讼离一事，在部分后世儒家卫道者眼中被视为“失节”污点。
- 后世神话/演义：在民间传说中被尊奉为“藕花神”，济南大明湖畔藕神祠有其供像；其形象在各类影视、戏剧作品中不断被演绎。

## 反差 / 鲜为人知点
- 据陆游记载，她晚年曾想将毕生才学传授给一位孙姓少女，却被对方以“才藻非女子事也”的理由拒绝。
- 国际天文联合会将水星上的一座撞击坑命名为“李清照撞击坑”，金星上也有一座以她命名的环形山。
- 其籍贯虽有章丘明水镇之说，但有学者考证认为她实为齐州历城（今济南历城区）人，且一生并未在济南定居过。
