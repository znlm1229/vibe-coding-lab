# 李治

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字为善 / 天皇大帝 / 弘孝皇帝 / 高宗 / 雉奴
- 生卒年 / 朝代区间 / 籍贯: 628年7月21日—683年12月27日 / 唐朝 (太宗贞观年间至高宗在位时期) / 长安 (今陕西西安)
- 主要职业 / 身份: 唐朝皇帝

## 主要事迹
- 贞观十七年（643年），因太子李承乾被废、魏王李泰失宠，被立为太子 [重要]
- 贞观二十三年（649年）即位，开启永徽之治，初期励精图治 [重要]
- 永徽六年（655年），废王皇后、萧淑妃，立武则天为皇后 [重要]
- 在位期间唐朝版图达到最大，东至朝鲜半岛，西包鹹海，北抵贝加尔湖，南达越南承天省 [重要]
- 显庆年间（656年起），因长期头痛与眼疾，时常无法决断，武则天开始介入并掌握权力 [重要]
- 总章年间（668-670年），唐朝国力进入顶峰 [一般]
- 弘道元年（683年）驾崩于洛阳紫微宫贞观殿 [一般]

## 性格 / 风格特征
- 仁孝宽厚：幼时即以仁孝闻名，太宗患病时亲吮痈疮，扶辇步从数日
- 优柔寡断：晚年因身体病痛（头痛、眼疾）而时常无法下判断，决策软弱
- 重视亲情：对同母兄弟及子女表现出关怀，但后期家庭关系因武则天介入而复杂化

## 典故 / 标志事件
- 晋王封爵：贞观五年（631年）获封晋王，幼年仁孝事迹流传
- 飞表奏事：太宗征高丽时，太子李治悲啼累日，请求飞驿递表起居，开创飞表奏事制度
- 永徽之治：即位初期政治清明，国力恢复，史称永徽之治
- 废立皇后：永徽六年废王皇后、萧淑妃，立武则天，引发后宫权力结构剧变
- 天皇大帝：死后谥号“天皇大帝”，后增谥为“天皇大圣大弘孝皇帝”

## 关键作品
- 永徽律令格式：颁布新律令格式于天下，完善法律体系
- 支持《五经正义》：颁孔颖达《五经正义》于天下，作为明经考试标准
- 扩建长安罗郭：命阎立德领丁夫四万筑长安罗郭，加强都城防御
- 设立安西都护府：以高昌故地置安西都护府，巩固西域统治
- 佛寺建设与支持：在昭陵侧建佛寺，并迎玄奘及慈恩寺碑文，支持佛教发展

## 关系网
- 父亲 / 母亲：唐太宗李世民 / 文德皇后长孙氏
- 兄弟姐妹：太子李承乾（兄）、魏王李泰（兄）、晋阳公主（胞妹）
- 妻子 / 妃嫔：王皇后（元配）、武则天（皇后）、萧淑妃、郑贵妃（宫人）、刘宫人、杨宫人
- 子女：燕王李忠（废太子）、原悼王李孝、泽王李上金、许王李素节、孝敬皇帝李弘、章怀太子李贤、唐中宗李显、唐睿宗李旦、义阳公主、高安公主、安定公主、太平公主
- 重要大臣：长孙无忌（初期辅政）、褚遂良（中书令）、于志宁（尚书左仆射）、李勣（同中书门下三品）、许敬宗（后期权臣）
- 政敌 / 对手：房遗爱、荆王元景、吴王恪等（永徽四年谋反案）

## 历史评价
- 正面：初期励精图治，延续贞观遗风，实现“永徽之治”；在位期间唐朝版图达到极盛；重视法律与文化教育，颁布律令、支持经学。
- 负面：晚年因健康问题优柔寡断，导致武则天逐步掌权，为后来武周代唐埋下隐患；对后宫争斗处置残酷（废后、杀妃）；部分时期政事依赖外戚与权臣。
- 后世神话/演义：常被描绘为软弱、受武则天控制的皇帝形象；在民间故事中，其眼疾与针灸治疗（秦鸣鹤故事）成为传奇片段；与武则天姐姐韩国夫人武顺及甥女魏国夫人的私情传闻也成为后世文学题材。

## 反差 / 鲜为人知点
- 李治并非一直软弱：在位初期曾亲自指挥平定西域叛乱（如讨伐贺鲁），并积极处理政务，展现出一定的统治能力。
- 他的眼疾治疗颇具风险：御医秦鸣鹤曾主张用针灸治疗其头痛眼疾，武则天一度大怒说“此人可斩首”，但高宗本人批准了治疗，且短暂奏效。
- 家庭关系复杂且多悲剧：其子女中多人死于非命或遭废黜（如李忠、李弘、李贤），后宫王皇后、萧淑妃被废杀，反映出宫廷斗争的残酷性，远超一般帝王家庭。
