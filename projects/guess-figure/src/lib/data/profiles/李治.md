# 唐高宗

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字为善 / 谥号天皇大帝 / 庙号高宗 / 无 / 无
- 生卒年 / 朝代区间 / 籍贯: 628年7月21日—683年12月27日 / 唐朝 / 长安（今陕西西安）
- 主要职业 / 身份: 唐朝皇帝

## 主要事迹
- 贞观十八年（645年）被立为太子，贞观二十三年（649年）即位为帝。[重要]
- 在位初期励精图治，开创“永徽之治”。[重要]
- 在位期间唐朝版图达到最大，东起朝鲜半岛，西至咸海。[重要]
- 永徽六年（655年）废王皇后、萧淑妃，立武则天为皇后。[重要]
- 晚年长期患有头痛与眼疾，一度失明，曾接受御医秦鸣鹤针灸治疗。[一般]
- 因身体原因，武则天逐渐介入并掌握朝政决策。[重要]
- 总章年间（668-670年）唐朝国力进入顶峰。[一般]
- 弘道元年（683年）驾崩于洛阳紫微宫，葬于乾陵。[次要]

## 性格 / 风格特征
- 幼时即以仁孝闻名，性格温和。
- 晚年因身体羸弱，时常无法独立做出判断，显得软弱。
- 对武则天既有依赖，也曾试图限制其权力（如欲立其为宸妃受阻）。

## 典故 / 标志事件
- 永徽之治：高宗在位初期的治世。
- 废王立武：废黜王皇后、萧淑妃，立武则天为皇后。
- 针灸明目：御医秦鸣鹤为其针灸百会穴以治疗眼疾。
- 版图极盛：唐代版图以高宗时为最大。
- 乾陵合葬：与武则天合葬于乾陵。

## 关键作品

## 历史评价
- 正面: 在位初期开创“永徽之治”，唐朝版图在其统治下达到极盛。
- 负面: 晚年体弱，决策依赖武则天，导致后者权力膨胀，为武周代唐埋下伏笔。
- 后世神话/演义: 常被描绘为受武则天控制的软弱皇帝形象。

## 反差 / 鲜为人知点
- 他是唐太宗李世民唯一一对（与晋阳公主）亲自抚养长大的亲生子女之一。
- 他曾试图设立“宸妃”封号以安置武则天，因后宫已有四妃而受阻。
- 其情妇包括武则天的姐姐韩国夫人武顺及外甥女魏国夫人贺兰氏。
