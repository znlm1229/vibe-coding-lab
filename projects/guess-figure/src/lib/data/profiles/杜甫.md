# 杜甫

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字子美 / 号少陵野老 / 谥号无 / 庙号无 / 别号杜工部
- 生卒年 / 朝代区间 / 籍贯: 712年2月12日—770年 / 唐朝 / 本襄阳人，后徙河南巩县
- 主要职业 / 身份: 诗人、官员（左拾遗、检校工部员外郎）

## 主要事迹
- 开元二十四年（735年）在洛阳参加进士考试落第 [一般]
- 天宝三载（744年）在洛阳与李白相遇，相约梁宋之游，结下深厚友谊 [重要]
- 天宝六载（747年）参加长安应试，因李林甫“野无遗贤”闹剧全部落选，后客居长安十年，献赋求官 [重要]
- 天宝十四载（755年）安史之乱爆发，次年杜甫为叛军俘虏，押至长安，写下《春望》等诗 [重要]
- 至德二载（757年）冒险逃至凤翔投奔肃宗，被授为左拾遗，后因营救房琯触怒肃宗被贬 [重要]
- 乾元元年（758年）被贬华州司功参军，途中目睹战乱写成“三吏三别”组诗 [重要]
- 乾元二年（759年）弃官，携家漂泊，经秦州、同谷等地，备尝艰辛 [一般]
- 上元元年（760年）于成都浣花溪畔筑茅屋而居，即成都杜甫草堂 [一般]
- 广德二年（764年）严武表荐杜甫为节度参谋、检校工部员外郎 [一般]
- 大历五年（770年）病死在湘江舟中，时年五十九岁 [次要]

## 性格 / 风格特征
- 性格褊躁无器度，恃恩放恣（据《旧唐书》记载曾醉酒登严武之床瞪视） [源自史料]
- 傲诞不拘检，与田夫野老相狎，纵酒啸咏（《旧唐书》载严武过之有时不冠） [源自史料]
- 现实主义诗人，风格浑朴沉郁，其诗因社会时代意义被誉为“诗史” [源自评价]
- 也有狂放不羁的一面，从其名作《饮中八仙歌》可见豪气干云 [源自作品]

## 典故 / 标志事件
- 醉登严武床榻瞪视曰：“严挺之乃有此儿！”
- 献《三大礼赋》得玄宗赏识，命待制集贤院但未得官职。
- 安史之乱中被叛军俘虏，冒险从金光门逃出长安投奔肃宗。
- 因营救宰相房琯触怒肃宗，被贬华州司功参军。
- 漂泊同谷时陷入饥寒交迫绝境，作《乾元中寓居同谷县作歌七首》。

## 关键作品

- “三吏”（《新安吏》《石壕吏》《潼关吏》）与“三别”（《新婚别》《垂老别》《无家别》）
- 《春望》
- 《登高》
- 《茅屋为秋风所破歌》

## 历史评价
- 正面: 被后人奉为“诗圣”，其诗被誉为“诗史”；元稹评价其诗“上薄风骚，下该沈宋，尽得古今之体势”。
- 负面: 《旧唐书》评其“性褊躁无器度，恃恩放恣”；李白曾讥其“龌龊”，有饭颗山之嘲诮。
- 后世神话/演义: 有传说其死于耒阳聂县令所赠牛肉白酒，饱食而死；后世尊称为“老杜”。

## 反差 / 鲜为人知点
- 杜甫外祖父母均为李唐皇室之后，其外祖母之父是唐太宗与韦贵妃之孙，母系有显赫皇室背景。
- 据《新唐书》记载，严武虽厚待杜甫，但因杜甫醉酒失言“严挺之乃有此儿”而怀恨在心，数次欲杀杜甫。
- 杜甫早年生活豪放轻狂，自称“放荡齐赵间，裘马颇清狂”，并非始终是沉郁困苦的形象。
