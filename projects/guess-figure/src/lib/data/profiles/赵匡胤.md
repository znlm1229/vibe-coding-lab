# 赵匡胤

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字元朗 / 无 / 无 / 太祖 / 无
- 生卒年 / 朝代区间 / 籍贯: 927年3月21日—976年11月14日 / 北宋 / 涿郡（今河北保定清苑）
- 主要职业 / 身份: 宋朝开国皇帝、政治家、军事家

## 主要事迹
- 948年投奔郭威，随军征讨李守贞，开始军事生涯。[一般]
- 954年高平之战中身先士卒，率军冲锋，大败北汉军，战后被擢升并负责整顿禁军。[重要]
- 956-958年间随周世宗征讨南唐，屡立战功，先后攻克涂山、清流关、滁州等地，并参与攻陷濠州、泗州、楚州。[重要]
- 960年发动陈桥兵变，黄袍加身，受禅登基，建立宋朝。[重要]
- 960年平定昭义节度使李筠和淮南节度使李重进的叛乱，巩固新政权。[一般]
- 961年通过“杯酒释兵权”，解除石守信等主要禁军将领的兵权。[重要]
- 依据“先南后北”策略，在位期间先后灭亡荆南、湖南、后蜀、南汉及南唐等南方割据政权。[重要]
- 推行一系列中央集权措施，包括财政上收归地方税赋、设置转运使；行政上以文臣“权知军州事”、设通判；司法上死刑案件上报中央、设县尉等。[重要]
- 969年再次“杯酒释兵权”，解除王彦超等地方藩镇节度使的兵权。[重要]
- 设立“封桩库”贮藏钱帛，意图赎回燕云十六州，但未成而逝世。[次要]

## 性格 / 风格特征
- 勇武善战，冲锋陷阵（如高平之战身先士卒）。
- 行事果断，兼具谋略（如策划陈桥兵变，杯酒释兵权）。
- 重视秩序与约束，治军严明（如下令“扬州士兵过六合者，砍断双腿！”以固守）。
- 对部下恩威并施，既以富贵相诱，又以潜在威胁警示（如杯酒释兵权中的对话）。

## 典故 / 标志事件
- 陈桥兵变，黄袍加身。
- 杯酒释兵权。
- 烛影斧声（暗示其死因可能与弟赵光义有关）。
- 先南后北的统一策略。
- 设立封桩库意图赎回燕云十六州。

## 关键作品
- 建立宋朝，结束五代乱局，开启北宋时代。
- 推行中央集权制度改革，削弱地方藩镇势力。
- 制定并实施“先南后北”的统一方略，奠定宋朝疆域基础。
- 整顿禁军，强化中央军事力量。

## 历史评价
- 正面: 结束五代割据，开创宋朝，基本统一南方；通过杯酒释兵权等举措加强中央集权，稳定政局；被视为有作为的开国君主。
- 负面: 陈桥兵变被视为篡位；其死因“烛影斧声”成谜，弟继兄位违反传统，引发猜测；对北汉援军降卒有过坑杀行为。
- 后世神话/演义: 常被描绘为天命所归的真命天子（如黄袍加身）；“烛影斧声”成为其死亡之谜的文学典故。

## 反差 / 鲜为人知点
- 早年游历曾投奔复州防御使王彦超和随州刺史董宗本，均不被接纳或遭欺侮，经历坎坷。
- 在高平之战中曾中流矢受伤。
- 其设立的“封桩库”最初目标非常具体：积攒钱财赎回被石敬瑭献给契丹的燕云十六州，而非一般意义上的国库储备。
