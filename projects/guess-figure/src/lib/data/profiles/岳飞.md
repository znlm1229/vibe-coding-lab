# 岳飞

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字鹏举 / 谥号武穆 / 别号岳武穆
- 生卒年 / 朝代区间 / 籍贯: 1103年3月24日—1142年1月27日 / 北宋末年至南宋初期 / 相州汤阴（今河南汤阴）
- 主要职业 / 身份: 军事家、抗金名将、民族英雄

## 主要事迹
- 北宋末年从军，在宗泽麾下抗金，于滑州以百余骑击退金军，升秉义郎。[重要]
- 建炎元年上书宋高宗反对南迁，力主北伐，因越职言事被夺官。[一般]
- 建炎四年收复建康（今南京），取得对金作战的重大胜利，升任通、泰镇抚使。[重要]
- 绍兴四年（1134年）首次北伐，成功收复襄阳六郡，奠定南宋中线防御基础。[重要]
- 绍兴五年（1135年）平定洞庭湖杨幺起义，收编其部众，增强了岳家军实力。[重要]
- 绍兴十年（1140年）挥师北伐，取得郾城、颍昌大捷，进军至朱仙镇。[重要]
- 因宋高宗执意求和，被十二道金字牌强令班师，北伐成果毁于一旦。[重要]
- 绍兴十一年（1141年）被解除兵权，任枢密副使。[一般]
- 同年遭秦桧、张俊等人诬陷，以“莫须有”罪名下狱。[重要]
- 绍兴十二年（1142年1月）与长子岳云、部将张宪一同遇害于临安大理寺狱中。[重要]

## 性格 / 风格特征
- 治军极严，岳家军以“冻死不拆屋，饿死不掳掠”著称，深得民心。
- 性格刚直，敢于直言进谏，曾因反对南迁和议而触怒皇帝与权臣。
- 用兵灵活，不拘泥古法，认为“阵而后战，兵法之常，运用之妙，存乎一心”。

## 典故 / 标志事件
- 岳母刺字（“精忠报国”）。
- 十二道金牌催令班师。
- 以“莫须有”罪名被冤杀。
- 岳家军“冻死不拆屋，饿死不掳掠”。
- 收复建康，取得对金作战关键胜利。

## 关键作品
- 词作《满江红·写怀》（怒发冲冠）。
- 其奏议、诗文等被编为《岳忠武王文集》。
- 军事上收复襄阳六郡，建立稳固防线。
- 书法作品如《书谢朓诗》（存疑，但常被提及）。

## 关系网
- 老师：周同（射术老师）、宗泽（早期赏识并教导其兵法）
- 同辈/部将：张宪（重要部将）、王贵（部将）、牛皋（部将）
- 政敌：秦桧（主和派核心）、张俊（同僚，后参与构陷）
- 家人：岳云（长子，一同遇害）、岳雷（次子）

## 历史评价
- 正面：南宋及后世尊为民族英雄，其精忠报国精神成为楷模；治军严明，战功卓著，是南宋中兴名将的代表。
- 负面：在当时，主和派及宋高宗视其坚持北伐为“跋扈”与“干政”，成为和议的障碍。
- 后世神话/演义：在《说岳全传》等文学作品中形象被高度神化，成为忠义化身，衍生出许多传奇故事（如枪挑小梁王）。

## 反差 / 鲜为人知点
- 岳飞并非出身贫苦农家，其家庭属于有能力接济乡邻的自耕农，父亲岳和在当地颇有声望。
- 早期曾因与主将王彦意见不合而脱离部队单干，一度面临军法处斩的风险，后被宗泽赦免。
- 在讨伐杨幺水军时，其部下多为北方人不习水战，他采用“以水寇攻水寇”的策反分化策略，迅速平定叛乱，展现了卓越的政治谋略。
