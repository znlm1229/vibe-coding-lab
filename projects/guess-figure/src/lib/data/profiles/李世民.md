# 唐太宗

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 文皇帝 / 太宗 / 天可汗
- 生卒年 / 朝代区间 / 籍贯: 598年—649年 / 唐朝 / 陇西成纪（今甘肃秦安）
- 主要职业 / 身份: 唐朝第二位皇帝，军事家，政治家

## 主要事迹
- 隋大业十一年（615年），十六岁时于雁门关献疑兵之计，助解隋炀帝之围。[一般]
- 隋末随父李渊于太原起兵反隋，任右大都督，在攻取霍邑等关键战役中发挥重要作用。[重要]
- 唐朝建立后，作为主要军事统帅，先后率军平定薛仁杲、刘武周、王世充、窦建德等割据势力，统一全国。[重要]
- 武德九年（626年），发动玄武门之变，诛杀太子李建成、齐王李元吉，随后被立为太子并即位。[重要]
- 在位期间，虚心纳谏，厉行节约，休养百姓，开创“贞观之治”。[重要]
- 贞观四年（630年）灭东突厥，被北方及西域诸族尊为“天可汗”。[重要]
- 设立文学馆、弘文馆，延揽文士，讨论经义，推动文化发展。[一般]
- 贞观二十三年（649年），病逝于翠微宫含风殿。[次要]

## 性格 / 风格特征
- 临机果断，富有谋略，善于采纳正确建议（如雁门献计、霍邑劝进）。
- 作战勇猛，常身先士卒，率轻骑冲锋陷阵（如救李渊于敌阵、虎牢关之战）。
- 心胸开阔，能推心置腹任用降将（如重用尉迟敬德）。
- 具备政治远见与权术，为达目的手段果决（如策划并发动玄武门之变）。

## 典故 / 标志事件
- 雁门献计解围。
- 霍邑之战哭谏不退兵。
- 浅水原之战平定薛仁杲。
- 虎牢之战一战擒两王（王世充、窦建德）。
- 发动玄武门之变。

## 关键作品
- 开创“贞观之治”的治国政绩。
- 其书法真迹《晋祠之铭并序》碑刻。
- 组织编纂《晋书》、《北堂书钞》等典籍（作为帝王推动）。
- 其言论与治国思想被记录于《贞观政要》。

## 历史评价
- 正面：被誉为中国古代杰出的政治家与军事家，开创了国力强盛、社会安定的“贞观之治”，善于纳谏，用人唯才。
- 负面：其皇位通过发动玄武门之变，弑兄杀弟、逼父退位获得，在权力争夺中手段残酷。
- 后世神话/演义：在《隋唐演义》等文学作品中，被塑造为“真命天子”形象，其麾下秦琼、尉迟恭等成为门神，故事广为流传。

## 反差 / 鲜为人知点
- 幼年时曾患重病，其父李渊为此赴草堂寺祈福，愈后造石佛供奉，李世民后来作诗赞颂草堂寺祖师鸠摩罗什。
- 他明确规定“世民”二字不连续时无须避讳，以纠正当时两字兼避造成文字废阙过多的弊端。
- 在虎牢之战前，曾力排众议（包括萧瑀、屈突通等重臣的退兵建议），坚持主动出击，最终取得关键胜利。
