# 道光

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 旻宁 / 宣宗 / 成帝
- 生卒年 / 朝代区间 / 籍贯: 1782年-1850年 / 清朝中期 / 北京
- 主要职业 / 身份: 清朝皇帝

## 主要事迹
(按时间序)
- 1821年即位，开启道光时代 [重要]
- 1828年平定张格尔之乱，严禁新疆与浩罕通商 [重要]
- 1839年任命林则徐为钦差大臣赴广东查禁鸦片 [重要]
- 1840年鸦片战争爆发，英军封锁广州 [重要]
- 1842年签订《南京条约》 [重要]
- 1844年签订《望厦条约》、《黄埔条约》 [一般]
- 1846年正式解除对天主教的禁令 [次要]
- 1847年平定新疆七和卓之乱 [一般]
- 1850年去世 [次要]

## 性格 / 风格特征
(源自史料记载)
- 节俭，甚至近乎吝啬，宫廷开支大幅缩减
- 优柔寡断，在鸦片战争和议和决策上反复摇摆
- 保守，对西方技术和思想持排斥态度

## 历史评价
- 正面: 试图整顿吏治，节俭治国，平定西北叛乱
- 负面: 鸦片战争失败，签订不平等条约，治国能力不足
- 后世神话/演义: 常被描绘为懦弱、无能的皇帝，鸦片战争的关键失败者

## 反差 / 鲜为人知点
(隐晦信息)
- 道光帝极度节俭，裤子破了补丁继续穿，宫廷饮食简朴
- 鸦片战争期间曾一度主张强硬，后又转向议和，决策反复
- 在位期间西藏阿里地区爆发森巴战争，清军击败锡克帝国多格拉兵团
