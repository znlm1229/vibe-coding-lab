# 秦始皇

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 无 / 无 / 祖龙
- 生卒年 / 朝代区间 / 籍贯: 前259年-前210年 / 战国末期秦国君主、秦朝开国皇帝 / 嬴城（今山东莱芜）
- 主要职业 / 身份: 君主、皇帝

## 主要事迹
- 前259年生于赵国邯郸，为秦质子之子。[一般]
- 前247年13岁即位秦王，国政初由吕不韦主持。[一般]
- 前239年平定其弟长安君成蟜（或蒲鶮）在屯留的叛乱。[次要]
- 前238年平定嫪毐之乱，车裂嫪毐，囚禁太后赵姬。[重要]
- 前237年罢免吕不韦相国职务，后吕不韦自杀。[一般]
- 前230年至前221年，重用李斯、尉缭等，发动灭六国之战，统一天下。[重要]
- 前221年创立“皇帝”称号，自称“始皇帝”，建立秦朝，推行中央集权与郡县制。[重要]
- 统一后推行“车同轨，书同文，行同伦”，统一度量衡与典章法制。[重要]
- 在位期间修筑长城、阿房宫、骊山陵等大型工程，征发大量徭役。[重要]
- 前210年于出巡途中驾崩。[一般]

## 性格 / 风格特征
- 雄才大略，有强烈的权力欲望与开创意识（首创皇帝称号，欲传万世）。
- 行事果决狠辣（对嫪毐、吕不韦及叛乱者处置严厉，囚禁生母）。
- 施政急躁，追求宏大功业（短期内完成统一并推行多项急政大型工程）。

## 典故 / 标志事件
- 奇货可居（吕不韦视其父异人为可投资的“奇货”）。
- 嫪毐之乱（太后宠臣嫪毐叛乱，被秦王政平定）。
- 焚书坑儒（统一后为控制思想，焚烧典籍、坑杀儒生）。
- 修筑长城（连接并扩建各国长城以御匈奴）。
- 沙丘驾崩（于出巡途中病逝于沙丘平台）。

## 关键作品
- 建立秦朝（中国第一个中央集权制统一王朝）。
- 推行郡县制与中央集权制度（奠定后世两千余年政治格局）。
- 统一度量衡、文字、车轨、法律（“车同轨，书同文，行同伦”）。
- 修筑万里长城（成为后世重要防御工程与文化象征）。
- 修建骊山陵（其陵墓规模宏大，包括兵马俑等）。

## 关系网
- 老师 / 同辈: 吕不韦（早期辅政者，称仲父） / 成蟜（弟，长安君）
- 弟子 / 政敌: 无 / 嫪毐（政敌，叛乱者）
- 家人: 秦庄襄王（父，异人/子楚） / 赵姬（母，太后） / 扶苏（长子）

## 历史评价
- 正面: 被明代李贄誉为“千古一帝”，奠定中国统一基础与专制政治格局。
- 负面: 施政急躁，徭役过重，严刑峻法，被认为是秦朝速亡的重要原因。
- 后世神话/演义: 常被描绘为暴君形象，亦有求仙长生、派遣徐福东渡等传说。

## 反差 / 鲜为人知点
- 身世存疑，据《史记·吕不韦列传》记载，可能是吕不韦的私生子（赵姬怀孕后嫁予庄襄王）。
- 早期秦国王政受三大外戚势力（楚系、韩系、赵系）倾轧影响，并非完全掌控。
- 嫪毐封长信侯的原因可能与其平定成蟜（或蒲鶮）叛乱有关，而非仅因太后宠幸。
