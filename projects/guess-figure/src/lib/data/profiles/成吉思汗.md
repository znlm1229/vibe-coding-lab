# 成吉思汗

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 圣武皇帝 / 太祖 / 成吉思汗
- 生卒年 / 朝代区间 / 籍贯: 约1162年—1227年8月25日 / 蒙古帝国 / 蒙古乞颜部孛儿只斤氏
- 主要职业 / 身份: 蒙古帝国创建者、军事征服者、大汗

## 主要事迹
- 约1187年，在十三翼之战中败于盟友札木合，后可能短暂臣服于金朝。[一般]
- 1203年，在合兰真沙陀之战中遭盟友脱斡邻勒突袭，撤退后在班朱尼河盟誓重整，最终击败脱斡邻勒。[重要]
- 1206年，于忽里勒台大会被推举为“成吉思汗”，建立大蒙古国。[重要]
- 1209-1210年，率军入侵西夏，迫使其接受蒙古条件。[一般]
- 1211-1215年，发动对金朝的战争，最终攻占中都（今北京）。[重要]
- 1219年起，因花剌子模处决蒙古使者，发动西征，摧毁花剌子模王朝。[重要]
- 1227年，在征讨西夏叛乱期间去世。[一般]

## 性格 / 风格特征
- 对追随者极为慷慨忠诚，但对敌人冷酷无情。
- 优柔寡断且有恐犬症（据《蒙古秘史》记载）。
- 广纳各方建言，深信天命（腾格里）注定其统治世界。

## 典故 / 标志事件
- 少年时因猎物分配争执，与弟合撒儿合力杀死同父异母兄长别克帖儿。
- 与札木合互赠踝骨结为“安答”（结拜兄弟），后反目成仇。
- 班朱尼河盟誓，与少数忠诚部众饮浑水盟誓，重整旗鼓。
- 处决试图挑战其权威的萨满阔阔出，巩固权力。
- 西征起因是花剌子模王朝处决了他派去的和平使者。

## 关键作品
- 建立大蒙古国及一套任人唯贤、效忠皇室的国家体制。
- 发动大规模军事征服，奠定蒙古帝国疆域基础。
- 促成欧亚大陆间前所未有的商业与文化交流（客观上）。

## 历史评价
- 正面: 蒙古民族视为国家缔造者和开国之父；现代西方学界部分重新审视其促进文化交流的作用。
- 负面: 在俄罗斯与阿拉伯世界被视为野蛮暴君；军事征服造成数百万人口死亡。
- 后世神话/演义: 藏传佛教传入蒙古后曾被神化；出生手握血块、感光受孕等传说流传。

## 反差 / 鲜为人知点
- 据《蒙古秘史》记载，他有恐犬症。
- 其长子术赤的身世存在争议，《蒙古秘史》暗示其可能为私生子（孛儿帖曾被掳掠）。
- 他的确切出生年份（1155、1162或1167）连其本人可能都不清楚，史料记载不一。
