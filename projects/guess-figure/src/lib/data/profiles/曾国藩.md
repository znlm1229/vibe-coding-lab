# 曾国藩

## 基本信息

- 主要职业 / 身份: 晚清重臣、政治家、军事家、湘军创始人和领袖

## 主要事迹
(5-10 件, 按时间序, 每条结尾标 [重要]/[一般]/[次要])
- 道光十八年 (1838年) 中进士，入翰林院，开启仕途 [一般]
- 咸丰二年 (1852年) 奉旨帮办湖南团练，后创立湘军 [重要]
- 咸丰四年 (1854年) 湘潭战役，湘军首次大捷，扭转湖南战局 [重要]
- 咸丰十年 (1860年) 授两江总督、钦差大臣，督办江南军务 [重要]
- 同治元年 (1862年) 任协办大学士，统筹江苏、安徽、江西、浙江四省军务 [重要]
- 同治三年 (1864年) 督曾国荃部攻破太平天国首都天京 (南京)，平定太平天国 [重要]
- 同治七年 (1868年) 改任直隶总督，督办剿捻军务 [一般]
- 同治九年 (1870年) 处理“天津教案”，因妥协遭舆论谴责 [重要]
- 同治九年 (1870年) 调任两江总督，审理马新贻被刺案 [一般]
- 同治十一年 (1872年) 病逝于南京两江总督署 [次要]

## 性格 / 风格特征
(2-4 条, 源自史料记载, 不要泛泛而论)
- 治军严谨，主张“以霹雳手段，显菩萨心肠”，小过即斩，被称“曾剃头”。
- 处事谨慎，功成不居，平定太平天国后拒绝部下劝进造反。
- 重视人才与关系网络，依靠师徒、亲戚、好友关系建立湘军，门生遍布天下。

## 典故 / 标志事件
(3-5 个, 每个 1 句话, 这些后续用作 d1-5 banlist)
- 靖港水战失利后投水寻死，被幕僚章寿麟救起。
- 攻破安庆、天京后，湘军屠城，被南京百姓称为“曾剃头”、“曾屠户”。
- 处理天津教案时，屈从法国要求，杀民20人，遭舆论唾骂为“卖国贼”。
- 晚年立遗嘱处理天津教案，自知中国非列强对手，主张对外让步。
- 创立安庆内军械所，派人购买机器，是洋务运动早期实践者。

## 关键作品
(3-5 个, 文学/著作/政绩等)
- 《曾文正公全集》
- 创立湘军及配套水师、陆师制度。
- 支持并推动早期洋务运动 (如安庆内军械所、江南制造总局)。
- 《讨粤匪檄》
- 与胡林翼并称“曾胡”，其治兵思想被蒋介石编为《曾胡治兵语录》。

## 历史评价
- 正面: 被视为“晚清四大名臣”之一，平定太平天国，延续清朝统治；提倡洋务，开近代工业化先声；蒋介石、毛泽东早年皆曾推崇其人格与能力。
- 负面: 因湘军屠城 (安庆、天京) 被称为“曾剃头”、“曾屠户”；处理天津教案妥协，被骂“卖国贼”、“曾国贼”；其崛起被部分朝臣视为“恐非国家福也”。
- 后世神话/演义: 被塑造为儒家修身、齐家、治国的典范人物；其治军、用人、处世之道被广泛研究和宣扬；在民间和部分历史叙述中，其形象复杂，兼具“中兴名臣”与“刽子手”双重色彩。

## 反差 / 鲜为人知点
(1-3 个, 这是 d1 难线索的源, 必须是不在维基主条目首段的隐晦信息)
- 早年科考并不顺利，道光十五年 (1835年) 会试落第，次年恩科会试再次落第。
- 咸丰帝即位之初，曾国藩上《敬陈圣德三端预防流弊疏》直言批评皇帝，咸丰帝怒掷其奏折于地，幸得军机大臣祁隽藻解围。
- 其父曾麟书是塾师，屡试不第，年近五十方才中秀才，因此一心栽培长子曾国藩科考。
