# 雍正帝

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 胤禛 (名) / 雍亲王 (爵号) / 宪皇帝 (谥号简称) / 世宗 (庙号) / 破塵居士 (别号)
- 生卒年 / 朝代区间 / 籍贯: 1678年12月13日 - 1735年10月8日 / 清朝 (1722-1735在位) / 北京 (生于紫禁城)
- 主要职业 / 身份: 清朝皇帝

## 主要事迹
- 康熙六十一年（1722年）继承皇位，次年改元雍正。[重要]
- 推行“摊丁入亩”政策，将人头税并入土地税。[重要]
- 设立军机处，强化皇帝对军政事务的直接控制。[重要]
- 严厉整顿吏治，清查亏空，惩治贪腐官员。[一般]
- 推行“改土归流”，削弱西南地区土司势力，加强中央管辖。[一般]
- 处理“曾静投书案”，并编纂《大义觉迷录》以辩驳关于其篡位的传言。[一般]
- 与准噶尔部进行战争，稳固西北边疆。[次要]
- 雍正十三年（1735年）去世，其子弘历继位。[次要]

## 性格 / 风格特征
- 勤政务实，自称“以勤先天下”，每日批阅奏折至深夜。
- 性格严厉果决，对官员要求苛刻，执法严峻。
- 心思缜密，注重细节，亲自处理大量政务文书。

## 典故 / 标志事件
- 雍正即位引发“篡位”谣言，成为其执政生涯的阴影。
- 设立军机处，成为清朝中枢权力核心机构。
- 推行“摊丁入亩”的赋税制度改革。
- 编纂《大义觉迷录》以回应关于其得位不正的指控。
- 其去世原因存在争议，有传闻称死于丹药中毒。

## 关键作品
- 《雍正朱批谕旨》（大量政务批示）
- 《大义觉迷录》
- 推行“摊丁入亩”政策（重要政绩）
- 设立军机处（重要制度创设）

## 历史评价
- 正面：被视为勤政有为的改革者，整顿了康熙晚年的吏治松弛，为乾隆盛世奠定基础。
- 负面：被认为刻薄寡恩、手段残酷，其即位合法性长期受到质疑。
- 后世神话/演义：民间传说和部分文学作品常描绘其阴险、篡位形象，与正史记载形成反差。

## 反差 / 鲜为人知点
- 他拥有“破塵居士”等文人化的别号，与其严厉的政治形象形成反差。
- 其去世前可能长期服用道士炼制的丹药以求长生，这与其理性务实的执政风格相悖。
- 他对部分宠臣（如年羹尧）先极度信任后又严厉惩处，展现了其性格中复杂多变的一面。
