# 雍正帝

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 胤禛 / 雍亲王 / 宪皇帝 / 世宗 / 破尘居士
- 生卒年 / 朝代区间 / 籍贯: 1678年12月13日 - 1735年10月8日 / 清朝 (1722-1735在位) / 北京
- 主要职业 / 身份: 清朝皇帝

## 主要事迹
- 康熙六十一年十一月，康熙帝病危时被召至畅春园，受命继位。[重要]
- 即位后，任命胤禩、胤祥、马齐、隆科多总理事务，召回抚远大将军胤禵。[一般]
- 雍正元年，设立秘密立储制度，将立储诏书藏于乾清宫“正大光明”匾后。[重要]
- 雍正元年至二年，平定青海罗卜藏丹津叛乱，任命年羹尧、岳钟琪为将。[重要]
- 雍正二年，颁布《圣谕广训》，推行教化。[一般]
- 雍正三年，以“朝乾夕惕”书写问题为由，开始清算年羹尧，最终将其赐死。[重要]
- 雍正四年，整治胤禩集团，削爵、圈禁，并改其名为“阿其那”。[重要]
- 推行“摊丁入亩”政策，将丁银摊入地粮征收。[重要]
- 整顿吏治，清查各省仓库亏空，限三年补足。[一般]
- 在位期间，勤于政务，每日御乾清门听政。[次要]

## 性格 / 风格特征
- 勤政务实，每日御门听政，事必躬亲。
- 性格严峻，对臣僚要求苛刻，处理政敌手段强硬。
- 注重细节，对奏章文书中的措辞极为敏感（如年羹尧案）。
- 崇尚节俭，即位初期即停止直省贡献方物。

## 典故 / 标志事件
- 秘密立储（正大光明匾后藏诏）。
- 平定青海罗卜藏丹津叛乱。
- 整治年羹尧（朝乾夕惕案）。
- 整治胤禩集团（改名为阿其那）。
- 推行摊丁入亩政策。

## 关键作品
- 《圣谕广训》（推广康熙圣谕的官方诠释）。
- 《朋党论》（阐述反对臣僚结党的政治文章）。
- 建立秘密立储制度（影响清代皇位继承）。
- 推行“摊丁入亩”税制改革。
- 整顿吏治与财政（清查亏空）。

## 历史评价
- 正面: 勤政有为，整顿吏治，改革税制（摊丁入亩），巩固边疆，为乾隆盛世奠定基础。
- 负面: 性格严酷，对待兄弟与功臣手段残忍，大兴文字狱（如钱名世案），统治时期政治氛围紧张。
- 后世神话/演义: 民间多有关于其篡位（改诏）及暴毙的传说，在影视作品中常被塑造为权谋深沉、性格复杂的形象。

## 反差 / 鲜为人知点
- 他曾有“破尘居士”的别号，流露出对世俗事务超脱的文人倾向，与其严峻的政治形象形成反差。
- 即位初期曾下令州县举荐“老农”并给予顶戴荣誉，体现其对基层农业生产的重视。
- 对西洋事物有一定接触，曾将澳门的天主堂改为公所并严禁入教，但同时也接纳了暹罗国进贡的稻种果树。
