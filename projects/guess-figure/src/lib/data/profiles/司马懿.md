# 司马懿

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 仲达 / 无 / 宣文侯 (后追谥宣皇帝) / 高祖 / 晋宣帝
- 生卒年 / 朝代区间 / 籍贯: 179年—251年 / 东汉末年至三国曹魏时期 / 河内郡温县 (今河南温县)
- 主要职业 / 身份: 政治家、军事家、曹魏太傅、西晋奠基人

## 主要事迹
- 建安十三年（208年），曹操为丞相后，以强制手段征召司马懿为文学掾，司马懿畏惧而就职，开始辅佐曹氏。[一般]
- 建安二十四年（219年），关羽攻襄樊时，司马懿献策联合孙权，坐观虎斗，解樊城之围。[重要]
- 魏太和二年（227年），司马懿八日内行军一千二百里，迅速平定新城太守孟达叛乱。[重要]
- 魏太和五年至青龙二年（231年—234年），司马懿多次抵御诸葛亮北伐，据险坚守，最终诸葛亮病逝五丈原。[重要]
- 景初二年（238年），司马懿率军远征辽东，平定公孙渊叛乱，筑“京观”震慑。[重要]
- 正始二年（241年），司马懿督军南征，击退围攻樊城的吴将朱然、孙伦。[一般]
- 正始四年（243年），司马懿出征诸葛恪，并大兴屯田，广修水利，充实国力。[一般]
- 晚年发动高平陵之变（249年），诛杀曹爽，夺取曹魏政权。[重要]

## 性格 / 风格特征
- 深沉多谋，善于隐忍与伪装（早年以风痹症拒绝曹操征召）。
- 行事果断，机变迅捷（平定孟达时倍道兼行）。
- 谨慎持重，善于防守（对阵诸葛亮时据险不战，以逸待劳）。

## 典故 / 标志事件
- 司马懿早年装病风痹，拒绝曹操征召。
- 献策联孙抗刘，导致关羽败亡。
- 八日行军一千二百里，平定孟达叛乱。
- 对阵诸葛亮，接受巾帼妇人饰以激将，实则坚守不战。
- 远征辽东，平定公孙渊，筑京观。

## 关键作品
- 军事战略：抵御诸葛亮北伐，奠定曹魏西线稳定。
- 政治举措：发动高平陵之变，为司马氏篡权铺路。
- 经济政策：大兴屯田，修广漕渠，引河入汴，充实淮北粮储。
- 诗歌创作：远征辽东途中作《征辽东诗》（“天地开辟，日月重光…”）。

## 历史评价
- 正面：善于谋略，军事才能卓越，为曹魏中流砥柱；治国注重农桑，兴修水利。
- 负面：晚年篡权，为子孙代魏奠基；性格阴鸷，有“狼顾之相”；平定辽东后筑京观，手段残酷。
- 后世神话/演义：在《三国演义》中被塑造为诸葛亮的主要对手，形象多智而阴险；“死诸葛走生仲达”典故流传。

## 反差 / 鲜为人知点
- 司马懿曾因耻于与曹洪来往，假装拄拐生病拒绝其召辟，后因曹操征召立刻扔拐杖效命（据《魏略》）。
- 他早年曾劝曹操趁刘备初得益州时立即攻蜀，曹操未采纳，事后证明若采纳可能改变局势。
- 对阵诸葛亮时，他通过询问使者诸葛亮饮食起居细节（如每日食米三四升），判断其将不久于人世。
