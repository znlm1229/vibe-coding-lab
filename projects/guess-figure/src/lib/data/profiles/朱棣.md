# 朱棣

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 文皇帝 / 成祖 / 文殊皇帝
- 生卒年 / 朝代区间 / 籍贯: 1360年5月2日—1424年8月12日 / 明朝 / 南直隶凤阳府（今安徽凤阳）
- 主要职业 / 身份: 明朝皇帝（第三位）、政治家、军事家

## 主要事迹
- 洪武二十三年（1390年），率军北伐，招降北元乃儿不花部，崭露军事才能。[一般]
- 建文元年（1399年），起兵发动“靖难之役”，反抗建文帝削藩政策。[重要]
- 建文四年（1402年），攻入南京，夺取帝位，建文帝下落不明。[重要]
- 永乐三年（1405年），派遣郑和首次出使西洋，开启航海壮举。[重要]
- 永乐五年（1407年），主持编修《永乐大典》，成就大型典籍。[重要]
- 永乐八年（1410年），首次亲征漠北，打击蒙古势力。[一般]
- 永乐十九年（1421年），正式迁都北京，巩固北方统治。[重要]
- 永乐二十二年（1424年），于第五次北征途中病逝于榆木川。[次要]

## 性格 / 风格特征
- 雄才大略，军事指挥能力出众，善于把握战机。
- 行事果决狠辣，对政敌（如建文旧臣）清洗严厉。
- 崇尚武力，晚年多次亲征漠北，展现强烈进取心。

## 典故 / 标志事件
- 发动“靖难之役”，以“清君侧”为名夺取皇位。
- 派遣郑和下西洋，宣扬国威并进行海外探索。
- 编修《永乐大典》，汇集古今文献。
- 迁都北京，奠定明朝政治中心北移格局。
- 设立东厂，强化特务监察体系。

## 关键作品
- 《永乐大典》（主持编修）
- 迁都北京（重大政治决策）
- 郑和下西洋（主导海外活动）
- 五次亲征漠北（军事行动）

## 历史评价
- 正面: 开创永乐盛世，国力强盛，文化成就（如《永乐大典》）突出，拓展外交（如郑和下西洋）。
- 负面: 篡位夺权，清洗建文旧臣手段残酷，重用宦官特务机构（如东厂）。
- 后世神话/演义: 在藏传佛教文献中被尊称为“文殊皇帝”；民间对其篡位过程多有演绎。

## 反差 / 鲜为人知点
- 起兵前曾佯装精神失常以麻痹朝廷，但被王府长史告发。
- 靖难期间，曾利用建文帝“不得伤害叔父”的禁令，在战场上单骑穿行敌营以脱险。
- 其庙号原为“太宗”，百余年后（嘉靖十七年）才被改为“成祖”。
