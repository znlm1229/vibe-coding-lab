# 张居正

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字叔大 / 号太岳 / 谥文忠 / 无 / 人称张江陵
- 生卒年 / 朝代区间 / 籍贯: 1525年5月26日—1582年7月9日 / 明朝嘉靖至万历年间 / 湖广江陵县（今湖北江陵县）
- 主要职业 / 身份: 政治家、改革家、内阁首辅

## 主要事迹
- 嘉靖二十六年（1547年）中进士，步入仕途，并与严嵩、徐阶等重臣保持良好关系。[一般]
- 隆庆年间，与高拱并为宰辅，促成俺答封贡，实现与鞑靼和解互市。[重要]
- 万历元年（1573年）推行考成法，加强对官员的考核，提高行政效率，强化内阁权力。[重要]
- 万历五年（1577年）父亲去世，在神宗支持下“夺情”留任，遭到众多官员猛烈反对并强力压制。[重要]
- 万历六年（1578年）下令清丈土地，清查隐瞒庄田。[重要]
- 万历九年（1581年）在全国推行一条鞭法，改革赋税制度，政府财政状况改善。[重要]
- 重用戚继光、李成梁等名将练兵镇守边防，任用潘季驯治理黄河。[重要]
- 万历十年（1582年）病逝，赠上柱国，谥文忠，为明代唯一生前受封太师的大臣。[一般]
- 死后不久被清算，家被抄，长子自尽，其他家人遭流放或饿死，至天启、崇祯年间逐步平反。[次要]

## 性格 / 风格特征
- 机警多智，深沉有谋略（《明神宗实录》称“性沉深機警，多智數”）。
- 执政后期性格偏执，作风操切（因“夺情”事件后与多数官员为敌，行事越发严厉）。
- 自信过度，不能谦虚谨慎，不肯对事实做必要让步（黄仁宇评价）。
- 生活后期骄奢专权（如归葬时乘坐32人抬豪华大轿，菜肴过百品犹嫌无下箸处）。

## 典故 / 标志事件
- 夺情事件：父亲去世不丁忧守孝，遭群臣反对并廷杖谏者。
- 一条鞭法：合并赋役，按亩征银，改革税制。
- 考成法：建立账簿考核官员，提高行政效率。
- 俺答封贡：与鞑靼和解互市，结束北方战事。
- 厉声纠帝读：神宗误读“勃”字，张居正厉声纠正，吓得皇帝惊惶。

## 关键作品
- 《张太岳集》（亦称《张文忠公全集》）
- 《书经直解》
- 推行“一条鞭法”税制改革
- 创立“考成法”官员考核制度
- 提出“厚农而资商，厚商而利农”的经济观点

## 关系网
- 老师 / 前辈: 顾璘（赏识并磨砺他的乡试主考官）、徐阶（能保持良好来往的重臣）
- 同辈 / 合作者: 高拱（早期同为辅臣，后成为政敌）、冯保（宦官，合作逐高拱）
- 政敌: 高拱（后被逐）、众多反对“夺情”的官员（如邹元标）
- 家人: 张文明（父）、赵氏（母）、张敬修（长子）、张嗣修（次子）

## 历史评价
- 正面: 《明史》赞其执政期间“海内殷阜，纪纲法度莫不修明”；梁启超将其列为中国六大政治家之一；后世有“救时之相”之称。
- 负面: 被指责贪权不孝（夺情）、骄奢专权、钳制言论（禁毁书院、禁止批评时政）。
- 后世神话/演义: 较少被神话，更多作为改革家与权臣的复杂形象存在于历史讨论中。

## 反差 / 鲜为人知点
- 原名张白圭，12岁时被荆州知府李士翱赏识并改名为张居正。
- 13岁考举人时，赏识他的乡试主考官顾璘故意让他落榜，意在“磨砺”以防其过早得意忘形。
- 其经济思想强调商业与农业的平衡，提出“商不得通有无以利农，则农病；农不得力本穑以资商，则商病”，并非单纯重农抑商。
