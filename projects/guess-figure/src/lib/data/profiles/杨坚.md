# 杨坚

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 文帝 / 高祖 / 那罗延
- 生卒年 / 朝代区间 / 籍贯: 541年7月21日—604年8月13日 / 隋朝 (581-604在位) / 弘农郡华阴县（今陕西省华阴市）
- 主要职业 / 身份: 隋朝开国皇帝

## 主要事迹
- 581年，接受北周静帝禅让，建立隋朝，统一北方。 [重要]
- 589年，派兵灭陈，统一全国，结束长达273年的分裂局面。 [重要]
- 推行“大索貌阅”和“输籍法”，清查户口，增加国家税收。 [一般]
- 废除九品中正制，开创科举制度雏形，不问门第选官。 [重要]
- 修订《开皇律》，简化刑罚，死刑仅设绞、斩二等。 [一般]
- 583年，罢诸郡，实行州县二级制，简化地方行政。 [一般]
- 603-604年，击败突厥达头可汗，启民可汗称其为“圣人可汗”。 [一般]
- 晚年猜忌多疑，滥杀功臣虞庆则、史万岁，并废太子杨勇立杨广。 [次要]

## 性格 / 风格特征
- 天资刻薄，晚年猜忌多疑，滥杀大臣。
- 励精图治，前期澄清吏治，不容贪污。
- 生活崇尚俭约，曾以此告诫太子。

## 典故 / 标志事件
- 受禅建隋，得国之易无有如杨坚者。
- 灭陈统一，结束西晋永嘉之乱后273年分裂。
- 开创科举制度先河，选官不问门第。
- 修订《开皇律》，简化酷刑。
- 晚年废长立幼，立杨广而废杨勇。

## 关键作品
- 建立隋朝，统一中国。
- 推行科举制度雏形。
- 修订颁布《开皇律》。
- 实行州县二级制改革。
- 击败突厥，被尊为“圣人可汗”。

## 历史评价
- 正面: 结束长期分裂，统一全国；开创科举，改革律法行政，奠定隋唐盛世基础。
- 负面: 晚年猜忌滥杀，刑罚转苛；废立太子失策，被视为隋朝败亡原因之一。
- 后世神话/演义: 出生时有尼姑智仙取名“那罗延”（金刚不坏），带有神话色彩。

## 反差 / 鲜为人知点
- 其父杨忠被赐鲜卑姓，故杨坚又名“普六茹坚”，掌权后下令恢复汉姓。
- 夺取政权时，曾对矫诏让其辅政的建议“固辞，不敢当”，在刘昉威胁下才接受。
- 屠杀北周宇文皇族时，拒绝李德林劝谏，称“君书生，不足与议此！”，导致李德林此后官职不升。
