# 关羽

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字云长 / 壮缪侯 / 关公 / 关帝 / 关圣帝君
- 生卒年 / 朝代区间 / 籍贯: 约160年 - 220年 / 东汉末年 / 河东解县（今山西运城）
- 主要职业 / 身份: 将领，刘备集团重要成员，荆州总督

## 主要事迹
- 早年随刘备起兵，参与讨伐黄巾军及徐州争夺 [一般]
- 200年，被曹操擒获后归降，于白马之战中斩杀颜良，解白马之围，被封汉寿亭侯 [重要]
- 立功后封存曹操赏赐，留书告辞，重返刘备阵营 [重要]
- 208年，率水军接应刘备于汉津，助其撤退至夏口 [一般]
- 刘备入蜀后，长期镇守荆州，总督荆州事务 [重要]
- 219年，发起襄樊战役，水淹七军，擒于禁、斩庞德，威震华夏 [重要]
- 219年末，遭孙权偷袭荆州，败走麦城，于临沮被擒杀 [重要]

## 性格 / 风格特征
- 刚而自矜：对士大夫态度傲慢，曾辱骂孙权求婚使者。
- 护前（争强好胜）：听闻马超归降，写信向诸葛亮询问其才能可比何人。
- 忠义重诺：虽受曹操厚待，但誓言与刘备共死，立功后毅然离去。

## 典故 / 标志事件
- 白马斩颜良：单骑冲入敌阵斩杀袁绍大将颜良。
- 刮骨疗毒：箭伤有毒，治疗时臂血流离，仍饮酒谈笑自若。
- 水淹七军：利用汉水暴涨，淹没曹军七军，擒于禁、斩庞德。
- 败走麦城：荆州失守后，突围至麦城，最终被擒杀。
- 身在曹营心在汉：归降曹操期间，心念刘备，立功后即离去。

## 关键作品
- 襄樊战役（军事行动）
- 总督荆州（军政管理）
- 无传世文学作品（其形象主要通过历史记载与后世演义塑造）

## 关系网
- 同辈 / 盟友：刘备（主君，恩若兄弟）、张飞（同僚，飞兄事之）
- 政敌 / 对手：曹操（曾归降其阵营，后为敌）、孙权（联盟破裂后的主要对手）
- 家人：关平（子，一同被杀）

## 历史评价
- 正面：陈寿评其为“万人之敌”、“世之虎臣”，有“国士之风”；后世因其忠义勇武形象被逐步神化，尊为“武圣”。
- 负面：陈寿指出其“刚而自矜”，与士大夫不和；对部下（糜芳、士仁）轻慢导致后方失守。
- 后世神话/演义：在《三国演义》中被塑造为“义绝”代表，与刘备、张飞结义；道教尊为协天大帝，佛教奉为伽蓝菩萨。

## 反差 / 鲜为人知点
- 曾因粮草不足，私自夺取孙权辖下湘关的粮食，此举激化了与东吴的矛盾。
- 对马超归降心生嫉妒，专门写信向诸葛亮询问马超才能，需诸葛亮安抚。
- 其子关平在《三国志》中记载随其一同被杀，但演义中常被描绘为养子或义子，细节有出入。
