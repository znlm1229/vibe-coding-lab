# 关羽

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字云长，本字长生；谥号壮缪侯；后世尊称关公、关帝、关圣帝君等。
- 生卒年 / 朝代区间 / 籍贯: 约160年—220年1月或2月；东汉末年；河东解县（今山西运城）。
- 主要职业 / 身份: 将领，刘备重要亲信，镇守荆州。

## 主要事迹
- 早年随刘备起兵，参与讨伐黄巾军及早期军事活动 [一般]
- 200年，刘备败于曹操后，关羽一度被擒投降曹操 [一般]
- 同年，于白马之战中单骑冲锋斩杀颜良，解白马之围，被封汉寿亭侯 [重要]
- 后封存曹操赏赐，留书告辞，重返刘备阵营 [重要]
- 208年赤壁之战后，参与围攻南郡曹仁，长期镇守荆州 [重要]
- 214年刘备占领益州后，关羽董督荆州事 [一般]
- 219年，发起襄樊战役，水淹七军，擒于禁，斩庞德，威震华夏 [重要]
- 同年，东吴吕蒙偷袭荆州，关羽败走麦城，被擒杀 [重要]

## 性格 / 风格特征
- 刚而自矜：听闻马超归降，特意写信问诸葛亮马超才能如何，得知诸葛亮评价马超不及自己后大喜 [源自史料]
- 善待卒伍而骄于士大夫：与士卒关系良好，但对士大夫阶层态度傲慢 [源自史料]
- 重义守信：投降曹操后坦言受刘备厚恩誓共死，立功报答曹操后毅然离去 [源自史料]

## 典故 / 标志事件
- 白马斩颜良：单骑冲入敌阵斩杀袁绍大将颜良。
- 刮骨疗毒：手臂中箭毒入骨，治疗时刮骨去毒，谈笑自若。
- 水淹七军：利用汉水暴涨淹灭曹操援军于禁七军。
- 败走麦城：荆州被袭后退守麦城，最终被擒。
- 千里走单骑：离开曹操阵营重返刘备（演义典故，但体现其“义”）。

## 关键作品
- 镇守荆州，构建刘备集团在长江中上游的战略支点。
- 发起并指挥襄樊战役，一度威震华夏。
- 其忠义形象成为后世文学（如《三国演义》）及儒家文化的重要原型。
- 被道教尊为协天大帝、伏魔大帝，佛教奉为伽蓝菩萨，形成关公信仰。
- 民间与刘备、张飞“义结金兰”的故事广为流传。

## 关系网
- 同辈 / 兄弟：刘备（主君，情同兄弟）、张飞（同僚，兄弟之称） [同僚/兄弟]
- 政敌 / 对手：曹操（曾投降又离去）、孙权（最终袭杀者） [对手/政敌]
- 部下：糜芳（荆州守将，后投降孙权）、士仁（公安守将，后投降孙权） [部下]

## 历史评价
- 正面：忠义勇武，万人敌，后世尊为武圣，与“文圣”孔子齐名。
- 负面：刚而自矜，骄于士大夫，最终因性格缺陷导致败亡。
- 后世神话/演义：从历史将领逐步神化，成为儒释道共同尊奉的神祇，民间信仰广泛。

## 反差 / 鲜为人知点
- 关羽曾因粮草不足，擅自夺取孙权辖下湘关的粮食，此举破坏了与东吴的脆弱关系。
- 关羽与荆州守将糜芳、士仁素有嫌隙，曾扬言回军后要惩治他们，导致二人心怀不安最终投降孙权。
- 孙权一度有意不杀关羽，想留作对付刘备和曹操的战力，但近臣以“野狼不可豢养”为由劝杀。
