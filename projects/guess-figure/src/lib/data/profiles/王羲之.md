# 王羲之

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 逸少 / 无 / 无 / 无 / 王右军
- 生卒年 / 朝代区间 / 籍贯: 303年—361年 / 东晋 / 琅邪郡临沂县（今山东省临沂市）
- 主要职业 / 身份: 书法家、官员、道士

## 主要事迹
- 出身琅琊王氏望族，十岁时随家族南迁至金陵。 [一般]
- 二十一岁时，因“坦腹东床”的率真行为被太尉郗鉴选中，娶其女郗璿。 [一般]
- 仕途起步较晚，约二十三岁任秘书郎，后升任护军将军，关心士卒疾苦。 [一般]
- 永和九年（353年），于会稽山阴兰亭主持雅集，写下被誉为“天下第一行书”的《兰亭集序》。 [重要]
- 因不受朝廷重用且向往修道，于永和十一年（355年）称病去郡，辞官退隐。 [一般]
- 晚年长期服食药石以求长生，可能损害健康，于361年去世，享年59岁。 [次要]

## 性格 / 风格特征
- 率真质朴，不喜矫饰（如“坦腹东床”选婿事件）。
- 早年不善言辞，但被名士周顗慧眼识才。
- 恪尽职守，体恤下属（任护军将军时发布《临护军教》关心士卒）。
- 向往自然与修道，有隐逸之心。

## 典故 / 标志事件
- 坦腹东床（郗鉴选婿时露腹躺床，被选中）。
- 写经换鹅（为山阴道士抄写《黄庭经》换取白鹅）。
- 墨池练字（练字洗笔，池水尽黑）。
- 笔力入木三分（为皇帝写祭文，笔迹透入木板三分）。
- 窃读《笔论》（少年时偷读父亲枕中藏蔡邕《笔论》，书法大进）。

## 关键作品
- 《兰亭集序》（行书，被誉为“天下第一行书”）。
- 《快雪时晴帖》（晚年行草书尺牍）。
- 《黄庭经》（楷书，又称《换鹅帖》）。
- 《丧乱帖》、《孔侍中帖》等尺牍（唐代摹本，展现其书风）。

## 关系网
- 老师：卫夫人（书法启蒙老师）。
- 同辈：谢安（曾指导其书法，关系友好）。
- 弟子：无明确记载弟子，但其子王献之承袭书风。
- 政敌：王述（太原王氏出身，因门户之见势同水火）。
- 家人：父亲王旷，妻子郗璿，七子一女（子王献之著名）。

## 历史评价
- 正面：被尊为“书圣”，书法“飘若浮云，矫若惊龙”，唐太宗赞其“尽善尽美”。
- 负面：无显著负面评价，但史载其因门户之见轻视王述。
- 后世神话/演义：其书法故事（如换鹅、入木三分）广为流传，形象被神化为书法至高典范。

## 反差 / 鲜为人知点
- 其书法真迹已全部失传，现存均为后世摹本或拓本。
- 晚年因长期服食道家药石（可能为有毒矿物），可能反而损害健康导致早逝。
- 其妻郗璿极其长寿，活到九十岁，而王羲之本人仅五十九岁去世。
