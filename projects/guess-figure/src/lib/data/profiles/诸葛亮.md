# 诸葛亮

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字孔明 / 号卧龙 / 谥号忠武侯 / 无庙号 / 别号诸葛武侯
- 生卒年 / 朝代区间 / 籍贯: 181年-234年 / 东汉末年至三国时期 / 琅琊郡阳都县（今山东临沂沂南县）
- 主要职业 / 身份: 蜀汉丞相、政治家、军事家、发明家

## 主要事迹
- 躬耕南阳，自比管仲乐毅，结交名士徐庶、崔州平 [一般]
- 刘备三顾茅庐，诸葛亮提出《隆中对》（三分天下战略） [重要]
- 赤壁之战前，赴江东说服孙权联合抗曹 [重要]
- 刘备入川后，诸葛亮留守荆州，督三郡赋税以充军实 [一般]
- 刘备称帝后，拜诸葛亮为丞相，录尚书事，假节 [重要]
- 白帝城托孤，刘备嘱诸葛亮辅佐刘禅，可“自取” [重要]
- 平定南中叛乱，稳定后方 [一般]
- 五次北伐中原，发明木牛流马运粮 [重要]
- 234年病逝于五丈原军中 [重要]

## 性格 / 风格特征
- 谨慎持重：治国“科教严明，赏罚必信”，行军“戎陈整齐，号令明”
- 自视甚高：年轻时“每自比于管仲、乐毅”
- 廉洁自律：临终表言“不使内有余帛，外有赢财”
- 长于巧思：改进连弩、发明木牛流马、推演兵法作八阵图

## 历史评价
- 正面: 陈寿评“治戎为长，理民之干优于将略”；后世视为忠诚与智慧的典范。
- 负面: 陈寿亦指“奇谋为短”，北伐“所与对敌，或值人杰”，未能成功。
- 后世神话/演义: 《三国演义》塑造为“智绝”，近乎神算；民间传说赋予其通晓阴阳、料事如神形象。

## 反差 / 鲜为人知点
- 其妻为黄承彦之女，史载“黄头黑色”，时人谚嘲“莫作孔明择妇”。
- 诸葛亮早年并非纯粹隐居，其家族与袁术、刘表等地方势力有旧，叔父诸葛玄曾任豫章太守。
- 据《魏略》记载，诸葛亮曾主动求见刘备，而非完全被动等待“三顾”，与主流叙事有异。
