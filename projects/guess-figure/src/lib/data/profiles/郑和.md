# 郑和

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 无 / 无 / 三保太监
- 生卒年 / 朝代区间 / 籍贯: 1371年9月23日—1433年 / 明朝 / 云南昆阳县（今云南省昆明市晋宁区）
- 主要职业 / 身份: 航海家、外交家、军事家、宦官

## 主要事迹
- 靖难之役中立功，获赐姓郑，升任内官监太监。[重要]
- 永乐元年（1403年）出使暹罗国。[一般]
- 永乐二年（1404年）奉命出使日本，宣谕并促成勘合贸易。[一般]
- 永乐三年（1405年）至宣德八年（1433年），先后七次率领庞大船队远航西洋，访问三十余国。[重要]
- 在旧港（三佛齐国）击败海盗陈祖义，擒获献俘。[一般]
- 在锡兰山（今斯里兰卡）遭国王亚烈苦奈儿劫掠，率军攻破其城，生擒国王。[重要]
- 在苏门答剌平定伪王子苏干剌叛乱，擒获苏干剌。[一般]
- 永乐十年（1412年）任南京监工官，奉命重建南京大报恩寺。[次要]
- 宣德年间奉命修理南京宫殿及天地坛等祭祀建筑。[次要]

## 性格 / 风格特征
- 智略知兵，武功高强，行如虎步，声音洪亮（据相士袁忠彻记述）。
- 作为外交官，对其他宗教（如伊斯兰教、佛教、道教）采取包容和尊重的手法。
- 奉使多用中贵，强力敢任，劳绩显著（《明史》评价）。

## 典故 / 标志事件
- 靖难立功赐姓郑。
- 七次下西洋访问三十余国。
- 旧港擒海盗陈祖义。

## 关键作品
- 主导七次下西洋的航海壮举（史称“郑和下西洋”）。
- 督修重建南京大报恩寺。
- 奉命修理南京宫殿及祭祀坛庙。

## 历史评价
- 正面: 明代盛事，远航规模空前，宣扬国威，促进外交与贸易；劳绩与侯显相亚，为明初锐意通四夷的代表。
- 负面: 下西洋耗废不赀（耗费巨大），远不如永乐时盛况；其背景（波斯裔宦官）与士大夫圈子格格不入。
- 后世神话/演义: 民间故事《三宝太监西洋记通俗演义》将其旅行探险神化；后世形成郑和崇拜，与妈祖信仰关联。

## 反差 / 鲜为人知点
- 早年是穆斯林，后成为道衍和尚的菩萨戒弟子，法名福吉祥（速南吒释），并刻印佛经，自称“奉佛信官”，宗教信仰复杂。
- 除航海外，还曾担任南京守备太监，负责督修宫殿、寺庙等大型建筑工程。
- 关于其死亡与葬处存在争议，《明史》记载模糊，有学者认为1433年死于古里国，也有认为只是副使前往，南京牛首山墓可能仅为衣冠冢。
