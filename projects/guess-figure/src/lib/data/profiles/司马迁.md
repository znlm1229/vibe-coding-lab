# 司马迁

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字子长 / 号太史公 / 谥号无 / 庙号无 / 别号史迁
- 生卒年 / 朝代区间 / 籍贯: 约前145年—前86年 / 西汉 / 龙门（今陕西韩城）
- 主要职业 / 身份: 史学家、文学家、太史令

## 主要事迹
- 早年师从孔安国、董仲舒，并游历江淮、中原等地考察风俗、采集传说。[一般]
- 仕为郎中，奉命出使西南夷，负责在西南设郡。[一般]
- 继承父职任太史令，并与唐都、落下闳等人共同制定“太初历”，改以正月为岁首。[重要]
- 秉承父亲遗志，开始撰写《史记》，开创纪传体史学。[重要]
- 因替投降匈奴的李陵辩护，触怒汉武帝，被判死刑，后因无钱赎罪而受腐刑（阉割）。[重要]
- 在遭受腐刑后，发愤著书，最终完成《史记》这部巨著。[重要]
- 《史记》完成后，其外孙杨恽在汉宣帝时将书献出，使之得以流传。[一般]
- 卒年与死因不详，一说善终，一说因怨言下狱而死。[次要]

## 性格 / 风格特征
- 坚韧不屈：遭受腐刑后，忍辱负重，发愤完成《史记》。
- 秉笔直书：坚持史实真实性与客观性，不避讳帝王过失。
- 责任感强：深感身为太史令，有责任记载“明圣盛德”与“功臣世家贤大夫之业”。

## 典故 / 标志事件
- 李陵之祸：因替败降匈奴的李陵辩护，触怒汉武帝，遭受腐刑。
- 发愤著书：受刑后，效法古之贤圣，发愤完成《史记》。
- 太史公自序：在《史记》末尾撰写《太史公自序》，阐述著述动机与家世传承。
- 究天人之际，通古今之变：司马迁自述著史目的在于探究天人关系与历史变化规律。
- 史家之绝唱，无韵之离骚：鲁迅对《史记》兼具史学与文学巅峰价值的评价。

## 关键作品
- 《史记》：中国第一部纪传体通史，被后世奉为正史典范。
- 《报任安书》（又称《报任少卿书》）：书信体散文，阐述受刑后心境与著史决心。
- 《悲士不遇赋》：抒发士人不遇于时的悲愤之情。

## 关系网
- 老师：孔安国（经学）、董仲舒（儒学）
- 同辈：唐都（天文历法合作者）
- 弟子：无明确记载
- 政敌：汉武帝（因李陵事件及史书记载触怒）
- 家人：父亲司马谈（太史令）、女儿（嫁杨敞，生杨恽）

## 历史评价
- 正面：被誉为“史圣”、“历史之父”，《史记》被赞为“史家之绝唱，无韵之离骚”，其纪传体开创后世正史体例。
- 负面：当时因“诬罔”罪名受刑，其史书部分内容曾被汉武帝削毁。
- 后世神话/演义：有侍妾随清娱的传说；明代史学家柯维骐效法其专心著史而自宫。

## 反差 / 鲜为人知点
- 其先祖司马错是秦国名将，曾与张仪辩论伐蜀并成功实施，为秦统一奠定基础。
- 司马迁可能并非《史记》全部内容的原创者，部分内容系抄篡古书，序和论赞也可能有父亲司马谈所作成分。
- 他制定的“太初历”改变了秦以来以十月为岁首的习惯，奠定了此后两千年农历以正月为岁首的基础。
