# 乾隆帝

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 名弘历 / 号长春居士、信天主人、古稀天子、十全老人 / 谥号法天隆运至诚先觉体元立极敷文奋武钦明孝慈神圣纯皇帝 / 庙号高宗 / 别号乾龍、亁隆
- 生卒年 / 朝代区间 / 籍贯: 1711年9月25日-1799年2月7日 / 清朝（在位1735-1796年，实际掌权至1799年）/ 生于北京雍亲王府
- 主要职业 / 身份: 清朝第六位皇帝，中国历史上实际执掌国家最高权力时间最长的皇帝

## 主要事迹
- 雍正十三年（1735年）即位，次年改元乾隆，开启长达六十余年的统治。[重要]
- 乾隆十三年（1748年），平定大小金川土司叛乱，巩固了清廷对西南边疆的控制。[一般]
- 乾隆二十四年（1759年），平定准噶尔汗国与大小和卓叛乱，将新疆正式纳入清朝版图，史称“乾隆平定准噶尔”。[重要]
- 乾隆三十八年（1773年），下诏编纂《四库全书》，历时十余年，是中国古代规模最大的丛书。[重要]
- 乾隆四十六年（1781年），甘肃冒赈案发，乾隆帝严厉惩处涉案官员，震动朝野。[一般]
- 乾隆五十八年（1793年），英国马戛尔尼使团访华，乾隆帝在承德避暑山庄接见，但拒绝了英国的贸易与外交请求。[重要]
- 乾隆六十年（1795年），宣布立皇十五子永琰为皇太子，并于次年正月举行禅位大典，改元嘉庆，但退位后仍训政。[重要]

## 性格 / 风格特征
- 极度自负，晚年自诩为“十全老人”，总结自己一生有“十全武功”。（源自其自我评价及史料记载）
- 勤政且精力旺盛，在位前期励精图治，每日清晨必御乾清门听政，处理政务。（源自《清史稿》对其早期听政的记载）
- 性格复杂，前期宽严相济，后期转向严苛，尤其在处理文字狱及贪污案件时手段严厉。（源自其执政前后期政策对比）
- 酷爱艺术收藏与创作，审美偏好繁缛华丽，对瓷器、玉器、书画等有浓厚兴趣，并常在其收藏品上题字、钤印。（源自其艺术活动记载）

## 典故 / 标志事件
- 乾纲独断，在位期间将君主专制推向顶峰。
- 自诩“十全武功”，并撰《御制十全记》。
- 六次南巡，耗费巨资，巡视河工与民情。
- 编纂《四库全书》，同时大兴文字狱。
- 晚年禅位嘉庆，但仍以太上皇身份训政。

## 关键作品
- 主持编纂《四库全书》（文化工程）。
- 著有大量诗文，结集为《御制诗文全集》。
- 扩建圆明园，融合中西建筑风格（建筑政绩）。
- 平定新疆，设立伊犁将军，完成“拓地二万余里”的疆域整合（军事政绩）。

## 关系网
- 老师: 福敏（启蒙老师）
- 同辈: 和亲王弘昼（弟弟）
- 弟子: 嘉庆帝永琰（儿子兼继承人）
- 政敌: 前期辅政大臣鄂尔泰与张廷玉的党争双方（曾为其平衡驾驭的对象）
- 家人: 孝贤纯皇后富察氏（元配皇后）、孝圣宪皇后钮祜禄氏（生母）

## 历史评价
- 正面: 在位期间清朝达到鼎盛，国力强盛，版图辽阔，社会经济繁荣，史称“乾隆盛世”。
- 负面: 统治后期闭关自守，吏治腐败，六次南巡奢靡无度，文字狱严酷，社会矛盾积聚，盛世之下危机潜伏。
- 后世神话/演义: 民间传说与影视作品中常将其塑造为风流天子，有大量下江南的轶事和情感故事，与历史形象有较大出入。

## 反差 / 鲜为人知点
- 他可能是中国历史上写诗最多的人，一生作诗四万余首，但文学成就与数量极不匹配，罕有佳作流传。
- 虽严厉禁止官员贪污，但其宠臣和珅却成为清朝乃至中国历史上最大的贪官之一，形成巨大讽刺。
- 他极度推崇朱熹理学并以此治国，但其个人生活（如收藏、艺术趣味）却深受当时传入的欧洲“洛可可”风格影响，追求奢华繁复。
