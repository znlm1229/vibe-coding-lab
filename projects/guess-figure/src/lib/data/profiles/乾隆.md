# 乾隆帝

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 弘历 / 十全老人 / 纯帝 / 高宗 / 信天主人
- 生卒年 / 朝代区间 / 籍贯: 1711年9月25日 - 1799年2月7日 / 清朝 (1735-1796在位) / 北京
- 主要职业 / 身份: 清朝皇帝

## 主要事迹
- 1735年即位，次年改元乾隆，开启长达六十年的统治 [重要]
- 平定准噶尔、大小金川、回疆等叛乱，完成“十全武功” [重要]
- 编纂《四库全书》，整理保存古代典籍 [重要]
- 六次南巡，视察河工、笼络江南士绅 [一般]
- 1796年禅位给嘉庆帝，但仍继续执掌朝政直至去世 [重要]
- 晚年宠信和珅，吏治腐败加剧 [一般]

## 性格 / 风格特征
- 精力充沛，勤于政务，在位期间每日处理奏章 [重要]
- 喜好奢华，热衷收藏艺术品与大兴土木 (如扩建圆明园) [一般]
- 晚年自矜自满，自称“十全老人”，陶醉于盛世表象 [一般]

## 典故 / 标志事件
- 乾隆六次南巡，耗费巨大。
- 编纂《四库全书》，寓禁于征。
- 平定准噶尔，完成“十全武功”。
- 禅位嘉庆后仍训政，称“太上皇帝”。
- 宠信权臣和珅，导致吏治腐败。

## 关键作品
- 《四库全书》 (主持编纂)
- 《御制诗集》 (个人诗歌创作)
- 扩建圆明园 (建筑艺术成就)
- 平定边疆系列战役 (“十全武功”)
- 确立“密缄立储”制度 (政治制度)

## 关系网
- 老师 / 同辈 / 弟子 / 政敌 / 家人: 福敏 (老师) / 和珅 (宠臣/同辈) / 纪昀 (臣子/学者) / 无明确政敌 / 雍正帝 (父亲)、孝贤皇后富察氏 (妻子)、嘉庆帝 (儿子)

## 历史评价
- 正面: 统治前期励精图治，国力达到顶峰，疆域空前辽阔，文化事业 (如《四库全书》) 有所成就。
- 负面: 统治后期闭关自守、吏治腐败、奢靡浪费，盛世表象下危机暗藏。
- 后世神话/演义: 民间传说中常与香妃、刘墉斗智等故事关联，形象复杂多变。

## 反差 / 鲜为人知点
- 早年曾险些在木兰围场被熊袭击，康熙帝因其临危不乱而更加钟爱。
- 精通火器射击，曾专门向庄亲王允禄学习。
- 对欧洲艺术风格 (洛可可) 感兴趣，并影响了当时的宫廷建筑与装饰。
