# 陶渊明

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字元亮 / 号五柳先生 / 私谥靖节先生 / 无 / 无
- 生卒年 / 朝代区间 / 籍贯: 约365年—427年 / 东晋末年至南朝宋初期 / 寻阳郡柴桑县
- 主要职业 / 身份: 诗人、隐士、曾任地方官员

## 主要事迹
- 约393年，因家贫出任江州祭酒，数月后因不堪官场与刺史倨傲而辞职 [一般]
- 399年，任镇军将军刘牢之参军，次年可能为避孙恩之乱战事辞官 [一般]
- 401年，于荆州刺史桓玄幕下任职，同年因母丧辞职归乡服丧三年 [一般]
- 405年八月，出任彭泽县令，同年十一月因妹丧辞官，自此彻底归隐 [重要]
- 归隐后二十余年，拒绝一切官聘，躬耕隐居，与当地官员如王弘、颜延之交游但保持独立 [重要]
- 408年，家中失火，旧屋尽毁，移居寻阳城南郊南村 [次要]
- 420年刘裕篡晋后，诗文不用刘宋年号，只题甲子纪年，以示不满 [一般]
- 426年，拒绝江州刺史檀道济邀请出仕并谢绝馈赠 [一般]
- 427年，于故乡去世 [次要]

## 性格 / 风格特征
- 孤高达观，厌恶官场腐败与繁文缛节，毅然选择归隐躬耕
- 外儒内道，坚守儒家伦理，又受道家爱好自由、顺从自然思想影响
- 生活平凡好酒，几乎“篇篇有酒”，但交往中保持独立，时有拒绝官员求见之举

## 典故 / 标志事件
- 不为五斗米折腰（辞去彭泽县令）
- 采菊东篱下，悠然见南山（《饮酒》其五）
- 撰《桃花源记》描绘理想社会
- 撰《五柳先生传》自述生平志趣
- 归隐后拒绝檀道济出仕邀请并谢绝馈赠

## 关键作品
- 《归园田居》（五首田园诗代表作）
- 《饮酒》其五（“采菊东篱下”名篇）
- 《桃花源记》（散文，描绘理想世界）
- 《归去来兮辞》（辞赋，抒发归隐之志）
- 《五柳先生传》（自传体散文）

## 历史评价
- 正面: 被誉为“田园诗人之祖”、“隐逸诗人之宗”，诗风天然真淳，对后世白居易、苏轼等影响巨大。
- 负面: 其归隐选择在当时及后世亦有被视为消极避世或未能坚持仕途理想的议论。
- 后世神话/演义: 其形象被高度理想化为高洁脱俗的隐士典范，《桃花源记》成为后世绘画与文学再创作的常见题材。

## 反差 / 鲜为人知点
- 出身仕宦大族（曾祖父陶侃权倾一时），并非纯粹寒门，其归隐有家族经济基础（拥有田产与僮客）。
- 家族世奉天师道（属溪族），其思想背景除儒道外，亦有潜在宗教家庭传统影响。
- 其五次辞官经历中，有一次（400年）可能直接是为了逃避因孙恩之乱爆发的战事，并非全因厌恶官场常态。
