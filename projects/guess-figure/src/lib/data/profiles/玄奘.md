# 玄奘

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无
- 生卒年 / 朝代区间 / 籍贯: 602年-664年 / 隋末唐初 / 洛州缑氏县（今河南洛阳偃师）
- 主要职业 / 身份: 僧人、佛经翻译家、法相唯识宗奠基人、旅行家

## 主要事迹
1. 少年出家，因“意欲远绍如来，近光遗法”之言被破格录取 [重要]
2. 贞观三年（629年）私自西行求法，历经西域诸国抵达天竺 [重要]
3. 在那烂陀寺师从戒贤大师学习《瑜伽师地论》等五年 [重要]
4. 贞观十七年（643年）在曲女城无遮法会上立“真唯识量论旨”，无人辩难，获“大乘天”与“解脱天”尊号 [重要]
5. 贞观十九年（645年）携657部佛经返回长安，受到太宗热烈欢迎 [重要]
6. 同年六月开始译经，先后在弘福寺、大慈恩寺、玉华宫主持译场，一生译经论75部1335卷 [重要]
7. 麟德元年（664年）译完《大般若经》后感叹“吾生涯亦尽”，不久于玉华宫圆寂 [一般]
8. 口述并由弟子辩机撰成《大唐西域记》，成为研究古印度史的重要文献 [重要]

## 性格 / 风格特征
1. 意志坚定：为求佛法真义，不惜违禁西行，途中面对高昌王的强留，以绝食相抗。
2. 治学严谨：深感各地佛理说法不一，故远赴天竺寻求原典，译经力求“准确完备，力保原面”。
3. 不慕权贵：回京后因每日应付朝臣权贵的访问供养而身心疲惫，多次请求隐居山林译经。

## 典故 / 标志事件
1. 破格出家：因向大理卿郑善果表明“意欲远绍如来，近光遗法”的宏大志愿而被特许出家。
2. 绝食拒留：西行至高昌国时，国王麹文泰欲强留为国师，玄奘以绝食迫使对方让步。
3. 曲女城论战：在天竺曲女城无遮法会上立论，十八日无人能驳，声震五印度。
4. 太宗迎归：贞观十九年正月，玄奘归国时，唐太宗亲自出迎，给予极高礼遇。
5. 玉华译经：晚年于玉华宫译经，译完《大般若经》后自知命尽，罢笔而逝。

## 关键作品
1. 翻译佛经论75部1335卷，如《大般若经》、《瑜伽师地论》等。
2. 口述并成书《大唐西域记》，是研究古代中亚与印度地理历史的关键文献。
3. 开创中国佛教法相唯识宗，其学说深刻影响了后世佛教思想。

## 关系网
- 老师：戒贤大师（那烂陀寺住持）
- 同辈：那烂陀寺诸多学僧
- 弟子：辩机（撰《大唐西域记》）、众多译场助手
- 支持者：唐太宗李世民、唐高宗李治、高昌王麹文泰
- 家人：二哥长捷法师（早年于洛阳净土寺出家）

## 历史评价
- 正面：被誉为中国历史上最伟大的译经师之一，其西行求法、翻译经典、创立宗派的行为被视为虔诚、坚毅与智慧的典范。《大唐西域记》具有极高的史料价值。
- 负面：部分学者（如陈寅恪）认为其译文过于艰涩生硬，不如鸠摩罗什的译本通俗易读。
- 后世神话/演义：在神魔小说《西游记》中成为主角“唐三藏”的原型，其形象被大幅神话和文学化。

## 反差 / 鲜为人知点
1. 出身官宦世家：其曾祖陈钦曾任东魏上党太守，祖父陈康为北齐国子监博士，父亲陈惠在隋初曾任江陵县令，并非普通百姓。
2. 具备行政才能：少年时曾负责管理营建仁寿宫的五百丁匠，账目清晰无误，令总监杨素大惊。
3. 顶骨舍利流散争议：其顶骨舍利在1944年于南京发现后，部分流散至日本、台湾等地，关于其真伪与迁葬问题至今存在学术争议。
