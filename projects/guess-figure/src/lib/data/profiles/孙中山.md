# 孙中山

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字德明 / 号逸仙 / 化名中山樵 / 别号孙大炮 / 无
- 生卒年 / 朝代区间 / 籍贯: 1866年11月12日—1925年3月12日 / 清末至民国 / 广东香山（今中山市）
- 主要职业 / 身份: 政治家、革命家、医师

## 主要事迹
- 1892年毕业于香港西医书院，后在澳门、广州行医。[一般]
- 1894年上书李鸿章提出革新政治主张被拒，同年赴檀香山组织兴中会，提出“振兴中华”口号。[重要]
- 1905年在日本东京组成中国同盟会，被推为总理，提出“驱除鞑虏，恢复中华，建立民国，平均地权”政纲及三民主义学说。[重要]
- 1911年武昌起义后，被推选为中华民国临时大总统，于1912年1月1日在南京宣布就职，建立中华民国临时政府。[重要]
- 1913年因宋教仁案起兵讨袁，失败后流亡日本，1914年在日本建立中华革命党。[一般]
- 1917年因段祺瑞解散国会，在广州组织护法军政府，获选为大元帅。[一般]
- 1924年1月召开中国国民党第一次全国代表大会，通过宣言，改组国民党。[重要]
- 1924年底为推动南北议和北上北京，1925年3月12日因胆囊癌病逝于北京。[一般]

## 性格 / 风格特征
- 幼年即具反抗精神，曾质疑一味背诵的教学方式，并主张反对缠足、蓄婢和赌博。
- 性格坚定果敢，早年因捣毁村中北帝庙神像不为乡人所容而逃离家乡。
- 具有强烈救国热忱，常审视中国地图慨叹“如此江山，付之非人”，并自称“革命言论之时代”。

## 典故 / 标志事件
- 捣毁翠亨村北帝庙神像。
- 上书李鸿章未被接见。
- 在檀香山成立第一个现代革命团体兴中会。
- 就任中华民国临时大总统。
- 因陈炯明事变退居上海。

## 关键作品
- 《上李鸿章书》
- 《实业计划》
- 主持制订《中华民国临时约法》
- 提出三民主义学说
- 创办《民报》、《建设》杂志宣传革命

## 关系网

- 弟子 / 政敌 / 家人: 宋教仁 (国民党同僚) / 袁世凯 (政敌) / 孙眉 (兄长)

## 历史评价
- 正面: 被尊为“中华民国国父”，中华人民共和国政府尊其为“中国民主革命的伟大先行者、杰出的爱国主义者和民族英雄”。
- 负面: 革命过程中多次遭遇挫折与失败，如讨袁失败、护法运动受军阀挟制去职。
- 后世神话/演义: 常被塑造为坚定不移的革命领袖形象，其早年轶事（如“石头仔”）亦流传。

## 反差 / 鲜为人知点
- 童年家境贫寒，自称“到15岁才有鞋穿”，主要食物是白薯。
- 曾因加入基督教意向强烈，劝说华侨工人勿拜关帝像，被兄长断绝资助送回家乡。
- 在西医书院学习期间，曾潜心研究“知难行易”学说，并跟随康德黎学打板球。
