# 谢安

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字安石 / 号东山 / 谥文靖 / 无 / 谢太傅
- 生卒年 / 朝代区间 / 籍贯: 320年—385年 / 东晋 / 陈郡阳夏（今河南太康）
- 主要职业 / 身份: 政治家、军事家

## 主要事迹
- 隐居东山，清谈吟诗，后因家族需要出仕桓温幕府 [一般]
- 桓温欲篡位时，镇定赴新亭会见，化解危机 [重要]
- 拖延桓温加九锡的诏书起草，直至桓温病故 [一般]
- 调和桓氏与晋室矛盾，达成“荆扬相衡”局面 [重要]
- 举荐侄子谢玄组建北府兵，加强长江下游防御 [重要]
- 淝水之战前镇定自若，稳定人心，战后淡然回应“小儿辈大破贼” [重要]
- 淝水之战后主持北伐，收复黄河以南大片领土 [重要]
- 功成名就后主动交权，出镇广陵，激流勇退 [一般]

## 性格 / 风格特征
- 从容镇定，临大事而不乱（如新亭会见桓温）
- 思虑深远，善于调和各方矛盾（平衡桓谢关系、荆扬相衡）
- 淡泊名利，功成后不恋权位（主动出镇广陵）
- 鉴赏力超群，评人论事精准（评刘牢之、王味之、顾恺之画）

## 典故 / 标志事件
- 东山再起（隐居后复出仕）
- 新亭会桓温（从容化解篡位危机）
- 围棋赌墅（淝水战前镇定弈棋）
- 小儿辈大破贼（淝水战后淡然回应）
- 谢公墩（王安石戏咏与其争地名）

## 关键作品
- 主持淝水之战并取得胜利
- 北伐收复黄河以南领土
- 调和东晋内部矛盾，维持政局稳定
- 多首《兰亭诗》及《与王胡之诗》
- 书法被米芾赞为“自发淡古”

## 关系网
- 老师 / 同辈 / 弟子 / 政敌 / 家人 (各列 1-3 人，标关系类型)
- 王羲之（书法老师）
- 桓温（前期合作者，后期潜在政敌）、王坦之（政治盟友）
- 谢玄（侄子兼重要将领）、谢石（弟弟兼将领）
- 谢奕（兄长）、刘氏（夫人）

## 历史评价
- 正面: 被视为良相典范，江左风流宰相，调和内部矛盾、抵御外敌、功成身退。
- 负面: 功高震主引起皇室猜忌，淝水战功当时未获封赏。
- 后世神话/演义: 民间尊奉为神祇（广惠王、护国尊王、戏神/音乐神老郎神）。

## 反差 / 鲜为人知点
- 曾上奏表梁山伯与祝英台墓为“义妇冢”，因听其故事深受感动。
- 因鄙薄王珣为人，不惜与琅琊王氏嫡系交恶，让侄女和女儿离婚改嫁。
- 死后两百年，墓穴被陈朝始兴王陈叔陵挖开，扔其灵柩以葬自己母亲。
