# 崇祯

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 庄烈愍皇帝 / 思宗 / 无
- 生卒年 / 朝代区间 / 籍贯: 1611年-1644年 / 明朝末期 / 北京
- 主要职业 / 身份: 明朝皇帝 (末代)

## 主要事迹
- 1627年，其兄明熹宗驾崩后即位，次年改元崇祯。[一般]
- 1629年，清算魏忠贤阉党，定立“逆案”，惩处255人。[重要]
- 1629年，己巳之变，后金军队首次突破长城防线，兵临北京。[重要]
- 1636年，高迎祥兵败被俘而死，农民军势力一度受挫。[一般]
- 1641年，北京爆发大规模鼠疫，造成人口大批死亡。[次要]
- 1644年，李自成攻占北京，崇祯帝于煤山自缢身亡，明朝灭亡。[重要]

## 性格 / 风格特征
- 勤政，试图挽救明朝危局，但常事必躬亲，决策犹豫。
- 多疑，频繁更换内阁大臣与将领，导致朝政不稳。
- 节俭，崇尚朴素，试图以身作则扭转奢靡风气。

## 典故 / 标志事件
- 崇祯帝煤山自缢。
- 清算魏忠贤阉党，定立“逆案”。
- 己巳之变，后金兵临北京。
- 北京爆发崇祯年间大鼠疫。
- 任用徐光启等编修《崇祯历书》。

## 关键作品
- 支持编修《崇祯历书》（采用西法）。
- 在位期间官方编纂的政令文书。
- 无显著个人文学著作。

## 历史评价
- 正面: 被认为勤政、节俭，试图力挽狂澜，非昏庸之主。
- 负面: 被认为性格多疑、优柔寡断，用人不专，加速了明朝灭亡。
- 后世神话/演义: 在民间传说和文学作品中常被描绘为悲剧性的末代皇帝，其自缢场景被反复渲染。

## 反差 / 鲜为人知点
- 其年号“崇祯”是从内阁拟定的多个备选年号（如“乾圣”、“兴福”）中圈选出来的，并非其独创。
- 明朝灭亡后，朝鲜王朝出于反清及“小中华”思想，在国内长期私下使用“崇祯”年号，称“崇祯纪元”。
- 清朝同治年间（1863年），甘肃回民起义首领穆生华起兵反清时，曾使用明朝崇祯年号，以示反清复明。
