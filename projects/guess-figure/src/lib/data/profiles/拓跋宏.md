# 拓跋宏

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 孝文 / 高祖 / 无
- 生卒年 / 朝代区间 / 籍贯: 467年10月13日—499年4月26日 / 南北朝（北魏） / 平城（今山西大同）
- 主要职业 / 身份: 皇帝，改革家

## 主要事迹
- 471年，四岁时受禅即位，由父亲献文帝禅让，改元延兴。[一般]
- 490年，祖母冯太后去世后亲政，继承并深化其改革政策。[一般]
- 推行均田制、三长法等，整顿吏治与经济。[重要]
- 493年，以南伐为名，将都城从平城迁至洛阳。[重要]
- 迁都后，全面推行汉化改革：改汉姓（拓跋改元）、穿汉服、说汉语、定籍洛阳、鼓励胡汉通婚，并严厉镇压反对派（如处死太子元恂）。[重要]
- 参照南朝典章，修改北魏政治制度。[一般]
- 499年，在南征途中病逝于谷塘原行宫。[次要]

## 性格 / 风格特征
- 雄才大略，爱奇好士，视下如伤，役己利物（《魏书》评价）。[源自史料]
- 用法虽严，但对大臣的小过常多宽略，如食中得虫、侍者误伤其手皆笑而赦之（《资治通鉴》记载）。[源自史料]
- 生活节俭，宫室非不得已不修，衣弊浣濯而服之，鞍勒用铁木而已（《资治通鉴》记载）。[源自史料]

## 典故 / 标志事件
- 以南伐为名迁都洛阳。
- 推行汉化改革，改拓跋姓为元姓。
- 因太子元恂反对改革而将其处死。
- 与崔光关于“君子劳心，小人劳力”的对话。
- 临终前下令遣散嫔妃回家另嫁。

## 关键作品
- 孝文帝改革（汉化运动）。
- 推行均田制。
- 确立三长法。
- 迁都洛阳。
- 改定官制，参照南朝典章。

## 历史评价
- 正面：北齐《魏书》赞其“雄才大略”，“经纬天地”，汉化改革促进了民族融合与社会经济发展。
- 负面：改革激化了部分鲜卑贵族内部矛盾，被视为后来北魏动荡的潜在远因之一（传统观点）。
- 后世神话/演义：主要作为杰出的改革君主形象出现在历史叙述中，演义色彩不浓。

## 反差 / 鲜为人知点
- 幼年时力大善射，能以指弹碎羊骨，射禽兽无不命中，但十五岁后便不再畋猎。
- 临终前特别下令遣散了所有嫔妃，让她们回家另嫁。
- 其子元愉、元悦在后来的政治动荡中曾短暂自立为帝。
