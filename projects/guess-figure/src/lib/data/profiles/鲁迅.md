# 鲁迅

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字豫才 / 无 / 无 / 无 / 笔名鲁迅
- 生卒年 / 朝代区间 / 籍贯: 1881年9月25日—1936年10月19日 / 清末至民国 / 浙江绍兴
- 主要职业 / 身份: 作家、新文化运动领袖、设计师、教育家

## 主要事迹
- 1898年入读江南水师学堂，后转入矿路学堂，学习矿务并自学德语 [一般]
- 1902年赴日留学，入弘文学院，参与创办《浙江潮》并发表《中国地质略论》 [一般]
- 1904年入仙台医学专门学校学医，成为该校首位中国留学生 [一般]
- 1906年观看日俄战争幻灯片后决定弃医从文，转向文艺以改变国民精神 [重要]
- 1909年回国，任教于浙江两级师范学堂等多所学校 [一般]
- 1912年进入中华民国教育部任职，参与制定注音符号、设计国徽与北大校徽 [重要]
- 1918年以笔名“鲁迅”发表中国第一篇现代白话小说《狂人日记》 [重要]
- 1926年南下任厦门大学教授，后转任中山大学文学系主任兼教务主任 [一般]
- 1930年在上海参与发起中国自由运动大同盟，加入左翼作家联盟 [重要]
- 1931年起大力倡导木刻版画，推动中国创作版画发展 [一般]

## 性格 / 风格特征
- 性格刚毅，不畏权势，曾因带领学界抗争而被教育总长免职。
- 对传统与旧势力持批判态度，但在袁世凯病亡后不顾流言前往吊唁，显示其复杂立场。
- 扶植文学青年，但对同辈学者如顾颉刚的快速晋升感到不服并公开抵触。

## 典故 / 标志事件
- 观看日俄战争幻灯片后弃医从文。
- 与周作人兄弟失和，搬离八道湾四合院。
- 以笔名“鲁迅”发表《狂人日记》。
- 因带领学生运动被北洋政府教育总长章士钊免职。
- 与许广平在广州同居。

## 关键作品

- 《阿Q正传》
- 《呐喊》（小说集）
- 《朝花夕拾》（散文集）
- 《中国矿产志》（与顾琅合著）

## 历史评价
- 正面: 被誉为中国现代文学的奠基人，新文化运动的旗手，其杂文与小说深刻批判社会，影响深远。
- 负面: 部分左翼文学团体批评其“开倒车”、“青年的绊脚石”；与兄弟决裂、家庭关系处理引发争议。
- 后世神话/演义: 常被塑造为不屈不挠的文化战士形象，其生平细节（如兄弟失和原因）存在多种演义说法。

## 反差 / 鲜为人知点
- 他曾是清政府官费派遣的矿路学堂毕业生，并获得“五品顶戴”资格，最初的专业方向是矿业与地质。
- 他设计过中华民国国徽草案及北京大学校徽，并长期在教育部负责图书情报业务，并非纯粹的文学作家。
- 1932年淞沪抗战期间，他与家人曾两次躲进日本友人内山完造的书店避难。
