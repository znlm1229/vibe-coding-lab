# 刘彻

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 孝武 / 世宗 / 汉武帝
- 生卒年 / 朝代区间 / 籍贯: 公元前156年 - 前87年 / 西汉 / 长安
- 主要职业 / 身份: 皇帝

## 主要事迹
- 公元前134年采纳董仲舒建议，推行察举制选拔官吏 [重要]
- 公元前133年发动马邑之谋，开启对匈奴的长期战争 [重要]
- 公元前127年颁行“推恩令”，削弱诸侯势力 [重要]
- 公元前121年及119年，派遣卫青、霍去病大规模反击匈奴，控制河西走廊 [重要]
- 公元前122年平定淮南王、衡山王谋反 [一般]
- 公元前112年平定南越，将其纳入版图 [重要]
- 公元前108年灭卫氏朝鲜，设四郡 [一般]
- 晚年发生巫蛊之祸，太子刘据被逼自杀 [重要]
- 公元前89年颁布《轮台诏》，反省政策失误，转向休养生息 [重要]

## 性格 / 风格特征
- 雄才大略，积极进取，在位期间频繁对外用兵和开疆拓土。
- 晚年多疑迷信，导致宫廷内部斗争激烈（如巫蛊之祸）。
- 善于权术，通过设立内朝、推行推恩令等手段强化中央集权。

## 典故 / 标志事件
- 颁行推恩令削弱诸侯。
- 派遣卫青、霍去病北击匈奴。
- 采纳董仲舒建议“罢黜百家，表彰六经”。
- 晚年发生巫蛊之祸。
- 颁布《轮台诏》罪己。

## 关键作品
- 推行察举制，建立系统的人才选拔制度。
- 首用年号纪年（建元）。
- 颁布《太初历》，统一历法。
- 设立乐府，收集民间歌辞。
- 创建太学，作为国家教育机构。

## 关系网
- 老师 / 同辈 / 弟子 / 政敌 / 家人 (各列 1-3 人, 标关系类型)
- 董仲舒（采纳其政策建议的学者）
- 卫青、霍去病（重用的大将）
- 主父偃（献策的官员，后被杀）
- 刘据（太子，后在巫蛊之祸中被逼自杀）
- 窦太后（早期制约其权力的太皇太后）

## 历史评价
- 正面: 开拓疆土，加强中央集权，奠定汉帝国强盛基础，被视为雄主。
- 负面: 连年战争导致社会负担沉重，晚年多疑迷信引发宫廷惨剧。
- 后世神话/演义: 在文学和民间传说中常被描绘为求仙问道、与方士交往的形象。

## 反差 / 鲜为人知点
- 早期曾受窦太后制约，并非一开始就完全掌权。
- 曾因“酎金”事件一次性剥夺百余列侯爵位，手段严厉。
- 晚年建“思子宫”并作“归来望思之台”，对逼死太子刘据表示追悔。
