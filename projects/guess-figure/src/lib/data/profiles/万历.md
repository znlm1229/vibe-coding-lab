# 万历

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 显皇帝 / 神宗 / 无
- 生卒年 / 朝代区间 / 籍贯: 1563年-1620年 / 明朝 (1572年-1620年在位) / 北直隶顺天府 (今北京)
- 主要职业 / 身份: 明朝第十三位皇帝

## 主要事迹
- 1572年，十岁即位，初期由张居正辅政，推行改革，出现“万历中兴”局面。[重要]
- 1582年张居正死后，清算其家族，并逐渐疏于朝政。[一般]
- 1586年起，因“国本之争”（立太子问题）与朝臣严重对立。[重要]
- 1589年后，开始长期怠政，创下“三十年不上朝”的纪录。[重要]
- 在位期间，进行了“万历三大征”（宁夏之役、朝鲜之役、播州之役），均获胜但消耗国力。[重要]
- 派遣矿监税使四处敛财，激起民变。[一般]
- 晚年朝廷党争（东林党争）加剧，埋下明朝衰亡的祸根。[重要]

## 性格 / 风格特征
- 早年聪慧机敏，幼时即能劝谏父皇注意安全，并善解两宫心意。（源自《明史》记载）
- 性格倔强固执，因“国本之争”受挫后，采取消极怠工的方式进行对抗。
- 后期贪婪怠惰，深居宫中，沉迷酒色财气，对朝政敷衍塞责。

## 典故 / 标志事件
- 万历中兴（张居正改革期间的短暂繁荣）。
- 国本之争（长达十五年的立太子之争）。
- 万历怠政（创下皇帝三十年不上朝的纪录）。
- 万历三大征（平定宁夏哱拜、援朝抗倭、平定播州杨应龙）。
- 矿税之弊（派遣宦官充当矿监税使，横征暴敛）。

## 关键作品
- 万历会计录（户部编纂的国家财政统计报告）。
- 定陵（其陵墓，为明十三陵中唯一被考古发掘的帝陵）。
- 万历邸钞（当时的官方新闻公报）。
- 对瓷器艺术风格产生影响，万历年间的瓷器装饰趋向自由化。

## 历史评价
- 正面: 在位前期任用张居正改革，经济军事有所起色，有“中兴”气象；万历三大征维护了国家疆域稳定。
- 负面: 中后期长期怠政，导致政府机构瘫痪；加剧党争；横征暴敛激化社会矛盾；其统治被认为是明朝由盛转衰的关键节点。
- 后世神话/演义: 在民间戏曲和小说中，其形象多与“怠政”、“贪财”相联系，是昏君的典型之一；关于“国本之争”和宫廷秘闻的传说颇多。

## 反差 / 鲜为人知点
- 幼年时并非顽劣，反而表现出色，曾劝谏父亲穆宗皇帝不要在宫中驰马，担心其安全，得到嘉奖。
- 在位早期曾亲笔写下“谨天戒、任贤能、亲贤臣、远嬖佞”等十二事箴言置于座右，以自我警示，与后期形象反差巨大。
- 虽然不上朝，但并未完全放弃权力，仍通过“留中”（将奏疏搁置宫中不发）和批阅奏章等方式掌控重大决策，是一种独特的消极统治方式。
