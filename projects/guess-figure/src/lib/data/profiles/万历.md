# 万历

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 无 / 神宗 / 无
- 生卒年 / 朝代区间 / 籍贯: 1573年—1620年 / 明朝 / 无
- 主要职业 / 身份: 明朝皇帝 (明神宗朱翊钧的年号)

## 主要事迹
- 1573年，明神宗朱翊钧即位，启用张居正，开启万历朝 [重要]
- 万历前期，张居正推行改革，社会经济发展，史称“万历中兴” [重要]
- 万历十年（1582年），张居正去世，改革逐渐废止 [一般]
- 因“国本之争”（立太子问题）与大臣矛盾激化 [重要]
- 万历中期开始，长期怠政，近三十年不上朝 [重要]
- 万历年间，爆发“万历三大征”（宁夏之役、朝鲜之役、播州之役） [重要]
- 朝廷党争加剧，东林党争兴起 [一般]
- 万历四十八年（1620年），神宗去世，年号终止 [次要]

## 性格 / 风格特征
- 前期受张居正影响，支持改革，励精图治。
- 后期固执己见，因立储问题与群臣对抗。
- 怠政期间，对政务消极，但仍通过奏疏掌控关键决策。

## 典故 / 标志事件
- 万历中兴（张居正改革带来的短暂繁荣）。
- 国本之争（关于太子人选的长达数十年的政治斗争）。
- 万历怠政（明神宗近三十年不上朝）。
- 万历三大征（明朝在西北、东北、西南的三次大规模军事行动）。
- 东林党争（万历年间开始并加剧的朝廷党派斗争）。

## 历史评价
- 正面：前期开创“万历中兴”，军事上取得“三大征”胜利。
- 负面：后期长期怠政，加剧党争，被视为明朝衰亡的起点。
- 后世神话/演义：在民间故事中常被描绘为任性、懒惰的皇帝形象。

## 反差 / 鲜为人知点
- 怠政期间虽不上朝，但仍通过批阅奏疏等方式处理重要军政事务，并非完全放弃权力。
- 万历瓷器装饰风格一反传统，较少使用龙凤等皇室象征纹饰，形式更为自由活泼。
- 其年号“万历”因避乾隆帝“弘曆”讳，在清朝文献中常被改写为“萬歷”、“萬厯”或“萬厤”。
