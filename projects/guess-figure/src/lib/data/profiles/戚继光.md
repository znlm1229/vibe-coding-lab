# 戚继光

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字元敬 / 号南塘 / 谥号武毅 / 无 / 无
- 生卒年 / 朝代区间 / 籍贯: 1528年11月12日—1588年1月17日 / 明朝 / 山东登州
- 主要职业 / 身份: 军事家、将领

## 主要事迹
- 1553年，出任山东都指挥佥事，负责山东沿海防倭，以纪律严明著称。[一般]
- 1559年，在浙江义乌招募并训练三千乡勇，创立“鸳鸯阵”，组建戚家军。[重要]
- 1561年，在台州沿海九战九捷，大破倭寇，升任都指挥使。[重要]
- 1562-1563年，率军援救福建，收复平海卫、兴化府，肃清福建沿海倭寇。[重要]
- 1568年起，受命总理蓟州、昌平、保定防务，担任蓟州总兵十四年，重修长城，建造塔楼，防御蒙古。[重要]
- 1571年，上奏设立车营，引入战车对付蒙古骑兵，并提议修筑长城堡垒。[一般]
- 1582年张居正逝世后，屡遭弹劾，被罢免蓟州总兵，调任广东总兵，后免职回乡。[次要]

## 性格 / 风格特征
- 治军严谨，纪律严明，实行连坐法，赏罚分明。
- 注重实际，练兵讲求实效，只招募农民，摒弃城市“狡猾无赖”者。
- 作战计划周密，临阵前详细侦察，善用地图甚至地形模型。

## 典故 / 标志事件
- 创立“鸳鸯阵”以对抗倭寇。
- 在台州沿海一个月内“九战九捷”。
- 镇守蓟州期间重修长城，建造空心敌台。
- 因与张居正关系密切，在其死后遭弹劾罢官。
- 晚年罢官居家，积蓄甚少，一贫如洗。

## 关键作品
- 《纪效新书》（练兵与战术总结）
- 《练兵实纪》（蓟州守边经验总结）
- 整顿边防，重修蓟镇长城
- 组建并训练戚家军

## 历史评价
- 正面: 后世誉为抗倭功臣、一代名将，军事著作影响深远。
- 负面: 张居正死后被指为“宫门外的猛兽”，遭政治清算。
- 后世神话/演义: 被塑造为民族英雄和杰出的军事改革者形象。

## 反差 / 鲜为人知点
- 他最初在北方戍守时曾上书陈述防御蒙古的策略，但后来镇守蓟州时，在张居正指示下主要采取守势，内心其实想对蒙古采取攻势。
- 他对火器（鸟铳、火炮）的威力有所认识，但因当时明朝制造的火器质量太差（炸膛、弹不合规），实战中并不依赖，只有限应用。
- 他晚年罢官后境遇凄凉，积蓄甚少，医药费都难以维持，朋友稀少。
