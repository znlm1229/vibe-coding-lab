# 张衡

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字平子 / 无 / 无 / 无 / 无
- 生卒年 / 朝代区间 / 籍贯: 78年 - 139年 / 东汉 / 南阳郡西鄂县（今河南省南阳市南召县）
- 主要职业 / 身份: 科学家（天文学家、数学家、发明家）、文学家、官员（太史令、侍中、尚书）

## 主要事迹
- 约100年，出任南阳太守鲍德主簿，后辞官专心读书。 [一般]
- 111年，被汉安帝征召，出任尚书台郎中，后迁尚书侍郎。 [一般]
- 115年，出任太史令，开始研制天文仪器，后制作浑天仪。 [重要]
- 132年，发明并制造了世界上第一台验震器“候风地动仪”。 [重要]
- 133年，洛阳地震后上疏批评宦官越权，主张大权归还天子。 [一般]
- 同年迁任侍中，因受宦官排挤，136年外调为河间相，治理有方。 [一般]
- 138年升迁尚书，次年去世。 [次要]
- 一生致力于天文观测与研究，确立浑天说，绘制星图，计算圆周率。 [重要]
- 创作《二京赋》、《归田赋》等文学作品，拓展汉赋文体与题材。 [重要]
- 曾上《请禁绝图谶疏》，反对当时流行的谶纬预言。 [一般]

## 性格 / 风格特征
- 从容淡静，不好交接俗人，不慕当世（名利）。 (源自《后汉书》记载)
- 意志坚定，好学不倦，通晓五经六艺。 (源自《后汉书》记载)
- 为人谦虚，虽才高于世，而无骄尚之情。 (源自《后汉书》记载)
- 坚持批判现实，对宫廷政治的污浊深感忧愤。 (源自维基材料)

## 典故 / 标志事件
- 制作浑天仪，以水力驱动，可演示天体运行。
- 发明候风地动仪，能探测地震方向，曾成功预警陇西地震。
- 反对图谖，上疏请禁绝虚妄的谶纬预言。
- 创作《二京赋》，精思傅会，十年乃成，用以讽谏王侯奢侈。
- 出任河间相时，下车治威严，整法度，收禽奸党。

## 关键作品
- 《二京赋》（文学代表作）
- 《灵宪》（天文宇宙论著作）
- 《浑天仪》（天文仪器原理解说）
- 《算罔论》（数学著作）
- 《归田赋》（抒情小赋代表作）

## 关系网
- 老师 / 同辈: 崔瑗 (同辈好友，曾讨论《太玄》) / 鲍德 (曾为其主簿，后推荐其入朝)
- 弟子 / 政敌: 无明确记载 / 当权宦官 (因其上疏批评而共谗之)
- 家人: 祖父张堪 (蜀郡太守)

## 历史评价
- 正面: 崔瑗称其“数术穷天地，制作侔造化”。后世誉为“汉赋四大家”之一，杰出的科学家和文学家。
- 负面: 其圆周率计算被刘徽批评为粗疏；部分宇宙论和思想受谶纬和道家神秘主义影响。
- 后世神话/演义: 地动仪被后世赋予神话色彩，但其具体构造和原理后世失传，引发诸多猜测与复原尝试。

## 反差 / 鲜为人知点
- 他虽反对图谖，但其宇宙论（如《灵宪》所述）却深受当时流行的谶纬思想影响。
- 他相信“风角”之术，认为可通过观风而知祸福，并非纯粹的理性主义者。
- 他曾制造“自飞木雕”（木鸟），腹内有机械，能飞起，是一项鲜为人知的机械发明。
