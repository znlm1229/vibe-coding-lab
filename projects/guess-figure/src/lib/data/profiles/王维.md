# 王维

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字摩诰 / 号摩诰居士 / 无谥号 / 无庙号 / 别号诗佛
- 生卒年 / 朝代区间 / 籍贯: 699年-761年 / 唐朝 (盛唐至中唐) / 太原祁县 (后移居蒲州)
- 主要职业 / 身份: 官员 / 诗人 / 画家 / 音乐家

## 主要事迹
- 717年进士及第，出任太乐丞，因排演禁舞“黄狮子”被贬济州司仓参军 [重要]
- 734年得宰相张九龄赏识，擢升为右拾遗，成为宫廷诗人 [重要]
- 740年左右购得宋之问蓝田别墅，营建辋川庄，隐居并创作《辋川集》 [重要]
- 756年安史之乱中，被叛军俘虏于长安，被迫接受伪职给事中 [重要]
- 757年作《凝碧池》诗暗表忠唐之心，乱平后因此诗及弟王缙求情得肃宗赦免，仅降阶为太子中允 [重要]
- 758年后累迁中书舍人、尚书右丞 [一般]
- 761年去世前将辋川别墅捐施为佛寺清凉寺 [一般]

## 性格 / 风格特征
- 笃信佛教，长期修禅，晚年长斋素食，不衣文彩，屏绝尘累 (源自本传“弟兄俱奉佛...晚年长斋”)
- 性情孝友，事母以孝闻，与弟王缙友悌，母丧时柴毁骨立 (源自本传“事母崔氏以孝闻...居母丧，柴毁骨立”)
- 艺术上观察敏锐，善于模仿继承，诗风柔和宁静闲逸，语言言简意赅 (源自维基材料“观察敏锐，善于模仿继承...诗风柔和宁静闲逸”)

## 典故 / 标志事件
- 扮乐师奏《郁轮袍》得岐王扶持及公主赏识
- 排演“黄狮子舞”被贬济州
- 购蓝田宋之问别墅营建辋川庄
- 安史之乱中被俘作《凝碧池》诗
- 辨《霓裳第三叠第一拍》乐图

## 关键作品
- 诗集《辋川集》二十首
- 诗《九月九日忆山东兄弟》
- 诗《送元二使安西》
- 诗《鸟鸣涧》
- 画作《辋川图》(传)

## 历史评价
- 正面: 被誉为“天下文宗”，苏轼评“诗中有画，画中有诗”，尊崇为水墨画始祖及山水画宗师；陷贼后因《凝碧池》诗及弟求情得赦，被视为忠贞有节。
- 负面: 安史之乱中被迫接受伪职，背负“贰臣”污名；性格有“傲诞”一面，如严武事件。
- 后世神话/演义: 被尊为“诗佛”，其艺术成就与佛教修为结合，形象趋于超脱。

## 反差 / 鲜为人知点
- 出身世家望族太原王氏，母族为博陵崔氏，家族地位是盛唐诗人中最高的，并非通常印象中的“小官僚出身”。
- 曾任皇家音乐机构“太乐丞”，因音乐才华得诸王赏识，其音乐天赋与辨乐能力是其多才多艺的重要一面，常被其诗画成就掩盖。
- 安史之乱被俘后，曾“服药取痢，伪称喑病”试图避免出任伪职，显示其并非甘心附逆，有挣扎抵抗之举。
