# 辛弃疾

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字幼安 / 号稼轩居士 / 谥忠敏 / 无 / 无
- 生卒年 / 朝代区间 / 籍贯: 1140年5月28日—1207年10月3日 / 南宋 / 山东东路济南府历城县（今山东济南历城区）
- 主要职业 / 身份: 词人、将领、地方官员

## 主要事迹
- 1161年，于济南聚众二千，加入耿京抗金义军，任掌书记，并劝耿京归宋。[重要]
- 1162年，耿京被叛徒张安国杀害后，率五十骑突入五万金营，擒张安国归宋斩首。[重要]
- 南归后屡次上书献策，如1165年献《美芹十论》，1168年献《九议》，力主恢复，但未被采纳。[一般]
- 1175年，任江西提刑，讨平以赖文政为首的茶商武装叛乱。[一般]
- 1180年，任湖南安抚使时，创建地方精锐军队“湖南飞虎军”，雄镇一方。[重要]
- 1181年，任江西安抚使，处理严重饥荒，稳定粮价，受宋孝宗嘉奖。[一般]
- 1181年冬，被御史王蔺弹劾“用钱如泥沙，杀人如草芥”而罢官，开始长期隐居。[重要]
- 1203年，64岁时被再度起用为绍兴知府、浙东安抚使，后调任镇江知府，积极备战，但很快又被弹劾去职。[一般]
- 1207年，北伐失败后，朝廷欲再度起用，诏命至时已病重，不久逝世，临终大呼“杀贼”。[一般]

## 性格 / 风格特征
- 豪爽尚气节，识拔英俊，所交多海内知名士。（源自《宋史》）
- 行事果决勇猛，有“青兕”之力，如单骑擒叛贼张安国。
- 为政务实干练，善斡旋，事皆立办，如创建飞虎军、赈灾等。（源自《宋史》记载）
- 性情刚直，持论劲直，不为迎合，故在官场“不为众人所容”。

## 典故 / 标志事件
- 率五十骑突入五万敌营擒拿叛徒张安国。
- 创建“湖南飞虎军”，为江上诸军之冠。
- 被弹劾“用钱如泥沙，杀人如草芥”而罢官隐居。
- 与陈亮于鹅湖寺相会，极论世事，共商恢复，是为第二次“鹅湖之会”。
- 在朱熹去世后，不顾“庆元党禁”，亲自前往祭奠并写祭文。

## 关键作品
- 《美芹十论》（政论文）
- 《九议》（政论文）
- 《稼轩长短句》（词集，代表作如《永遇乐·京口北固亭怀古》、《破阵子·为陈同甫赋壮词以寄之》、《青玉案·元夕》等）

## 历史评价
- 正面: 被誉为“词中之龙”，与苏轼合称“苏辛”，词风豪放悲壮，代表南宋豪放词最高成就；其勇略与政绩亦受称道。
- 负面: 在南宋朝廷被视为“归正人”难以完全信任；政敌弹劾其行事“残暴”、“聚敛”；部分词作有议论化、堆砌典故之弊。
- 后世神话/演义: 其墓旁传说有“疾声大呼”鸣不平的灵异事件（见《宋史》谢枋得记载）；文学形象多聚焦其爱国豪杰与悲愤词人两面。

## 反差 / 鲜为人知点
- 早年曾两次参加金朝的科举考试（燕京），实为受祖父命考察军情。
- 镇压茶商起义（赖文政）后，上奏指出“贪吏迫使为盗”，对农民充满同情，与“杀人如草芥”的弹劾形象形成反差。
- 虽以“豪放”著称，但其词集中有大量描写田园山水、农村生活的清新词作。
