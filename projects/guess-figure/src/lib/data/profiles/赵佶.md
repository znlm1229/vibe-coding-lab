# 宋徽宗

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 教主道君皇帝 / 显孝皇帝 / 徽宗 / 昏德公
- 生卒年 / 朝代区间 / 籍贯: 1082年11月2日—1135年6月4日 / 北宋 (1100-1126在位) / 开封 (北宋都城)
- 主要职业 / 身份: 皇帝、艺术家

## 主要事迹
- 1100年，因其兄宋哲宗无子，在向太后支持下即位为帝。[一般]
- 亲政后，任用蔡京、童贯等，大兴“花石纲”，修建艮岳，导致民不聊生。[重要]
- 崇信道教，自称“教主道君皇帝”。[一般]
- 1120年，与金国缔结“海上之盟”，联合灭辽。[重要]
- 1125年，金兵南下入侵，次年禅位给儿子钦宗，自称太上皇。[重要]
- 1127年，靖康之变中被金军俘虏，与钦宗一同被押北上。[重要]
- 被俘后，被封为“昏德公”，流放至五国城。[一般]
- 1135年，病死于五国城。[次要]
- 1142年，棺柩在宋金议和后归葬南宋永祐陵。[次要]

## 性格 / 风格特征
- 轻佻浮夸，章惇曾评价其“轻佻不可以君天下”。
- 酷爱艺术，追求极致审美，将画家的地位提到历史最高。
- 好大喜功，不顾国力与民生，追求奢侈与祥瑞。
- 缺乏政治担当，危难时急于禅位逃避责任。

## 典故 / 标志事件
- 花石纲：为修建园林宫殿，在江南大规模搜刮奇花异石运往汴京。
- 青楼天子：不顾帝王身份，多次通过地道前往青楼与名妓李师师相会。
- 靖康之变：1127年被金军俘虏北上，北宋灭亡的标志性事件。
- 瘦金体：自创一种独特的书法字体，称为“瘦金书”。
- 教主道君皇帝：崇信道教，给自己加封此尊号。

## 关键作品
- 瘦金体书法（如《诗帖》）。
- 设立翰林书画院，以画取士，推动绘画艺术。
- 组织编纂《宣和书谱》、《宣和画谱》、《宣和博古录》。
- 绘画作品（如《柳鸦芦雁图》、《池塘秋晚图》等，部分为御题画）。
- 使用著名的“天下一人”花押。

## 关系网
- 老师 / 同辈 / 弟子 / 政敌 / 家人 (各列 1-3 人，标关系类型)
- 家人：宋神宗 (父亲)、宋哲宗 (兄长)、宋钦宗赵桓 (儿子)、向太后 (支持者)。
- 政敌：金太宗完颜吴乞买 (敌方君主)、完颜宗翰 (敌方将领)。
- 宠臣/佞臣：蔡京 (宰相)、童贯 (宦官将军)、高俅 (宠臣)。
- 艺术关联：周邦彦 (词人，曾与徽宗争风吃醋于李师师)。

## 历史评价
- 正面：艺术成就极高，在书法、绘画、艺术鉴赏与推动方面贡献巨大，被誉为“诸事皆能”。
- 负面：作为皇帝极度失败，政治腐败、穷奢极欲、用人不当、决策失误，导致北宋灭亡，被俘受辱。
- 后世神话/演义：民间故事多渲染其与李师师的风流韵事，以及其艺术家的浪漫形象，淡化其亡国之君的罪责。

## 反差 / 鲜为人知点
- 被俘期间，金国将领完颜宗望曾要求不得像虐待辽天祚帝那样对待徽钦二帝，并给予田地让其“种莳自给”。
- 被软禁期间，徽宗仍雅好写诗，读唐代李泌传时感触颇深。
- 其棺柩归宋后，暂葬的陵墓原名“永固陵”，后改名为“永祐陵”。
