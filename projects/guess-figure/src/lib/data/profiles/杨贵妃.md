# 杨贵妃

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 号太真 / 小字玉环 (一说玉奴)
- 生卒年 / 朝代区间 / 籍贯: 719年6月22日—756年7月15日 / 唐朝 / 蒲州永乐 (今山西永济)
- 主要职业 / 身份: 唐玄宗宠妃 / 寿王妃 (原)

## 主要事迹
- 开元二十三年 (736年) 受封为寿王李瑁王妃 [一般]
- 开元二十八年 (740年) 为窦太后祈福出家为女道士，道号“太真” [一般]
- 天宝四年 (745年) 被唐玄宗册封为贵妃，礼遇等同皇后 [重要]
- 其家族因宠幸显贵，堂兄杨国忠任右相，姊妹封国夫人 [重要]
- 天宝五年、九年两次忤旨被遣出宫，旋即召回，宠幸更甚 [次要]
- 天宝十四年 (755年) 安史之乱爆发 [重要]
- 天宝十五年 (756年) 随玄宗奔蜀，途经马嵬驿发生兵变 [重要]
- 756年7月15日于马嵬驿被赐死 (缢死于佛堂梨树下) [重要]

## 性格 / 风格特征
- 姿色冠代，备受玄宗专宠 (“三千宠爱在一身”) [源自史料]
- 擅长舞蹈，尤精霓裳羽衣曲与胡旋舞 [源自史料]
- 未约束家族行为，杨氏一门飞扬跋扈 [源自史料]
- 两次被遣出宫，显示其性格亦有忤逆玄宗之时 [源自史料]

## 典故 / 标志事件
- 贵妃喜食荔枝，玄宗置骑传送，杜牧诗“一骑红尘妃子笑”。
- 马嵬驿兵变，将士诛杀杨国忠，贵妃被缢死，“宛转蛾眉马前死”。
- 两次被遣出宫又召回，显示玄宗对其宠爱至极。
- 以“太真”道士身份为过渡，由寿王妃变为玄宗贵妃。
- 安史之乱常被归咎于玄宗宠幸贵妃及杨家乱政。

## 关键作品
- (舞蹈代表作) 霓裳羽衣曲
- (舞蹈) 胡旋舞
- (间接相关) 引发白居易创作《长恨歌》
- (间接相关) 引发李白创作《清平调》
- (间接相关) 后世大量戏曲小说题材 (如《长生殿》)

## 关系网
- 家人: 唐玄宗 (夫君) / 寿王李瑁 (前夫) / 杨国忠 (堂兄，权臣)
- 政敌 / 对立者: 陈玄礼 (禁军将领，马嵬驿兵变主导者)
- 关键人物: 高力士 (宦官，劝玄宗赐死贵妃)

## 历史评价
- 正面: 中国古代四大美人之一，以美貌和艺术才华 (舞蹈) 著称。
- 负面: 被视为导致玄宗怠政、杨家乱政、引发安史之乱的“祸国红颜”。
- 后世神话/演义: 衍生出未死于马嵬驿、流落民间甚至东渡日本的传说。

## 反差 / 鲜为人知点
- 正史未载其名，只记其号“太真”，“杨玉环”之名源于后世小说笔记。
- 其体态并非后世“环肥”想象的肥胖，史料描述多为“纤秾中度”、“骨肉匀”，作为舞者应为匀称。
- 最初葬于驿西道侧，玄宗密令改葬时发现“肌肤已坏，而香囊仍在”。
