# 苏轼

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字子瞻，一字和仲，号东坡居士、铁冠道人，谥号文忠
- 生卒年 / 朝代区间 / 籍贯: 1037年1月8日—1101年8月24日 / 北宋 / 眉州眉山（今四川省眉山市）
- 主要职业 / 身份: 文学家、政治家、艺术家

## 主要事迹
- 嘉祐二年（1057年）进士，以《刑赏忠厚论》得欧阳修赏识，殿试中乙科。[重要]
- 熙宁年间反对王安石变法，尤其反对科举改革与青苗法等，屡遭新党排挤，求外任杭州通判。[重要]
- 元丰二年（1079年）因“乌台诗案”被捕入狱，几死，后贬为黄州团练副使，在黄州躬耕东坡，自号东坡居士。[重要]
- 元丰八年（1085年）哲宗即位后，被召回朝，历任翰林学士、知制诰等职，但因反对司马光等旧党尽废新法，再遭旧党排斥。[重要]
- 元祐四年（1089年）出知杭州，疏浚西湖，修筑苏堤。[重要]
- 绍圣元年（1094年）新党再度执政，被贬至惠州，后于绍圣四年（1097年）再贬至儋州（海南岛）。[重要]
- 元符三年（1100年）遇赦北归，次年病卒于常州。[一般]

## 性格 / 风格特征
- 生性放达，好交友，不拘小节，与渔樵杂处。[源自史料记载]
- 遇事直言，“如食内有蝇，吐之乃已”，在新旧党争中因直言两边不讨好。[源自史料记载]
- 好美食、品茗，雅好游山林，善于在困顿中寻找生活情趣。[源自史料记载]

## 典故 / 标志事件
- 乌台诗案：因诗文被指谤讪朝政而入狱，险死，后贬黄州。
- 筑室东坡：贬谪黄州时躬耕营地以东土地，自号东坡居士。
- 修筑苏堤：任杭州知州时疏浚西湖，取淤泥筑堤，即苏堤。
- 一肚皮不合时宜：侍妾王朝云戏称其在新旧党争中两边不讨好。
- 海南贬谪：晚年被贬至儋州（海南岛），是北宋被贬最远的文臣之一。

## 关键作品
- 《东坡先生大全集》（文集）
- 《东坡乐府》（词集）
- 《赤壁赋》（前后两篇）
- 《刑赏忠厚论》（早年成名文章）
- 书法代表作（如《寒食帖》等，名列宋四家之首）

## 历史评价
- 正面: 文学艺术通才，散文、诗、词、书法、绘画皆有极高成就；为官清正，为民兴利除弊（如苏堤）。
- 负面: 在新旧党争中立场摇摆不定，被双方排斥；性格过于直率，有时被视为“不合时宜”。
- 后世神话/演义: 民间故事中常与佛印禅师斗智，成为智慧与豁达的象征；美食传说（东坡肉等）广为流传。

## 反差 / 鲜为人知点
- 苏轼曾因被诬告“贩运苏木（染料）”而影响仕途升迁，此事查无实据但曾被神宗怀疑。[源自维基材料]
- 苏轼反对王安石变法，但在金陵会见王安石时，却直言劝谏其应为“大兵大狱”之事进言，展现了超越党争的个人情谊与责任感。[源自《宋史》本传]
- 苏轼在贬谪惠州期间，曾典卖自己的腰带以资助当地修桥。[源自维基材料]
