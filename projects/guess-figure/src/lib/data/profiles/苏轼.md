# 苏轼

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字子瞻，号东坡居士，谥号文忠
- 生卒年 / 朝代区间 / 籍贯: 1037年1月8日 - 1101年8月24日 / 北宋 / 眉州眉山（今四川省眉山市）
- 主要职业 / 身份: 文学家、政治家、艺术家

## 主要事迹
- 嘉祐二年进士，因《刑赏忠厚论》得欧阳修赏识，但避嫌列为第二 [重要]
- 熙宁年间反对王安石变法，上书论其不便，遭新党打压 [重要]
- 元丰二年因“乌台诗案”被捕入狱，贬为黄州团练副使 [重要]
- 元丰年间在黄州垦荒东坡，筑雪堂，自号东坡居士 [一般]
- 元祐年间短暂回朝任翰林学士等职，卷入新旧党争，与程颐等人不和 [重要]
- 元祐四年出知杭州，疏浚西湖，修筑苏堤 [一般]
- 绍圣年间遭章惇等新党执政者迫害，贬谪至惠州、儋州 [重要]
- 徽宗即位后赦免北归，途中病逝于常州 [一般]

## 性格 / 风格特征
- 生性放达，好交友，不拘小节
- 直言敢谏，不惧权贵，新旧党争中两边不讨好
- 处逆境时能“放浪山水之间，与渔樵杂处”，豁达自适

## 典故 / 标志事件
- 乌台诗案

- 修筑苏堤

## 关系网
- 老师: 欧阳修 (赏识提拔)
- 同辈: 苏辙 (弟)，黄庭坚 (并称“苏黄”)
- 政敌: 王安石，程颐，章惇
- 家人: 苏洵 (父)，王弗 (妻)

## 历史评价
- 正面: 文学艺术通才，散文、诗、词、赋、书法、绘画皆有极高成就；为官时为民兴利除弊，政绩颇善。
- 负面: 仕途因直言和党争屡遭贬谪，被认为“一肚皮不合时宜”；部分政敌指责其“讪谤朝政”。
- 后世神话/演义: 形象被进一步塑造为豁达、智慧的文人典范，诸多美食传说（如东坡肉）附会其身。

## 反差 / 鲜为人知点
- 早年科举时，其文章被欧阳修误认为是门生曾巩所作，为避嫌而未列为第一。
- 与凤翔府长官陈希亮个人关系极差，却与其子陈慥相交甚欢。
- 被贬海南儋州时，初到无住所，暂住官舍，后当地人为他建“桄榔庵”。
