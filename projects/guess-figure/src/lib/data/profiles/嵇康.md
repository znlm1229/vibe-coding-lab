# 嵇康

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字叔夜 / 号无 / 谥号无 / 庙号无 / 别号嵇中散
- 生卒年 / 朝代区间 / 籍贯: 223年—263年 / 三国曹魏 / 谯郡铚县（今安徽濉溪）
- 主要职业 / 身份: 文学家、思想家、音乐家

## 主要事迹
- 早年自学成才，涉猎广泛，不修仪表，不求仕进。[一般]
- 娶曹魏宗室长乐亭主，官拜郎中、中散大夫等闲职。[一般]
- 隐居山阳，与阮籍等人交游，成为“竹林七贤”领袖之一。[重要]
- 以锻铁为生，躲避司马昭征辟，冷遇钟会拜访。[重要]
- 因山涛举荐，写下《与山巨源绝交书》，表明不愿为官的态度。[重要]
- 卷入吕安诉讼案，出面作证申冤，因而下狱。[重要]
- 在狱中写下《幽愤诗》与《家诫》。[一般]
- 临刑前于刑场弹奏《广陵散》，慨叹“《广陵散》于今绝矣”，随后被处斩。[重要]

## 性格 / 风格特征
- 桀骜不驯，崇尚自由，不愿受世俗规范约束。
- 性格慵懒散漫，厌恶官场礼仪与事务。
- 为人刚直，重情谊，为朋友冤屈不惜挺身涉险。

## 典故 / 标志事件
- 竹林之游：与阮籍等七人隐居山阳，肆意酣畅，号为“竹林七贤”。
- 锻铁避世：于门庭前锻铁，以此为由拒绝司马昭征辟与钟会拜访。
- 绝交山涛：因好友山涛举荐自己代职，写下《与山巨源绝交书》宣告绝交。
- 广陵绝响：临刑前索琴弹奏《广陵散》，曲罢叹此曲从此断绝。
- 三千请命：被捕后，三千太学生联名请愿，请求让其执教太学。

## 历史评价
- 正面：被视为魏晋玄学重要代表，思想与人格自由、清廉正直、不畏强权的典范。
- 负面：因其桀骜不驯、激烈抨击名教，被当权者视为威胁，最终因言论获罪被杀。
- 后世神话/演义：死后被神仙化，生平事迹掺入神怪传奇内容；其临刑弹琴的形象成为艺术创作经典题材。

## 反差 / 鲜为人知点
- 其父早亡，母亲与兄长抚养时偏于娇惯，缺乏严格管束，这被认为塑造了他自由不羁的性格。
- 他并非纯粹的谋生而锻铁，亦有说法称其锻铁行为本身具有象征意义，并非仅为生计。
- 他与吕巽结识早于吕安，最初曾劝吕安勿揭发家丑以保全门第清誉，此事成为其最终卷入诉讼的伏笔。
