# 明世宗 (嘉靖帝)

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 肃帝 / 世宗 / 嘉靖帝
- 生卒年 / 朝代区间 / 籍贯: 1507年9月16日—1567年1月23日 / 明朝 (1521年-1567年在位) / 安陆州 (今湖北钟祥)
- 主要职业 / 身份: 明朝皇帝

## 主要事迹
- 1521年以藩王身份即位，引发持续多年的“大礼议”之争，追尊生父为皇帝。[重要]
- 在位初期，清除宦官和权臣势力，总揽朝纲，史称“嘉靖中兴”。[一般]
- 嘉靖二十一年（1542年）遭遇“壬寅宫变”后，移居西苑，长期不上朝，潜心修道。[重要]
- 在位后期，东南沿海倭患严重，任用胡宗宪、戚继光等人进行抗击。[一般]
- 嘉靖二十九年（1550年），蒙古俺答汗兵临北京城下，史称“庚戌之变”。[重要]
- 晚年宠信严嵩，导致朝政腐败，后由徐阶等人扳倒严嵩父子。[一般]
- 嘉靖四十五年（1566年），海瑞上《治安疏》直言批评其昏聩，世宗震怒但仍未杀海瑞。[次要]

## 性格 / 风格特征
- 个性刚愎自用，在“大礼议”中坚持己见，与群臣激烈对抗。
- 中年后迷信道教方术，追求长生，长期在西苑斋醮炼丹。
- 统治手腕强硬，虽多年不临朝，但仍通过奏疏和宦官牢牢掌控朝局。

## 典故 / 标志事件
- 大礼议：为生父兴献王争皇帝名分，与廷臣激烈对抗。
- 壬寅宫变：宫女杨金英等人意图勒死嘉靖帝未遂。
- 庚戌之变：蒙古俺答汗率军兵临北京城下，大肆劫掠。
- 青词宰相：严嵩因擅长撰写道教祭天青词而得宠专权。
- 海瑞上疏：户部主事海瑞上《治安疏》直言批评皇帝怠政修玄。

## 关键作品
- 年号“嘉靖”（使用长达四十五年，为明朝第二长）。
- 推动并深度参与道教斋醮仪式与青词创作。
- 在位期间平定东南沿海倭患（虽非其亲自指挥，为重要政绩）。

## 关系网
- 老师 / 同辈 / 弟子 / 政敌 / 家人 (各列 1-3 人，标关系类型)
- 政敌：杨廷和 (首辅，大礼议中主要反对者)
- 宠臣/政敌：严嵩 (长期把持内阁的首辅)
- 家人：朱厚熜生父 兴献王朱祐杬 (追尊为皇帝)

## 历史评价
- 正面：即位初期有所作为，出现“嘉靖中兴”；后期平定倭患。
- 负面：中年后荒怠朝政、沉迷修道、宠信奸臣，导致国势渐衰。
- 后世神话/演义：民间故事和文学作品中常以其修道、宫变、权臣故事为题材。

## 反差 / 鲜为人知点
- 虽以修道怠政闻名，但其在位前期曾大力推行改革，如清理勋戚庄田、裁革冗官和镇守宦官。
- 在“壬寅宫变”中险些被宫女勒死，此事件极大影响其心理，成为其移居西苑、疏远朝臣的转折点。
- 面对海瑞言辞激烈的《治安疏》，虽极度愤怒并将其下狱，却始终未下令处死海瑞，并私下表示“此人可比比干”。
