# 嘉靖

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 肃帝 / 世宗 / 嘉靖帝
- 生卒年 / 朝代区间 / 籍贯: 1507年9月16日 - 1567年1月23日 / 明朝 (1521-1567在位) / 湖北安陆 (兴王府)
- 主要职业 / 身份: 明朝皇帝

## 主要事迹
- 1521年即位，引发“大礼议”之争，确立自身权威 [重要]
- 嘉靖初年推行改革，清除宦官与权臣势力，史称“嘉靖中兴” [重要]
- 嘉靖二年（1523年）因宁波争贡事件厉行海禁 [一般]
- 嘉靖二十一年（1542年）经历“壬寅宫变”，此后转向深居修玄，疏于朝政 [重要]
- 嘉靖二十九年（1550年）应对“庚戌之变”，俺答汗兵临北京 [重要]
- 在位后期任用严嵩等，朝政陷入党争 [一般]
- 平定东南沿海倭患（嘉靖后期） [重要]
- 嘉靖四十五年（1566年）接受海瑞上《治安疏》直言批评 [一般]

## 性格 / 风格特征
- 即位初期展现改革魄力，意图强化皇权与整顿朝纲（见于早期改革）
- 中年后性格转向固执迷信，沉迷道教斋醮与炼丹求长生（见于修玄记载）
- 统治后期偏听偏信，宠信权臣如严嵩，导致朝政腐败（见于史料对其后期评价）

## 典故 / 标志事件
- 大礼议之争确立皇权
- 壬寅宫变后转向修玄
- 庚戌之变蒙古兵临京师
- 任用严嵩导致朝政腐败
- 海瑞上《直言天下第一事疏》

## 关键作品
- 推行“嘉靖中兴”初期改革（如清理勋戚庄田、裁革冗官）
- 在位期间青花瓷及多彩珐琅瓷工艺发展（艺术成就）
- 应对倭患与边防事务（军事政策）
- 其年号“嘉靖”出自《尚书·无逸》“嘉靖殷邦”

## 关系网
- 老师 / 同辈 / 弟子 / 政敌 / 家人 (各列 1-3 人, 标关系类型)
- 政敌：杨廷和（早期首辅，大礼议中被罢） / 海瑞（直言进谏的官员）
- 宠信臣僚：严嵩（后期权臣） / 徐阶（最终扳倒严嵩者）
- 家人：明穆宗朱载垕（儿子，后继者） / 明武宗朱厚照（堂兄，前任皇帝）

## 历史评价
- 正面: 早期被视为有能力开启“嘉靖中兴”，整顿吏治，强化皇权。
- 负面: 后期评价多为消极，批评其沉迷道教、疏于朝政、宠信奸臣、导致国力衰退。
- 后世神话/演义: 在民间故事与文学中常被描绘为迷信炼丹的昏君形象，尤以“壬寅宫变”和海瑞上疏故事流传。

## 反差 / 鲜为人知点
- 其年号“嘉靖”原非首选，内阁最初备选为“明良”与“绍治”。
- 在位期间并非完全闭关，葡萄牙人于嘉靖三十二年（1553年）取得澳门居住权。
- 嘉靖大地震（1556年）发生在其在位期间，是历史上死亡人数最多的地震之一，但其应对记载较少。
