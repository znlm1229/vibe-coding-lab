# 韩愈

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 退之 / 昌黎 / 文 / 无 / 韩吏部
- 生卒年 / 朝代区间 / 籍贯: 768年-824年 / 唐朝 / 河南府河阳县（今河南省孟州市）
- 主要职业 / 身份: 官员、文学家、古文运动领袖

## 主要事迹
- 贞元八年进士及第，后三试吏部不中，愤慨上书宰相求荐举 [一般]
- 贞元十九年任监察御史，因关中旱灾上《御史台上论天旱人饥状》，弹劾京兆尹李实，贬阳山令 [重要]
- 元和六年任国子博士，作《进学解》，受裴度赏识，擢礼部郎中 [一般]
- 元和十二年随裴度征淮西，因功擢刑部侍郎，撰《平淮西碑》 [重要]
- 元和十四年上《谏迎佛骨表》，反对宪宗迎佛骨，贬潮州刺史 [重要]
- 在潮州任内，作《祭鳄鱼文》，组织百姓驱杀鳄鱼 [一般]
- 袁州任上，设法赎还被债主奴役的百姓，并禁止借贷逾期沦为奴婢的陋俗 [次要]
- 长庆四年因病告假，十二月卒于长安靖安里宅中 [一般]

## 性格 / 风格特征
- 操行坚正，发言真率，无所畏避（《旧唐书》载）
- 性弘通，与人交，荣悴不易，能诱厉后进（《旧唐书》载）
- 自负辞艺，以为合知制诰，久未如愿则郁郁不乐（《旧唐书》李翱传关联记载）

## 典故 / 标志事件
- 谏迎佛骨（“忠犯人主之怒”）
- 驱鳄鱼（《祭鳄鱼文》）
- 蓝关示侄孙湘（“雪拥蓝关马不前”）
- 不平淮西碑（李愬妻诉碑文不实，碑被推倒）
- 赎袁州奴婢（改革地方陋俗）

## 关键作品
- 《谏迎佛骨表》（《论佛骨表》）
- 《师说》
- 《进学解》
- 《祭十二郎文》
- 《平淮西碑》（后被磨）

## 关系网

- 弟子/后进: 李翱

- 家人: 韩会（长兄，抚养人）、韩老成（侄，十二郎）、郑氏（嫂）

## 历史评价
- 正面: “文起八代之衰”，古文运动领袖，儒家道统重要继承者
- 负面: 部分“谀墓”之作受讥斥，收受丰厚润笔；性格有时“狷直”、“恃才肆意”
- 后世神话/演义: 被与八仙之一的韩湘子故事关联（“韩湘”）；在台湾有“诽韩案”争议

## 反差 / 鲜为人知点
- 虽激烈反佛，但与潮州大颠和尚等僧人交游酬唱，并非完全排斥佛教。
- 反对道教丹药迷信，称“杀人不可计”，但自己诗中却提及服用“金丹”、“硫黄”。
- 晚年于长安城南兴建豪宅，并有绛桃、柳枝等妓，能歌舞，生活并非一贯清苦。
