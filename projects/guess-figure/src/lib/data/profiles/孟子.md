# 孟子

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 无 / 邹国公 / 无 / 亚圣
- 生卒年 / 朝代区间 / 籍贯: 约前372年-前289年 / 战国时期 / 邹国（今山东邹城）
- 主要职业 / 身份: 思想家、哲学家、儒家代表人物

## 主要事迹
- 早年受业于子思门人，学习儒家思想 [一般]
- 效仿孔子，带领门徒游说各国，宣扬仁政王道，不为各国接受 [重要]
- 退隐与弟子一起著述，著有《孟子》七篇 [重要]
- 其思想主张“民贵君轻”，提出仁政与王道学说 [重要]
- 后世被尊为“亚圣”，其著作《孟子》被列入“四书”，成为科举必考内容 [重要]

## 性格 / 风格特征
- 长于辩论，文章说理畅达，气势充沛（源自史料对其文风的评价）
- 具有强烈的民本思想，敢于提出“民为贵，社稷次之，君为轻”
- 自信而执着，虽游说不成仍坚持著述传道

## 典故 / 标志事件
- 孟母三迁（孟母为教育孟子而三次搬迁住处）
- 断机教子（孟母剪断织布机上的线来教育孟子学习不可中断）
- 五十步笑百步（孟子用以讽刺梁惠王治国不力）
- 民贵君轻（孟子提出的著名政治主张）
- 浩然正气（孟子提出的道德修养概念）

## 关键作品
- 《孟子》（七篇，由其弟子万章等参与编撰）
- 性善论（提出人性本善的哲学理论）
- 仁政学说（主张君主应以仁德治国）
- 王道思想（主张通过保民使国家富强安定）

## 关系网
- 老师: 子思门人（师承关系）
- 同辈: 荀子（同为儒家思想家，观点对立）
- 弟子: 万章（主要弟子，参与著述）
- 政敌: 无明确记载的政敌
- 家人: 孟母仉氏（母亲，教育者）

## 历史评价
- 正面: 被尊为“亚圣”，是继承孔子道统的一代宗师；其民本思想与仁政学说对后世影响深远。
- 负面: 宋代以前地位不高，有打油诗讽刺其故事不合逻辑；朱元璋不满其民本思想，曾删节《孟子》。
- 后世神话/演义: 在民间故事中被塑造为孟母严格教育的成功典范；在一些小说（如《大秦帝国》）中与其他人物有虚构论战。

## 反差 / 鲜为人知点
- 其字號在漢代以前的古書未有記載，曹魏、晉代之後才傳出子車、子居等不同字號，真实性存疑。
- 据《孟子世家谱》，其祖先是从鲁国迁居到邹国的孟孙氏后代，并非邹国土著。
- 东条英机在监狱中曾引用《孟子》的话，显示其思想影响范围之广，甚至触及二战战犯。
