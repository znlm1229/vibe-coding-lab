# 屈原

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 字原 (一说字灵均)
- 生卒年 / 朝代区间 / 籍贯: 约前342年-前278年 / 战国时期 / 楚国丹阳 (一说秭归，一说纪南)
- 主要职业 / 身份: 楚国左徒、三闾大夫、诗人

## 主要事迹
1. 早年任楚怀王左徒，参与国政，主张联齐抗秦 [重要]
2. 遭贵族反对，楚怀王听信谗言，逐渐疏远屈原 [重要]
3. 楚怀王十六年 (前313年)，张仪骗楚绝齐，屈原反对无效 [一般]
4. 楚怀王二十四年 (前305年)，楚背齐合秦，屈原竭力反对，遭第一次流放至汉北地区 [重要]
5. 楚怀王被诱入武关扣押，屈原与昭雎等力劝怀王勿行 [一般]
6. 顷襄王六年 (前293年)，谋划与秦讲和，屈原斥责顷襄王与子兰，遭第二次流放至江南地区 [重要]
7. 流放期间创作大量诗篇，抒发忧国忧民之情 [重要]
8. 顷襄王二十一年 (前278年)，秦将白起攻破郢都，屈原悲愤，于农历五月五日投汨罗江自尽 [重要]

## 性格 / 风格特征
1. 博闻强志，娴于辞令 (司马迁评价)
2. 改革精神强烈，招致贵族反对
3. 眷顾楚国，系心怀王，不忘欲反
4. 在流放中接触底层民众与楚国民间文化

## 典故 / 标志事件
1. 张仪骗楚绝齐，屈原反对无效
2. 力劝楚怀王勿入武关
3. 斥责顷襄王与子兰，遭第二次流放
4. 秦破郢都，屈原投汨罗江自尽
5. 端午节吃粽子纪念屈原

## 关系网
- 同辈 / 政敌: 上官大夫靳尚 (政敌)、子兰 (政敌)
- 家人: 父伯庸、姊女媭

## 历史评价
- 正面: 忠君爱国，文学成就高，中国浪漫主义文学奠基人
- 负面: 性格刚直，改革激进，遭贵族排挤
- 后世神话/演义: 端午节起源与之相关，被塑造为忠臣与诗人典范

## 反差 / 鲜为人知点
1. 屈原的“字灵均”可能与巫祝身份有关，并非单纯文学化名
2. 其家族虽为楚国公族屈氏，但到他一代时已衰落，自称“忽忘身之贱贫”
3. 关于其出生地存在秭归说与纪南说 (郢都) 的争议，并非定论
