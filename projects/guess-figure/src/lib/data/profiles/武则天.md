# 武则天

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 无 / 天后 / 则天皇后 / 无 / 武媚娘
- 生卒年 / 朝代区间 / 籍贯: 624年—705年12月16日 / 唐朝（武周时期） / 并州文水县（今山西省文水县）
- 主要职业 / 身份: 皇后、皇太后、皇帝（中国历史上唯一的女皇帝）

## 主要事迹
- 十四岁入宫为唐太宗才人，后出家为尼。[一般]
- 高宗时复入宫为昭仪，通过宫斗取代王皇后与萧淑妃，于655年被立为皇后。[重要]
- 高宗患病后开始临朝当政，与高宗并称“二圣”，内辅国政数十年。[重要]
- 高宗崩后，以皇太后身份临朝称制，先后废黜中宗、睿宗。[重要]
- 690年自立为帝，改国号为周，建立武周政权。[重要]
- 掌权期间扩大科举，发展农业，户口大增，对边疆抚慰与讨伐并用。[一般]
- 大兴告密，严刑峻法，清洗唐宗室及旧臣，晚年豪奢专断。[一般]
- 705年，宰相张柬之等发动神龙革命，迫其退位，同年于洛阳上阳宫去世。[重要]

## 性格 / 风格特征
- 素多智计，兼涉文史，政治手腕高超。
- 行事果断，手段强硬，为巩固权力不惜清洗宗室与旧臣。
- 晚年豪奢专断，大兴土木（如造明堂、天枢）。

## 典故 / 标志事件
- 二圣临朝（与高宗并称“二圣”共同执政）。
- 废王立武（通过宫斗取代王皇后成为皇后）。
- 造明堂（毁乾元殿建造明堂，作为政治象征）。
- 洛水获图（武承嗣伪造“圣母临人，永昌帝业”瑞石，武则天借此加尊号）。
- 神龙革命（宰相张柬之等发动政变迫其退位，恢复唐朝）。

## 关键作品
- 颁布《垂拱格》（亲自撰写的法律条文）。
- 推行扩大科举的政策（开创殿试等，增加取士人数）。
- 资助文艺，推崇佛教（如令诸州置大云寺）。
- 发展农业，户口大增（治理有方，经济人口增长）。
- 造天枢（于端门外造天枢纪功，显示其权威）。

## 历史评价
- 正面: 治国积极，扩大科举，发展农业，人口增长，边疆政策有效，掌权期间国家总体稳定发展。
- 负面: 大兴告密，严刑峻法，清洗宗室及旧臣，晚年豪奢专断，任用酷吏，制造恐怖政治氛围。
- 后世神话/演义: 常被描绘为权谋狠辣、狐媚惑主的形象，在民间故事和文学作品中多有演绎（如“武媚娘”故事）。

## 反差 / 鲜为人知点
- 其确切原名在正史中并无记载，近代学界有“武约”、“武照”、“武元华”、“武华命”等多种推测，但均无定论。
- 出生地存在极大争议，有山西文水、江苏扬州、河南洛阳、四川广元、陕西西安等多种说法，学界未有共识。
- 曾为避讳（可能为其祖父武华或自身名讳），在称帝前后将含“华”字的官员、行政区改名（如华州改为太州）。
