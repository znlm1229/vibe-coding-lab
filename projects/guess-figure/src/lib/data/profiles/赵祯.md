# 赵祯

## 基本信息
- 字 / 号 / 谥号 / 庙号 / 别号: 受益 (原名) / 无 / 睿哲明孝 (谥号简称) / 仁宗 / 无
- 生卒年 / 朝代区间 / 籍贯: 1010年5月30日—1063年4月30日 / 北宋 (1022-1063在位) / 汴梁 (今河南开封)
- 主要职业 / 身份: 皇帝

## 主要事迹
- 1018年，进封昇王，同年立为皇太子，赐名禎 [重要]
- 1022年，真宗崩，即位，时年13岁，由刘太后摄政 [重要]
- 1033年，刘太后去世，仁宗亲政 [重要]
- 1042-1044年，宋夏战争，最终达成“庆历和议” [重要]
- 1043-1044年，支持范仲淹等人推行“庆历新政”，后因反对势力庞大而中止 [重要]
- 1048年，处理坤寧宮事變 (疑似後宮爭鬥) [一般]
- 1052-1053年，任用狄青平定侬智高叛乱 [一般]
- 1063年，驾崩于汴梁，在位41年，为宋朝在位时间最长的皇帝 [重要]

## 性格 / 风格特征
- 天性恭俭仁恕，常怀恻隐之心 (史料赞其“恭俭仁恕，出于天性”)
- 宽仁少断，任事易变 (王夫之评其“无定志”、“宽仁少断”，官员更迭频繁)
- 生活简朴，自律克制 (如夜饥思烧羊，却戒勿宣索，恐膳夫戕贼物命)

## 典故 / 标志事件
- 狸猫换太子 (民间戏曲传说，主角影射仁宗)
- 庆历新政 (范仲淹等提出的改革，短暂推行)
- 庆历和议 (结束宋夏战争，换取和平)
- 重熙增币 (应对辽国威胁，增加岁币)
- 赤脚仙人 (幼年常赤足行走，宫中呼为“赤脚仙人”)

## 关键作品
- 维持“澶渊之盟”后的和平局面 (外交政绩)
- 奠定“仁宗盛治”基础 (内政时期)
- 发行交子 (天圣元年，刘太后下旨，但其在位期间)
- 潜藩升府制度雏形 (以其封地昇州升为江宁府为例)
- 无特定文学著作，但其治世为文化发展提供环境

## 关系网
- 老师 / 同辈: 刘太后 (养母/摄政者)、杨太妃 (养母，更亲近)
- 弟子: 无明确弟子
- 政敌: 西夏景宗李元昊 (军事对手)
- 家人: 郭皇后 (元配，后被废)、曹皇后 (继室)、张贵妃 (宠妃，追册温成皇后)、苗贵妃、周贵妃等

## 历史评价
- 正面: 被誉为仁爱勤俭，君臣上下有忠厚之政，为宋朝三百余年之基奠基。
- 负面: 被认为“无定志”、“宽仁少断”，政策因人事频繁更迭而无法贯彻，导致官吏无所适从。
- 后世神话/演义: 民间“狸猫换太子”故事主角，形象被艺术加工。

## 反差 / 鲜为人知点
- 疑似患有家族遗传性高血压和早老性失智症，晚年出现中风、失语、谵妄等症状。
- 其出生日，真宗曾赏赐群臣包子，成为宫廷轶事。
- 虽贵为皇帝，却常赤足行走宫中，被称为“赤脚仙人”，可能与某种修行观念或个人习惯有关。
