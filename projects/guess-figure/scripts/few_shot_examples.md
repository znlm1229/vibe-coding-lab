# Few-shot Pool — 003 线索质量好/坏对比示例

> 来自 003 Stage 1-3 中的真实观察 + prototype 验证。生产 pipeline 时,clues_from_profile 从本池随机选 2 对(1 好 1 坏)inject 到 prompt 作为 few-shot example。
>
> **维护方式**:Phase 6/7 跑出新坏例 → 加坏例条目 + 改进建议;Phase 8 用户实测发现好例 → 加好例。
>
> **初版** 由 AI 提案,Stage 7 T2 完成时用户审稿入库。

---

## 坏示例(5 个,实际穿底案例)

### 坏 #1 — 语义穿底(d1 用同义词暴露 alias)

- **figure**:乾隆(aliases: 弘历 / 清高宗 / **十全老人** / 古稀天子 / 长春居士)
- **难度**:d1
- **反例文本**:"他曾于暮年自诩拥有**十项武功**,并以此作为其统治期间最为自豪的政绩"
- **问题**:"十项武功" 是 alias「十全老人」的同义表达,字符级 grep 抓不到但语义直接命中
- **改进文本**:"他曾因晚年自比某些数量级的边疆功业而被史官评价,然其实际军事成果与自我评价之间存在明显落差"(改用「评价反差」而非具体数字)

### 坏 #2 — alias 子串穿底(d6/d7 出现 alias 字符)

- **figure**:关羽(aliases: **关云长** / 关公 / 关帝君 / 关圣帝君 / 伽蓝菩萨)
- **难度**:d7
- **反例文本**:"他是蜀汉五虎上将之首,**字云长**,河东解人"
- **问题**:"字云长" 三字包含 alias「关云长」的关键子串「云长」,d6/d7 prompt 没禁子串等于直接给答案
- **改进文本**:"他是蜀汉早期核心将领,死后被历代帝王不断追封,民间常称其为关二爷或某帝"(用「关二爷」+ 模糊「某帝」替代直接给字)

### 坏 #3 — 直接给 alias(d7 出现完整 alias)

- **figure**:刘备(aliases: 刘玄德 / **汉昭烈帝** / **昭烈皇帝** / 汉先主 / **烈祖**)
- **难度**:d7
- **反例文本**:"作为蜀汉的先主,他因三顾茅庐的典故而闻名,**谥号为昭烈,庙号烈祖**"
- **问题**:"昭烈" 直接出现在 aliases「汉昭烈帝/昭烈皇帝」,"烈祖" 直接是 alias
- **改进文本**:"作为蜀汉开国之主,他因三顾茅庐的典故而闻名,其谥号与庙号均蕴含「英烈」之意"(用「蕴含意」绕过)

### 坏 #4 — 标志事件穿底(d2 出现典故 banlist 词)

- **figure**:刘备
- **难度**:d2
- **反例文本**:"他以**织席贩履**为业起家,在乱世中凭借仁义之名广纳贤才,曾与两位**结义兄弟**共同征战四方"
- **问题**:"织席贩履" 是刘备最标志的早年经历,"结义兄弟" 直指桃园结义,d2 不该出现这种"一句话锁定身份"的事件
- **改进文本**:"他出身寒微,早年常游走于各路诸侯之间寻求机会,身边逐渐聚拢起几位结拜兄弟"(把"织席贩履"模糊为"出身寒微","结义兄弟"换"结拜兄弟" — 但仍要小心,极佳的话用「结义」「桃园」都该避开)

### 坏 #5 — 难度梯度反转(d5 反而比 d1 更易猜)

- **figure**:乾隆
- **难度**:d5 vs d1(对比)
- **反例 d1**:"他曾于暮年自诩拥有十项武功,并以此作为其统治期间最为自豪的政绩"
- **反例 d5**:"他不仅是清朝入关后的第四位统治者,也是历史上实际掌权时间最长的君主"
- **问题**:d5 直接给"清朝入关后的第四位统治者" — 历史课答案;d1 用"十项武功"虽穿底但需要识破,d5 反而更易猜。**难度梯度乱**
- **改进方向**:d1 必须用画像「反差/鲜为人知点」section(如"娶丑女""被同时人否定"),d5 才出"哪个朝代第几位皇帝"这种确定性强的标识

## 好示例(5 个,prototype 跑出的真实 d1 + 模板示意)

### 好 #1 — 用反差点产 d1(prototype 实证,claude-haiku 产出)

- **figure**:诸葛亮(aliases: 孔明 / 卧龙 / 忠武侯 / 诸葛武侯)
- **难度**:d1
- **正例文本**:"他曾因**择偶标准被乡里编成谚语嘲笑**,且在躬耕陇亩时**常自比古之名将,却在很长一段时间内被周围的士人所轻视与否定**"
- **来源**:profile「反差 / 鲜为人知点」section 的「娶丑女被乡里讥」+「自比管仲乐毅时人莫之许」
- **为何好**:① 不含 aliases 字符 + 不含「三顾茅庐/隆中对/出师表/木牛流马」典故;② 普通人脱离这些常识只能凭"被嘲笑+被否定"猜,但仍能猜出来(线索本质未抽空);③ 与"智绝"的固有印象形成反差,prompt"反差点"机制 work

### 好 #2 — 用反差点产 d1(prototype 实证,deepseek 产出)

- **figure**:诸葛亮
- **难度**:d1
- **正例文本**:"他曾因**择偶标准异于常人而沦为乡里笑谈**,且**史料中存在其主动投奔而非被动受邀的记载**,展现出与后世刻板印象截然不同的进取心态"
- **来源**:profile「反差/鲜为人知点」的「黄氏丑女」+「《魏略》主动求见 vs 三顾茅庐」
- **为何好**:① 引用《魏略》少数派记载(普通人不知);② "与后世刻板印象截然不同"明示反差;③ 不含任何穿底字眼

### 好 #3 — d2 用历史评价(模板示意)

- **figure**:诸葛亮
- **难度**:d2
- **正例文本**:"他常以春秋时期的名臣自比,一生追求宏大的政治抱负,**却被后世史官评价为治国理政之才远胜于带兵打仗的将略**"
- **来源**:profile「历史评价」section 的陈寿原评「治戎为长,奇谋为短」+「理民之干,优于将略」
- **为何好**:① 触历史评价(d2 允许);② 不出现「诸葛/孔明/卧龙」+ 不出现「赤壁/北伐/出师表」;③ 用"治国理政 vs 将略"的反差强化 d2 难度

### 好 #4 — d3 用关系网抽象描述(模板示意,基于王维 profile)

- **figure**:王维(aliases: 摩诘 / 王右丞 / 诗佛)
- **难度**:d3
- **正例文本**:"他与一位同朝官至宰相的诗坛挚友早年共结诗社,**曾师从一位以山水诗见长的隐居高僧研习禅理**,二人关系亦师亦友"
- **来源**:profile「关系网」section(假设:孟浩然 / 大荐福寺道光禅师)
- **为何好**:① 触关系网抽象(d3 允许);② 不出现「摩诘/王右丞/诗佛」+ 不出现具体作品;③ "山水诗见长" + "研习禅理" 暗示禅意+山水诗派,信息密度梯度合理

### 好 #5 — d6 可触作品/典故的求救线索(模板示意)

- **figure**:王维
- **难度**:d6
- **正例文本**:"作为唐代山水田园诗派的代表人物,他在终南山经营辋川别业,创作了大量诗画相融的作品,被后世尊为某宗派的开山祖师"
- **为何好**:① d6 求救范围允许触作品/典故(辋川别业 / 山水田园诗派 OK);② 不出现「王维」「摩诘」「右丞」「诗佛」aliases 字符;③ "某宗派" 用代称避免直说「南宗画」「诗佛」

## 使用方式(production pipeline)

`scripts/generate_figures.py` 的 `clues_from_profile()` 函数:

1. 读 `scripts/few_shot_examples.md`,parse 出 5 好 + 5 坏列表
2. 每次生成时,随机选 1 好 + 1 坏作为 few-shot 注入 prompt(避免 LLM overfit 同一对)
3. 在 prompt 中明示"以下是好/坏对比,你的输出**像好示例**,**避免坏示例的问题模式**"
